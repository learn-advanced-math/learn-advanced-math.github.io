7.9 欧拉方程

Euler Equation

7.9.1 欧拉方程的形式

欧拉方程的定义

形如：

$$x^n y^{(n)} + a_1 x^{n-1} y^{(n-1)} + a_2 x^{n-2} y^{(n-2)} + \cdots + a_{n-1} x y' + a_n y = f(x)$$

的微分方程称为$n$ 阶欧拉方程，其中 $a_1, a_2, \ldots, a_n$ 是常数，$x > 0$。

二阶欧拉方程的标准形式

$$x^2 y'' + px y' + qy = f(x)$$

其中 $p$、$q$ 是常数

特点：欧拉方程的特点是每一项中 $x$ 的幂次与对应导数的阶数相同。

欧拉方程的历史背景

欧拉方程是由瑞士数学家莱昂哈德·欧拉（Leonhard Euler，1707-1783）首先研究的一类重要的变系数线性微分方程。虽然系数不是常数，但可以通过巧妙的变量替换转化为常系数线性微分方程。

7.9.2 欧拉方程的求解方法

变量替换法

解欧拉方程的关键是进行变量替换：

$$x = e^t \quad \text{或} \quad t = \ln x$$

这样可以将欧拉方程转化为常系数线性微分方程。

欧拉方程
$$x^2 y'' + px y' + qy = 0$$
→
常系数方程
$$\frac{d^2y}{dt^2} + (p-1)\frac{dy}{dt} + qy = 0$$
导数变换公式推导

设 $y = y(x)$，$x = e^t$，则：

一阶导数：

$$\frac{dy}{dx} = \frac{dy}{dt} \cdot \frac{dt}{dx} = \frac{dy}{dt} \cdot \frac{1}{x} = \frac{1}{x}\frac{dy}{dt}$$

二阶导数：

$$\frac{d^2y}{dx^2} = \frac{d}{dx}\left(\frac{1}{x}\frac{dy}{dt}\right) = -\frac{1}{x^2}\frac{dy}{dt} + \frac{1}{x}\frac{d^2y}{dt^2}\cdot\frac{dt}{dx}$$ $$= \frac{1}{x^2}\left(\frac{d^2y}{dt^2} - \frac{dy}{dt}\right)$$

因此：

$$x\frac{dy}{dx} = \frac{dy}{dt}, \quad x^2\frac{d^2y}{dx^2} = \frac{d^2y}{dt^2} - \frac{dy}{dt}$$
标准求解步骤
步骤1：进行变量替换 $x = e^t$（或 $t = \ln x$）
步骤2：利用导数变换公式，将欧拉方程转化为常系数线性微分方程
步骤3：求解常系数线性微分方程
步骤4：将解中的 $t$ 替换回 $\ln x$，得到原方程的解

7.9.3 典型例题解析

例题1：基本欧拉方程

求解欧拉方程：$x^2 y'' - 2xy' + 2y = 0$

解题过程：

步骤1：变量替换

设 $x = e^t$，则 $t = \ln x$

步骤2：导数变换

$x \frac{dy}{dx} = \frac{dy}{dt}$，$x^2 \frac{d^2y}{dx^2} = \frac{d^2y}{dt^2} - \frac{dy}{dt}$

步骤3：代入原方程

原方程变为：

$$\left(\frac{d^2y}{dt^2} - \frac{dy}{dt}\right) - 2\frac{dy}{dt} + 2y = 0$$ $$\frac{d^2y}{dt^2} - 3\frac{dy}{dt} + 2y = 0$$

步骤4：求解常系数方程

特征方程：$r^2 - 3r + 2 = 0$

解得：$(r-1)(r-2) = 0$，即 $r_1 = 1$，$r_2 = 2$

通解：$y = C_1 e^t + C_2 e^{2t}$

步骤5：回代

将 $t = \ln x$ 代入：

$$y = C_1 e^{\ln x} + C_2 e^{2\ln x} = C_1 x + C_2 x^2$$

例题2：复根情况

求解欧拉方程：$x^2 y'' + xy' + y = 0$

解题过程：

步骤1：变量替换和导数变换

设 $x = e^t$，代入得：

$$\frac{d^2y}{dt^2} - \frac{dy}{dt} + \frac{dy}{dt} + y = 0$$ $$\frac{d^2y}{dt^2} + y = 0$$

步骤2：求解常系数方程

特征方程：$r^2 + 1 = 0$

解得：$r_{1,2} = \pm i$

通解：$y = C_1 \cos t + C_2 \sin t$

步骤3：回代

将 $t = \ln x$ 代入：

$$y = C_1 \cos(\ln x) + C_2 \sin(\ln x)$$

例题3：非齐次欧拉方程

求解欧拉方程：$x^2 y'' - xy' + y = x^2$

解题过程：

步骤1：求齐次方程的解

齐次方程：$x^2 y'' - xy' + y = 0$

设 $x = e^t$，得：$\frac{d^2y}{dt^2} - 2\frac{dy}{dt} + y = 0$

特征方程：$r^2 - 2r + 1 = 0$，即 $(r-1)^2 = 0$

齐次通解：$y_h = (C_1 + C_2 t)e^t = (C_1 + C_2 \ln x)x$

步骤2：处理非齐次项

$f(x) = x^2 = e^{2t}$，非齐次方程变为：

$$\frac{d^2y}{dt^2} - 2\frac{dy}{dt} + y = e^{2t}$$

步骤3：求特解

设特解 $y_p = Ae^{2t}$

代入得：$4Ae^{2t} - 4Ae^{2t} + Ae^{2t} = e^{2t}$

解得：$A = 1$，特解：$y_p = e^{2t} = x^2$

步骤4：通解

$$y = (C_1 + C_2 \ln x)x + x^2$$

7.9.4 欧拉方程与常系数方程的对照

通解形式对照表

特征根类型 常系数方程通解 欧拉方程通解
不相等实根 $r_1, r_2$ $C_1 e^{r_1 t} + C_2 e^{r_2 t}$ $C_1 x^{r_1} + C_2 x^{r_2}$
相等实根 $r$（二重） $(C_1 + C_2 t) e^{rt}$ $(C_1 + C_2 \ln x) x^r$
复根 $\alpha \pm \beta i$ $e^{\alpha t}(C_1 \cos \beta t + C_2 \sin \beta t)$ $x^{\alpha}(C_1 \cos(\beta \ln x) + C_2 \sin(\beta \ln x))$

记忆技巧与解题要点

  • 识别特征："$x$ 的幂次 = 导数阶数"
  • 替换公式：$x = e^t$ 是万能钥匙
  • 导数关系：$x \frac{dy}{dx} = \frac{dy}{dt}$，$x^2 \frac{d^2y}{dx^2} = \frac{d^2y}{dt^2} - \frac{dy}{dt}$
  • 解的形式：$e^{rt} \rightarrow x^r$，$t \rightarrow \ln x$
  • 定义域：欧拉方程要求 $x > 0$

7.9.5 练习题

习题1 中等难度

求解欧拉方程：$x^2 y'' + 3xy' + y = 0$

解题步骤：

步骤1：变量替换

设 $x = e^t$，代入导数变换公式

步骤2：转化为常系数方程

$$\frac{d^2y}{dt^2} - \frac{dy}{dt} + 3\frac{dy}{dt} + y = 0$$ $$\frac{d^2y}{dt^2} + 2\frac{dy}{dt} + y = 0$$

步骤3：求解特征方程

$r^2 + 2r + 1 = 0$，即 $(r+1)^2 = 0$

二重根：$r = -1$

步骤4：写出通解

$y = (C_1 + C_2 t)e^{-t} = (C_1 + C_2 \ln x)x^{-1}$

$$y = \frac{C_1 + C_2 \ln x}{x}$$

习题2 较难

求解欧拉方程：$x^2 y'' - xy' + 2y = 0$

解题步骤：

步骤1：变量替换

设 $x = e^t$，转化得：

$$\frac{d^2y}{dt^2} - 2\frac{dy}{dt} + 2y = 0$$

步骤2：求解特征方程

$r^2 - 2r + 2 = 0$

使用求根公式：$r = \frac{2 \pm \sqrt{4-8}}{2} = 1 \pm i$

步骤3：写出通解

$\alpha = 1$，$\beta = 1$

$y = e^t(C_1 \cos t + C_2 \sin t)$

步骤4：回代

$$y = x(C_1 \cos(\ln x) + C_2 \sin(\ln x))$$

习题3 很难

求解非齐次欧拉方程：$x^2 y'' + xy' - y = x \ln x$

解题步骤：

步骤1：求齐次方程的解

设 $x = e^t$，齐次方程变为：

$$\frac{d^2y}{dt^2} - y = 0$$

特征方程：$r^2 - 1 = 0$，$r = \pm 1$

齐次通解：$y_h = C_1 e^t + C_2 e^{-t} = C_1 x + C_2 x^{-1}$

步骤2：处理非齐次项

$f(x) = x \ln x = e^t \cdot t$

非齐次方程变为：$\frac{d^2y}{dt^2} - y = te^t$

步骤3：求特解

由于 $1$ 是单重特征根，设特解：$y_p = t(at + b)e^t$

代入方程求得：$a = \frac{1}{4}$，$b = -\frac{1}{4}$

特解：$y_p = \frac{t^2 - t}{4}e^t = \frac{(\ln x)^2 - \ln x}{4} \cdot x$

步骤4：通解

$$y = C_1 x + \frac{C_2}{x} + \frac{(\ln x)^2 - \ln x}{4} \cdot x$$
