7.3 齐次方程

Homogeneous Differential Equations

7.3.1 齐次方程的概念及求解

齐次方程的定义

形如 $\frac{dy}{dx} = f\left(\frac{y}{x}\right)$ 的一阶微分方程称为齐次方程。

或者等价地，如果方程可以写成：

$$M(x, y)dx + N(x, y)dy = 0$$

其中 $M(x, y)$ 和 $N(x, y)$ 都是 $x$ 和 $y$ 的 $n$ 次齐次函数，即：

  • $M(tx, ty) = t^n M(x, y)$
  • $N(tx, ty) = t^n N(x, y)$

那么此方程也称为齐次方程。

标准形式一
$$\frac{dy}{dx} = f\left(\frac{y}{x}\right)$$

函数的右端只依赖于 $\frac{y}{x}$ 的比值

标准形式二
$$M(x, y)dx + N(x, y)dy = 0$$

其中 $M$ 和 $N$ 是同次齐次函数

齐次方程的求解方法

通过变量替换 $v = \frac{y}{x}$（即 $y = vx$）可以将齐次方程转化为可分离变量的方程。

求解步骤：
  1. 令 $v = \frac{y}{x}$，则 $y = vx$
  2. 求导：$\frac{dy}{dx} = v + x\frac{dv}{dx}$
  3. 代入原方程：$v + x\frac{dv}{dx} = f(v)$
  4. 整理得：$x\frac{dv}{dx} = f(v) - v$
  5. 分离变量：$\frac{dv}{f(v) - v} = \frac{dx}{x}$
  6. 两边积分求解
  7. 将 $v = \frac{y}{x}$ 代回得到最终解

例1：标准齐次方程

求解微分方程：$\frac{dy}{dx} = \frac{x + y}{x}$

解：
步骤1：验证齐次性

方程可写成：$\frac{dy}{dx} = 1 + \frac{y}{x}$，这是齐次方程的标准形式。

步骤2：变量替换

令 $v = \frac{y}{x}$，则 $y = vx$，$\frac{dy}{dx} = v + x\frac{dv}{dx}$

步骤3：代入原方程

$v + x\frac{dv}{dx} = 1 + v$

简化得：$x\frac{dv}{dx} = 1$

步骤4：分离变量并积分

$dv = \frac{dx}{x}$

积分：$\int dv = \int \frac{dx}{x}$

得到：$v = \ln|x| + C$

步骤5：回代并得到最终解

由 $v = \frac{y}{x}$，得：$\frac{y}{x} = \ln|x| + C$

最终解：$y = x(\ln|x| + C)$

例2：一般形式的齐次方程

求解微分方程：$(x^2 + y^2)dx - xy dy = 0$

解：
步骤1：验证齐次性

$M(x, y) = x^2 + y^2$，$N(x, y) = -xy$

$M(tx, ty) = t^2x^2 + t^2y^2 = t^2(x^2 + y^2) = t^2M(x, y)$

$N(tx, ty) = -t^2xy = t^2(-xy) = t^2N(x, y)$

因此这是齐次方程。

步骤2：化为标准形式

$\frac{dy}{dx} = \frac{x^2 + y^2}{xy} = \frac{x}{y} + \frac{y}{x}$

步骤3：变量替换

令 $v = \frac{y}{x}$，则 $\frac{dy}{dx} = v + x\frac{dv}{dx}$

原方程变为：$v + x\frac{dv}{dx} = \frac{1}{v} + v$

简化：$x\frac{dv}{dx} = \frac{1}{v}$

步骤4：分离变量并积分

$v dv = \frac{dx}{x}$

积分：$\int v dv = \int \frac{dx}{x}$

得到：$\frac{v^2}{2} = \ln|x| + C$

步骤5：回代得到最终解

$\frac{1}{2}\left(\frac{y}{x}\right)^2 = \ln|x| + C$

最终解：$y^2 = 2x^2(\ln|x| + C)$

7.3.2 可化为齐次的方程的求解

可化为齐次方程的类型

有些方程虽然本身不是齐次方程，但通过适当的变量替换可以转化为齐次方程。主要有以下几种类型：

类型一：形如 $\frac{dy}{dx} = f\left(\frac{ax + by + c}{dx + ey + f}\right)$ 的方程

类型二：形如 $(a_1x + b_1y + c_1)dx + (a_2x + b_2y + c_2)dy = 0$ 的方程

类型一的求解方法

对于方程 $\frac{dy}{dx} = f\left(\frac{ax + by + c}{dx + ey + f}\right)$：

当 $c = f = 0$ 时：

方程已经是齐次的，直接用齐次方程的方法求解。

当 $c \neq 0$ 或 $f \neq 0$ 时：
  1. 求解线性方程组：$\begin{cases} ax + by + c = 0 \\ dx + ey + f = 0 \end{cases}$
  2. 设解为 $(x_0, y_0)$，作变量替换：$\begin{cases} X = x - x_0 \\ Y = y - y_0 \end{cases}$
  3. 原方程转化为齐次方程：$\frac{dY}{dX} = f\left(\frac{aX + bY}{dX + eY}\right)$
  4. 用齐次方程方法求解
当线性方程组无解时：

说明两直线平行，此时 $\frac{a}{d} = \frac{b}{e}$，可用其他方法求解。

类型二的求解方法

对于方程 $(a_1x + b_1y + c_1)dx + (a_2x + b_2y + c_2)dy = 0$：

情况1：$\frac{a_1}{a_2} \neq \frac{b_1}{b_2}$

与类型一相同，通过平移坐标系使其变为齐次方程。

情况2：$\frac{a_1}{a_2} = \frac{b_1}{b_2} = k$

此时令 $u = a_2x + b_2y$，方程变为：

$(ku + c_1)dx + (u + c_2)dy = 0$

这是关于 $u$ 和 $x$（或 $y$）的可分离变量方程。

例3：可化为齐次的方程

求解微分方程：$\frac{dy}{dx} = \frac{x + y - 1}{x - y + 3}$

解：
步骤1：求线性方程组的解

解方程组：$\begin{cases} x + y - 1 = 0 \\ x - y + 3 = 0 \end{cases}$

解得：$x_0 = -1$，$y_0 = 2$

步骤2：变量替换

令 $X = x + 1$，$Y = y - 2$

则 $x = X - 1$，$y = Y + 2$

$\frac{dy}{dx} = \frac{dY}{dX}$

步骤3：代入得到齐次方程

$\frac{dY}{dX} = \frac{(X-1) + (Y+2) - 1}{(X-1) - (Y+2) + 3} = \frac{X + Y}{X - Y}$

步骤4：用齐次方程方法求解

令 $v = \frac{Y}{X}$，则 $Y = vX$，$\frac{dY}{dX} = v + X\frac{dv}{dX}$

$v + X\frac{dv}{dX} = \frac{X + vX}{X - vX} = \frac{1 + v}{1 - v}$

$X\frac{dv}{dX} = \frac{1 + v}{1 - v} - v = \frac{1 + v - v(1 - v)}{1 - v} = \frac{1 + v^2}{1 - v}$

步骤5：分离变量并积分

$\frac{1 - v}{1 + v^2}dv = \frac{dX}{X}$

积分左边：$\int \frac{1 - v}{1 + v^2}dv = \int \frac{1}{1 + v^2}dv - \int \frac{v}{1 + v^2}dv$

$= \arctan v - \frac{1}{2}\ln(1 + v^2)$

因此：$\arctan v - \frac{1}{2}\ln(1 + v^2) = \ln|X| + C$

步骤6：回代得到最终解

$v = \frac{Y}{X} = \frac{y-2}{x+1}$

最终解：$\arctan\frac{y-2}{x+1} - \frac{1}{2}\ln\left(1 + \frac{(y-2)^2}{(x+1)^2}\right) = \ln|x+1| + C$

例4：比例系数相等的情况

求解微分方程：$(2x + 3y + 1)dx + (4x + 6y + 5)dy = 0$

解：
步骤1：检查比例关系

$\frac{2}{4} = \frac{3}{6} = \frac{1}{2}$，所以 $a_1 : a_2 = b_1 : b_2$

方程可写成：$(2x + 3y + 1)dx + 2(2x + 3y) + 5)dy = 0$

步骤2：变量替换

令 $u = 2x + 3y$，则 $du = 2dx + 3dy$

从而 $dx = \frac{du - 3dy}{2}$

步骤3：代入原方程

$(u + 1) \cdot \frac{du - 3dy}{2} + (2u + 5)dy = 0$

展开：$\frac{(u + 1)(du - 3dy)}{2} + (2u + 5)dy = 0$

$\frac{(u + 1)du}{2} - \frac{3(u + 1)dy}{2} + (2u + 5)dy = 0$

$\frac{(u + 1)du}{2} + \left(2u + 5 - \frac{3(u + 1)}{2}\right)dy = 0$

步骤4：分离变量

整理系数：$2u + 5 - \frac{3(u + 1)}{2} = \frac{4u + 10 - 3u - 3}{2} = \frac{u + 7}{2}$

方程变为：$\frac{(u + 1)du}{2} + \frac{(u + 7)dy}{2} = 0$

即：$(u + 1)du + (u + 7)dy = 0$

步骤5：求解并回代

这仍然不是直接可分离的。重新考虑，用 $y$ 作为独立变量：

从 $u = 2x + 3y$ 得 $x = \frac{u - 3y}{2}$

原方程最终可化简求解得到隐式解。

习题1 中等难度

求解微分方程：$\frac{dy}{dx} = \frac{x^2 + xy + y^2}{x^2}$

解：

方程可写成：$\frac{dy}{dx} = 1 + \frac{y}{x} + \left(\frac{y}{x}\right)^2$

这是齐次方程。令 $v = \frac{y}{x}$，则：

$v + x\frac{dv}{dx} = 1 + v + v^2$

$x\frac{dv}{dx} = 1 + v^2$

分离变量：$\frac{dv}{1 + v^2} = \frac{dx}{x}$

积分：$\arctan v = \ln|x| + C$

解：$\arctan\frac{y}{x} = \ln|x| + C$

习题2 较难

求解微分方程：$(x - y + 1)dx + (x + y - 3)dy = 0$

解：

步骤1：求线性方程组解：$\begin{cases} x - y + 1 = 0 \\ x + y - 3 = 0 \end{cases}$

解得：$x_0 = 1$，$y_0 = 2$

步骤2：变量替换：$X = x - 1$，$Y = y - 2$

方程变为：$X dx + Y dy + (-Y + X)dx + (X + Y)dy = 0$

简化：$(X - Y)dX + (X + Y)dY = 0$

步骤3：这是齐次方程，用 $v = \frac{Y}{X}$ 求解

$\frac{dY}{dX} = \frac{Y - X}{X + Y} = \frac{v - 1}{1 + v}$

步骤4：分离变量并积分得到解

最终解：$(x-1)^2 + (y-2)^2 = C(x-1)$

习题3 很难

求解微分方程：$xy\frac{dy}{dx} = x^2 + 2y^2$，并求满足初始条件 $y(1) = 1$ 的特解。

解：
步骤1：判断方程类型并变形

原方程：$xy\frac{dy}{dx} = x^2 + 2y^2$

变形为：$\frac{dy}{dx} = \frac{x^2 + 2y^2}{xy} = \frac{x}{y} + \frac{2y}{x}$

这是齐次方程，因为右端可以表示为 $\frac{y}{x}$ 的函数。

步骤2：变量替换

令 $v = \frac{y}{x}$，则 $y = vx$

求导：$\frac{dy}{dx} = v + x\frac{dv}{dx}$

将 $\frac{dy}{dx} = \frac{x}{y} + \frac{2y}{x}$ 用 $v$ 表示：

$\frac{dy}{dx} = \frac{x}{vx} + \frac{2vx}{x} = \frac{1}{v} + 2v$

步骤3：建立关于v的微分方程

代入得：$v + x\frac{dv}{dx} = \frac{1}{v} + 2v$

简化：$x\frac{dv}{dx} = \frac{1}{v} + 2v - v = \frac{1}{v} + v$

进一步简化：$x\frac{dv}{dx} = \frac{1 + v^2}{v}$

步骤4：分离变量并积分

分离变量：$\frac{v dv}{1 + v^2} = \frac{dx}{x}$

左边积分：$\int \frac{v dv}{1 + v^2}$

令 $u = 1 + v^2$，则 $du = 2v dv$，所以 $v dv = \frac{1}{2}du$

$\int \frac{v dv}{1 + v^2} = \int \frac{1}{2u}du = \frac{1}{2}\ln|u| = \frac{1}{2}\ln(1 + v^2)$

右边积分：$\int \frac{dx}{x} = \ln|x|$

步骤5：求通解

积分结果：$\frac{1}{2}\ln(1 + v^2) = \ln|x| + C_1$

整理：$\ln(1 + v^2) = 2\ln|x| + 2C_1 = \ln x^2 + \ln C^2$（其中 $C = e^{C_1}$）

因此：$1 + v^2 = Cx^2$

回代 $v = \frac{y}{x}$：$1 + \frac{y^2}{x^2} = Cx^2$

通解：$x^2 + y^2 = Cx^4$

步骤6：利用初始条件求特解

初始条件：$y(1) = 1$

代入通解：$1^2 + 1^2 = C \cdot 1^4$

$2 = C$

特解：$x^2 + y^2 = 2x^4$

步骤7：验证解的正确性

将特解 $x^2 + y^2 = 2x^4$ 对 $x$ 求导：

$2x + 2y\frac{dy}{dx} = 8x^3$

解得：$\frac{dy}{dx} = \frac{8x^3 - 2x}{2y} = \frac{4x^3 - x}{y}$

而原方程右端：$\frac{x^2 + 2y^2}{xy}$

从特解知 $y^2 = 2x^4 - x^2$，代入验证解的正确性。
