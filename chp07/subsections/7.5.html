7.5 可降阶的高阶微分方程

Reducible Higher-Order Differential Equations

7.5.1 可降阶高阶微分方程的概念

可降阶的高阶微分方程

某些高阶微分方程虽然不是线性的，但可以通过适当的变量替换将其阶数降低，转化为低阶微分方程求解，这类方程称为可降阶的高阶微分方程。

核心思想：通过变量替换，将 $n$ 阶微分方程化为 $n-1$ 阶或更低阶的微分方程，从而简化求解过程。

常见的可降阶高阶微分方程类型

类型一：$y^{(n)} = f(x)$

右端仅含 $x$ 的函数

类型二：$F(x, y^{(k)}, y^{(k+1)}, \ldots, y^{(n)}) = 0$

不显含 $y, y', \ldots, y^{(k-1)}$

类型三：$F(y, y', y'') = 0$

不显含自变量 $x$

类型四：$F(x, y', y'') = 0$

不显含未知函数 $y$

7.5.2 类型一：$y^{(n)} = f(x)$ 型方程

求解方法

对于形如 $y^{(n)} = f(x)$ 的方程，可以通过连续积分 $n$ 次来求解：

$$y^{(n-1)} = \int f(x)dx + C_1$$ $$y^{(n-2)} = \int \left[\int f(x)dx + C_1\right]dx + C_2$$ $$\vdots$$ $$y = \int \cdots \int f(x)(dx)^n + C_1x^{n-1} + C_2x^{n-2} + \cdots + C_n$$
求解步骤
步骤1：对方程 $y^{(n)} = f(x)$ 进行第一次积分
步骤2：继续积分，每次积分增加一个任意常数
步骤3：重复积分 $n$ 次，得到通解
步骤4：如有初始条件，确定任意常数

例题1：简单的 $n$ 阶导数型

求解微分方程：$y''' = 6x$

解题过程：

步骤1：第一次积分

$$y'' = \int 6x dx = 3x^2 + C_1$$

步骤2：第二次积分

$$y' = \int (3x^2 + C_1) dx = x^3 + C_1x + C_2$$

步骤3：第三次积分

$$y = \int (x^3 + C_1x + C_2) dx = \frac{x^4}{4} + \frac{C_1x^2}{2} + C_2x + C_3$$

通解：$y = \frac{x^4}{4} + \frac{C_1x^2}{2} + C_2x + C_3$

7.5.3 类型二：不显含低阶导数的方程

变量替换法

对于形如 $F(x, y^{(k)}, y^{(k+1)}, \ldots, y^{(n)}) = 0$ 的方程（不显含 $y, y', \ldots, y^{(k-1)}$），

令 $z = y^{(k)}$，则原方程化为关于 $z$ 的 $(n-k)$ 阶方程：

$$F(x, z, z', z'', \ldots, z^{(n-k)}) = 0$$

例题2：不显含 $y$ 和 $y'$ 的二阶方程

求解微分方程：$xy'' + y' = x$

解题过程：

步骤1：变量替换

令 $p = y'$，则 $y'' = p' = \frac{dp}{dx}$

原方程化为：$x\frac{dp}{dx} + p = x$

步骤2：求解关于 $p$ 的一阶线性方程

$$\frac{dp}{dx} + \frac{p}{x} = 1$$

这是一阶线性方程，$P(x) = \frac{1}{x}$，$Q(x) = 1$

步骤3：应用一阶线性方程通解公式

积分因子：$\mu(x) = e^{\int \frac{1}{x}dx} = e^{\ln x} = x$

$$p = \frac{1}{x}\left[\int 1 \cdot x dx + C_1\right] = \frac{1}{x}\left[\frac{x^2}{2} + C_1\right] = \frac{x}{2} + \frac{C_1}{x}$$

步骤4：求 $y$

$$y = \int p dx = \int \left(\frac{x}{2} + \frac{C_1}{x}\right) dx = \frac{x^2}{4} + C_1\ln|x| + C_2$$

通解：$y = \frac{x^2}{4} + C_1\ln|x| + C_2$

7.5.4 类型三：$F(y, y', y'') = 0$ 型方程

以 $y$ 为自变量的变换

对于不显含自变量 $x$ 的二阶方程 $F(y, y', y'') = 0$，

令 $p = y'$，利用 $y'' = \frac{dp}{dx} = \frac{dp}{dy} \cdot \frac{dy}{dx} = p\frac{dp}{dy}$，

原方程化为关于 $y$ 和 $p$ 的一阶方程：

$$F\left(y, p, p\frac{dp}{dy}\right) = 0$$
求解步骤
步骤1：设 $p = y'$，用链式法则得 $y'' = p\frac{dp}{dy}$
步骤2：将原方程化为 $F(y, p, p\frac{dp}{dy}) = 0$
步骤3：求解关于 $p$ 和 $y$ 的一阶方程，得 $p = \phi(y, C_1)$
步骤4：由 $\frac{dy}{dx} = p = \phi(y, C_1)$ 分离变量求出 $y(x)$

例题3：不显含 $x$ 的二阶方程

求解微分方程：$yy'' = (y')^2$

解题过程：

步骤1：变量替换

设 $p = y'$，则 $y'' = p\frac{dp}{dy}$

原方程化为：$y \cdot p\frac{dp}{dy} = p^2$

步骤2：分离变量

当 $p \neq 0$ 时，两边除以 $p$：

$$y\frac{dp}{dy} = p$$ $$\frac{dp}{p} = \frac{dy}{y}$$

步骤3：积分

$$\int \frac{dp}{p} = \int \frac{dy}{y}$$ $$\ln|p| = \ln|y| + \ln|C_1|$$ $$p = C_1y$$

步骤4：求 $y(x)$

由 $\frac{dy}{dx} = C_1y$，分离变量：

$$\frac{dy}{y} = C_1dx$$ $$\ln|y| = C_1x + C_2$$ $$y = C_3e^{C_1x}$$

步骤5：检验特殊解

当 $p = 0$ 时，$y' = 0$，即 $y = \text{常数}$

代入原方程：$y \cdot 0 = 0^2$ ✓，所以 $y = C$ 也是解

通解：$y = Ce^{C_1x}$ 或 $y = C$（常数）

7.5.5 类型四：$F(x, y', y'') = 0$ 型方程

直接变量替换

对于不显含未知函数 $y$ 的二阶方程 $F(x, y', y'') = 0$，

令 $p = y'$，则 $y'' = \frac{dp}{dx}$，

原方程化为关于 $p$ 和 $x$ 的一阶方程：

$$F\left(x, p, \frac{dp}{dx}\right) = 0$$

例题4：不显含 $y$ 的二阶方程

求解微分方程：$x^2y'' + xy' = 1$

解题过程：

步骤1：变量替换

设 $p = y'$，则 $y'' = \frac{dp}{dx}$

原方程化为：$x^2\frac{dp}{dx} + xp = 1$

步骤2：化为标准一阶线性方程

$$\frac{dp}{dx} + \frac{p}{x} = \frac{1}{x^2}$$

$P(x) = \frac{1}{x}$，$Q(x) = \frac{1}{x^2}$

步骤3：求解一阶线性方程

积分因子：$\mu(x) = e^{\int \frac{1}{x}dx} = x$

$$p = \frac{1}{x}\left[\int \frac{1}{x^2} \cdot x dx + C_1\right] = \frac{1}{x}\left[\int \frac{1}{x} dx + C_1\right]$$ $$= \frac{1}{x}[\ln|x| + C_1] = \frac{\ln|x|}{x} + \frac{C_1}{x}$$

步骤4：求 $y$

$$y = \int p dx = \int \left(\frac{\ln|x|}{x} + \frac{C_1}{x}\right) dx$$

计算第一个积分：

设 $u = \ln|x|$，$du = \frac{1}{x}dx$

$$\int \frac{\ln|x|}{x} dx = \int u du = \frac{u^2}{2} = \frac{(\ln|x|)^2}{2}$$

所以：

$$y = \frac{(\ln|x|)^2}{2} + C_1\ln|x| + C_2$$

通解：$y = \frac{(\ln|x|)^2}{2} + C_1\ln|x| + C_2$

降阶法记忆技巧

  • 识别口诀："缺谁换谁，降一阶"
  • 类型一：只有最高阶导数 → 连续积分
  • 类型二：缺低阶导数 → 设 $z = y^{(k)}$
  • 类型三：不含 $x$ → 设 $p = y'$，$y'' = p\frac{dp}{dy}$
  • 类型四：不含 $y$ → 设 $p = y'$，$y'' = \frac{dp}{dx}$
  • 关键：正确识别方程类型，选择合适的变量替换

7.5.7 练习题

习题1 中等难度

求解微分方程：$y'' = 2e^x$

解题步骤：

类型：$y^{(n)} = f(x)$ 型，连续积分求解

步骤1：第一次积分

$$y' = \int 2e^x dx = 2e^x + C_1$$

步骤2：第二次积分

$$y = \int (2e^x + C_1) dx = 2e^x + C_1x + C_2$$

答案：$y = 2e^x + C_1x + C_2$

习题2 较难

求解微分方程：$(y')^2 = 2yy''$

解题步骤：

类型：$F(y, y', y'') = 0$ 型，不显含 $x$

步骤1：变量替换

设 $p = y'$，则 $y'' = p\frac{dp}{dy}$

原方程：$p^2 = 2y \cdot p\frac{dp}{dy}$

步骤2：分离变量

当 $p \neq 0$ 时：$p = 2y\frac{dp}{dy}$

$$\frac{dp}{p} = \frac{dy}{2y}$$

步骤3：积分

$$\ln|p| = \frac{1}{2}\ln|y| + C_1$$ $$p = C_1\sqrt{y}$$

步骤4：求 $y(x)$

$$\frac{dy}{dx} = C_1\sqrt{y}$$ $$\frac{dy}{\sqrt{y}} = C_1dx$$ $$2\sqrt{y} = C_1x + C_2$$ $$y = \frac{(C_1x + C_2)^2}{4}$$

答案：$y = \frac{(C_1x + C_2)^2}{4}$ 及 $y = 0$

习题3 很难

求解微分方程：$2xy'' - y' = x$

解题步骤：

类型：$F(x, y', y'') = 0$ 型，不显含 $y$

步骤1：变量替换

设 $p = y'$，则 $y'' = \frac{dp}{dx}$

原方程：$2x\frac{dp}{dx} - p = x$

步骤2：化为标准一阶线性方程

$$\frac{dp}{dx} - \frac{p}{2x} = \frac{1}{2}$$

$P(x) = -\frac{1}{2x}$，$Q(x) = \frac{1}{2}$

步骤3：求解一阶线性方程

积分因子：$\mu(x) = e^{-\int \frac{1}{2x}dx} = e^{-\frac{1}{2}\ln|x|} = \frac{1}{\sqrt{x}}$

$$p = \sqrt{x}\left[\int \frac{1}{2} \cdot \frac{1}{\sqrt{x}} dx + C_1\right]$$ $$= \sqrt{x}\left[\int \frac{1}{2\sqrt{x}} dx + C_1\right]$$ $$= \sqrt{x}\left[\sqrt{x} + C_1\right] = x + C_1\sqrt{x}$$

步骤4：求 $y$

$$y = \int (x + C_1\sqrt{x}) dx = \frac{x^2}{2} + C_1 \cdot \frac{2x^{3/2}}{3} + C_2$$ $$= \frac{x^2}{2} + \frac{2C_1x^{3/2}}{3} + C_2$$

答案：$y = \frac{x^2}{2} + \frac{2C_1x^{3/2}}{3} + C_2$
