7.7 常系数齐次线性微分方程

Homogeneous Linear Differential Equations with Constant Coefficients

7.7.1 二阶常系数齐次线性微分方程

二阶常系数齐次线性微分方程的标准形式

形如：

$$y'' + py' + qy = 0$$

的微分方程称为二阶常系数齐次线性微分方程，其中 $p$、$q$ 是常数。

特征方程法

解二阶常系数齐次线性微分方程的基本方法是特征方程法。

设方程的解为 $y = e^{rx}$，代入原方程得：

$$r^2 e^{rx} + pr e^{rx} + q e^{rx} = 0$$

由于 $e^{rx} \neq 0$，得到特征方程：

特征方程

$$r^2 + pr + q = 0$$
求解步骤
步骤1：写出微分方程 $y'' + py' + qy = 0$ 的特征方程 $r^2 + pr + q = 0$
步骤2：求解特征方程，得到特征根 $r_1$、$r_2$
步骤3：根据特征根的情况，写出通解

根据判别式的三种情况

情况1：两个不相等实根

当 $\Delta = p^2 - 4q > 0$ 时

特征根：$r_1 \neq r_2$（实数）

通解

$$y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$$
情况2：两个相等实根

当 $\Delta = p^2 - 4q = 0$ 时

特征根：$r_1 = r_2 = r$（重根）

通解

$$y = (C_1 + C_2 x) e^{rx}$$
情况3：一对共轭复根

当 $\Delta = p^2 - 4q < 0$ 时

特征根：$r_{1,2} = \alpha \pm \beta i$

通解

$$y = e^{\alpha x}(C_1 \cos \beta x + C_2 \sin \beta x)$$

7.7.2 $n$阶常系数齐次线性微分方程

$n$阶常系数齐次线性微分方程的标准形式

形如：

$$y^{(n)} + a_1 y^{(n-1)} + a_2 y^{(n-2)} + \cdots + a_{n-1} y' + a_n y = 0$$

的微分方程称为n阶常系数齐次线性微分方程，其中 $a_1, a_2, \ldots, a_n$ 是常数。

$n$阶特征方程

$$r^n + a_1 r^{n-1} + a_2 r^{n-2} + \cdots + a_{n-1} r + a_n = 0$$

通解的构成规律

设特征方程有 $n$ 个根（计重数），根据根的性质，通解由以下规律构成：

  1. 单实根 $r$：对应解 $e^{rx}$
  2. $k$ 重实根 $r$：对应解 $e^{rx}, xe^{rx}, x^2e^{rx}, \ldots, x^{k-1}e^{rx}$
  3. 单复根 $\alpha \pm \beta i$：对应解 $e^{\alpha x}\cos \beta x, e^{\alpha x}\sin \beta x$
  4. $k$ 重复根 $\alpha \pm \beta i$：对应解：
$$e^{\alpha x}\cos \beta x, xe^{\alpha x}\cos \beta x, \ldots, x^{k-1}e^{\alpha x}\cos \beta x$$ $$e^{\alpha x}\sin \beta x, xe^{\alpha x}\sin \beta x, \ldots, x^{k-1}e^{\alpha x}\sin \beta x$$

7.7.3 典型例题解析

例题1：两个不相等实根

求解微分方程：$y'' - 5y' + 6y = 0$

解题过程：

步骤1：写出特征方程

$$r^2 - 5r + 6 = 0$$

步骤2：解特征方程

因式分解：$(r - 2)(r - 3) = 0$

得特征根：$r_1 = 2$，$r_2 = 3$

步骤3：写出通解

因为 $r_1 \neq r_2$，所以通解为：

$$y = C_1 e^{2x} + C_2 e^{3x}$$

例题2：两个相等实根

求解微分方程：$y'' - 4y' + 4y = 0$

解题过程：

步骤1：写出特征方程

$$r^2 - 4r + 4 = 0$$

步骤2：解特征方程

因式分解：$(r - 2)^2 = 0$

得二重特征根：$r_1 = r_2 = 2$

步骤3：写出通解

因为是二重实根，所以通解为：

$$y = (C_1 + C_2 x) e^{2x}$$

例题3：一对共轭复根

求解微分方程：$y'' + 2y' + 5y = 0$

解题过程：

步骤1：写出特征方程

$$r^2 + 2r + 5 = 0$$

步骤2：解特征方程

使用求根公式：

$$r = \frac{-2 \pm \sqrt{4 - 20}}{2} = \frac{-2 \pm \sqrt{-16}}{2} = \frac{-2 \pm 4i}{2} = -1 \pm 2i$$

步骤3：确定 $\alpha$ 和 $\beta$

$\alpha = -1$，$\beta = 2$

步骤4：写出通解

$$y = e^{-x}(C_1 \cos 2x + C_2 \sin 2x)$$

记忆技巧与解题要点

  • 特征方程口诀："导数变成乘方，系数保持不变"
  • 实根规律：单根用指数，重根乘多项式
  • 复根规律：实部做指数，虚部做三角
  • 判别式：$\Delta > 0$ 两实根，$\Delta = 0$ 重根，$\Delta < 0$ 复根
  • 验证方法：将求得的通解代入原方程验证

7.7.5 练习题

习题1 中等难度

求解微分方程：$y'' + y' - 6y = 0$

解题步骤：

步骤1：写出特征方程

$$r^2 + r - 6 = 0$$

步骤2：解特征方程

因式分解：$(r + 3)(r - 2) = 0$

得特征根：$r_1 = -3$，$r_2 = 2$

步骤3：写出通解

因为是两个不相等的实根，所以通解为：

$$y = C_1 e^{-3x} + C_2 e^{2x}$$

习题2 较难

求解微分方程：$y''' - 6y'' + 11y' - 6y = 0$

解题步骤：

步骤1：写出特征方程

$$r^3 - 6r^2 + 11r - 6 = 0$$

步骤2：解特征方程

尝试 $r = 1$：$1 - 6 + 11 - 6 = 0$，所以 $r = 1$ 是一个根

进行多项式除法：$r^3 - 6r^2 + 11r - 6 = (r - 1)(r^2 - 5r + 6)$

继续分解：$(r - 1)(r - 2)(r - 3) = 0$

得特征根：$r_1 = 1$，$r_2 = 2$，$r_3 = 3$

步骤3：写出通解

因为是三个不相等的实根，所以通解为：

$$y = C_1 e^{x} + C_2 e^{2x} + C_3 e^{3x}$$

习题3 很难

求解初值问题：$y'' - 2y' + 2y = 0$，$y(0) = 1$，$y'(0) = 3$

解题步骤：

步骤1：写出特征方程

$$r^2 - 2r + 2 = 0$$

步骤2：解特征方程

使用求根公式：

$$r = \frac{2 \pm \sqrt{4 - 8}}{2} = \frac{2 \pm \sqrt{-4}}{2} = \frac{2 \pm 2i}{2} = 1 \pm i$$

步骤3：确定通解形式

$\alpha = 1$，$\beta = 1$，所以通解为：

$$y = e^{x}(C_1 \cos x + C_2 \sin x)$$

步骤4：利用初始条件

由 $y(0) = 1$：

$$e^{0}(C_1 \cos 0 + C_2 \sin 0) = C_1 = 1$$

求导数：$y' = e^{x}(C_1 \cos x + C_2 \sin x) + e^{x}(-C_1 \sin x + C_2 \cos x)$

$y' = e^{x}[(C_1 + C_2)\cos x + (C_2 - C_1)\sin x]$

由 $y'(0) = 3$：

$$e^{0}[(C_1 + C_2)\cos 0 + (C_2 - C_1)\sin 0] = C_1 + C_2 = 3$$

步骤5：求解常数

由 $C_1 = 1$ 和 $C_1 + C_2 = 3$，得 $C_2 = 2$

步骤6：特解

$$y = e^{x}(\cos x + 2\sin x)$$
