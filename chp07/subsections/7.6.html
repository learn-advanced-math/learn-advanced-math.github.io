7.6 高阶线性微分方程

Higher-Order Linear Differential Equations

7.6.1 二阶线性微分方程的基本概念

二阶线性微分方程的一般形式

二阶线性微分方程的一般形式为：

$$y'' + P(x)y' + Q(x)y = f(x)$$

其中：

  • $P(x)$、$Q(x)$ 称为系数函数
  • $f(x)$ 称为自由项或非齐次项
  • 当 $f(x) = 0$ 时，称为齐次线性微分方程
  • 当 $f(x) \neq 0$ 时，称为非齐次线性微分方程

物理背景：弹簧振动系统

考虑一个质量为 $m$ 的物体悬挂在弹性系数为 $k$ 的弹簧上，受到阻尼系数为 $c$ 的阻力和外力 $F(t)$ 的作用。

根据牛顿第二定律，可以建立微分方程：

$$m\frac{d^2x}{dt^2} + c\frac{dx}{dt} + kx = F(t)$$

除以 $m$ 得到标准形式：

$$\frac{d^2x}{dt^2} + \frac{c}{m}\frac{dx}{dt} + \frac{k}{m}x = \frac{F(t)}{m}$$

这正是二阶线性微分方程的形式，其中：

  • $x(t)$ 表示物体偏离平衡位置的位移
  • $\frac{c}{m}$ 是阻尼系数
  • $\frac{k}{m}$ 是弹性系数
  • $\frac{F(t)}{m}$ 是外力项

$n$ 阶线性微分方程的一般形式

$n$ 阶线性微分方程的一般形式为：

$$y^{(n)} + P_1(x)y^{(n-1)} + P_2(x)y^{(n-2)} + \cdots + P_{n-1}(x)y' + P_n(x)y = f(x)$$

其中 $y^{(k)}$ 表示 $y$ 的 $k$ 阶导数。

7.6.2 线性微分方程解的结构

齐次线性微分方程解的性质

对于齐次线性微分方程 $y'' + P(x)y' + Q(x)y = 0$：

  1. 叠加原理：如果 $y_1$ 和 $y_2$ 是方程的解，则 $C_1y_1 + C_2y_2$ 也是方程的解
  2. 解空间：所有解构成一个二维线性空间
  3. 基本解组：两个线性无关的解 $y_1$、$y_2$ 构成基本解组
  4. 通解：$y = C_1y_1 + C_2y_2$（$C_1$、$C_2$ 为任意常数）

Wronskian行列式（朗斯基行列式）

对于两个函数 $y_1$ 和 $y_2$，定义Wronskian行列式为：

$$W(y_1, y_2) = \begin{vmatrix} y_1 & y_2 \\ y_1' & y_2' \end{vmatrix} = y_1y_2' - y_1'y_2$$

线性无关的判定：

  • 如果在某区间内 $W(y_1, y_2) \neq 0$，则 $y_1$ 和 $y_2$ 线性无关
  • 如果 $y_1$ 和 $y_2$ 是齐次方程的解且线性相关，则 $W(y_1, y_2) \equiv 0$

非齐次线性微分方程解的结构

对于非齐次线性微分方程 $y'' + P(x)y' + Q(x)y = f(x)$：

$$\text{通解} = \text{齐次方程的通解} + \text{非齐次方程的一个特解}$$

即：$y = y_h + y_p$，其中：

  • $y_h = C_1y_1 + C_2y_2$ 是对应齐次方程的通解
  • $y_p$ 是非齐次方程的任意一个特解

重要性质：两个非齐次方程的解的差是对应齐次方程的解。

7.6.3 常数变易法求解高阶线性微分方程

常数变易法的基本思想

设齐次方程 $y'' + P(x)y' + Q(x)y = 0$ 的通解为：

$$y_h = C_1y_1 + C_2y_2$$

对于非齐次方程，假设特解的形式为：

$$y_p = u_1(x)y_1 + u_2(x)y_2$$

其中 $u_1(x)$ 和 $u_2(x)$ 是待定函数，这就是"常数变易"的含义。

常数变易法的求解步骤
步骤1：求出齐次方程 $y'' + P(x)y' + Q(x)y = 0$ 的通解 $y_h = C_1y_1 + C_2y_2$
步骤2：设非齐次方程 $y'' + P(x)y' + Q(x)y = f(x)$ 的特解为 $y_p = u_1y_1 + u_2y_2$
步骤3：建立约束条件：$u_1'y_1 + u_2'y_2 = 0$ (为了使 $y_p'$ 的二阶导数不含有 $u_1', u_2'$)
步骤4：计算 $y_p'$ 和 $y_p''$，代入原非齐次方程
步骤5：得到方程组： $$\begin{cases} u_1'y_1 + u_2'y_2 = 0 \\ u_1'y_1' + u_2'y_2' = f(x) \end{cases}$$
步骤6：解方程组求出 $u_1'$ 和 $u_2'$，然后积分得 $u_1$ 和 $u_2$

Cramer法则求解

由方程组可以用Cramer法则求解：

$$u_1' = -\frac{y_2 f(x)}{W(y_1, y_2)}, \quad u_2' = \frac{y_1 f(x)}{W(y_1, y_2)}$$

其中 $W(y_1, y_2) = y_1y_2' - y_1'y_2$ 是Wronskian行列式。

积分得到：

$$u_1 = -\int \frac{y_2 f(x)}{W(y_1, y_2)} dx, \quad u_2 = \int \frac{y_1 f(x)}{W(y_1, y_2)} dx$$

7.6.4 典型例题解析

例题1：验证线性无关性

验证 $y_1 = e^x$ 和 $y_2 = e^{2x}$ 在 $(-\infty, +\infty)$ 上线性无关。

解题过程：

方法：计算Wronskian行列式

$$W(y_1, y_2) = \begin{vmatrix} e^x & e^{2x} \\ e^x & 2e^{2x} \end{vmatrix}$$ $$= e^x \cdot 2e^{2x} - e^{2x} \cdot e^x = 2e^{3x} - e^{3x} = e^{3x}$$

由于 $e^{3x} > 0$ 对所有 $x \in (-\infty, +\infty)$ 成立，所以 $W(y_1, y_2) \neq 0$。

结论：$y_1 = e^x$ 和 $y_2 = e^{2x}$ 在 $(-\infty, +\infty)$ 上线性无关。

例题2：用常数变易法求特解

已知齐次方程 $y'' - 3y' + 2y = 0$ 的通解为 $y_h = C_1e^x + C_2e^{2x}$，用常数变易法求非齐次方程 $y'' - 3y' + 2y = e^x$ 的通解。

解题过程：

步骤1：确定基本解组

$y_1 = e^x$，$y_2 = e^{2x}$

步骤2：计算Wronskian行列式

$$W(y_1, y_2) = \begin{vmatrix} e^x & e^{2x} \\ e^x & 2e^{2x} \end{vmatrix} = e^{3x}$$

步骤3：应用常数变易法公式

对于 $f(x) = e^x$：

$$u_1' = -\frac{y_2 f(x)}{W} = -\frac{e^{2x} \cdot e^x}{e^{3x}} = -1$$ $$u_2' = \frac{y_1 f(x)}{W} = \frac{e^x \cdot e^x}{e^{3x}} = \frac{1}{e^x}$$

步骤4：积分求 $u_1$ 和 $u_2$

$$u_1 = \int (-1) dx = -x$$ $$u_2 = \int e^{-x} dx = -e^{-x}$$

步骤5：得到特解

$$y_p = u_1y_1 + u_2y_2 = (-x)e^x + (-e^{-x})e^{2x} = -xe^x - e^x$$

化简：$y_p = -xe^x - e^x = -e^x(x + 1)$

步骤6：写出通解

$$y = y_h + y_p = C_1e^x + C_2e^{2x} - e^x(x + 1)$$ $$= (C_1 - x - 1)e^x + C_2e^{2x}$$

重新命名常数：$y = C_1e^x + C_2e^{2x} - xe^x$

例题3：弹簧振动问题

质量为 $2$ kg的物体悬挂在弹性系数为 $8$ N/m的弹簧上，阻尼系数为 $4$ N·s/m。如果初始时刻物体从平衡位置向下 $0.1$ m处以向上的初速度 $0.2$ m/s开始运动，求运动方程。

解题过程：

步骤1：建立微分方程

根据牛顿第二定律：$m\frac{d^2x}{dt^2} + c\frac{dx}{dt} + kx = 0$

代入数值：$2\frac{d^2x}{dt^2} + 4\frac{dx}{dt} + 8x = 0$

化简：$\frac{d^2x}{dt^2} + 2\frac{dx}{dt} + 4x = 0$

步骤2：求解特征方程

$r^2 + 2r + 4 = 0$

$r = \frac{-2 \pm \sqrt{4 - 16}}{2} = \frac{-2 \pm 2\sqrt{3}i}{2} = -1 \pm \sqrt{3}i$

步骤3：写出通解

$\alpha = -1$，$\beta = \sqrt{3}$

$x(t) = e^{-t}(C_1\cos(\sqrt{3}t) + C_2\sin(\sqrt{3}t))$

步骤4：利用初始条件

初始条件：$x(0) = 0.1$，$x'(0) = -0.2$（向上为负）

由 $x(0) = 0.1$：$C_1 = 0.1$

$x'(t) = -e^{-t}(C_1\cos(\sqrt{3}t) + C_2\sin(\sqrt{3}t)) + e^{-t}(-C_1\sqrt{3}\sin(\sqrt{3}t) + C_2\sqrt{3}\cos(\sqrt{3}t))$

$x'(0) = -C_1 + C_2\sqrt{3} = -0.1 + C_2\sqrt{3} = -0.2$

$C_2\sqrt{3} = -0.1$，$C_2 = -\frac{0.1}{\sqrt{3}} = -\frac{\sqrt{3}}{30}$

步骤5：最终解

$$x(t) = e^{-t}\left(0.1\cos(\sqrt{3}t) - \frac{\sqrt{3}}{30}\sin(\sqrt{3}t)\right)$$

这表示一个衰减振动，物体最终会回到平衡位置。

高阶线性微分方程记忆要点

  • 解的结构：齐次通解 + 非齐次特解
  • 线性无关：Wronskian行列式不为零

7.6.6 练习题

习题1 中等难度

验证 $y_1 = \cos x$ 和 $y_2 = \sin x$ 是方程 $y'' + y = 0$ 的线性无关解，并写出通解。

解题步骤：

步骤1：验证是解

对于 $y_1 = \cos x$：$y_1' = -\sin x$，$y_1'' = -\cos x$

代入方程：$-\cos x + \cos x = 0$ ✓

对于 $y_2 = \sin x$：$y_2' = \cos x$，$y_2'' = -\sin x$

代入方程：$-\sin x + \sin x = 0$ ✓

步骤2：验证线性无关

$$W(y_1, y_2) = \begin{vmatrix} \cos x & \sin x \\ -\sin x & \cos x \end{vmatrix}$$ $$= \cos^2 x + \sin^2 x = 1 \neq 0$$

步骤3：写出通解

$$y = C_1\cos x + C_2\sin x$$

习题2 较难

用常数变易法求解：$y'' - 4y' + 4y = \frac{e^{2x}}{x}$

解题步骤：

步骤1：求齐次方程的通解

特征方程：$r^2 - 4r + 4 = 0$，$(r-2)^2 = 0$

重根：$r_1 = r_2 = 2$

齐次通解：$y_h = (C_1 + C_2x)e^{2x}$

基本解组：$y_1 = e^{2x}$，$y_2 = xe^{2x}$

步骤2：计算Wronskian行列式

$$W = \begin{vmatrix} e^{2x} & xe^{2x} \\ 2e^{2x} & e^{2x} + 2xe^{2x} \end{vmatrix}$$ $$= e^{2x}(e^{2x} + 2xe^{2x}) - xe^{2x} \cdot 2e^{2x} = e^{4x}$$

步骤3：应用常数变易法

$$u_1' = -\frac{y_2 f(x)}{W} = -\frac{xe^{2x} \cdot \frac{e^{2x}}{x}}{e^{4x}} = -1$$ $$u_2' = \frac{y_1 f(x)}{W} = \frac{e^{2x} \cdot \frac{e^{2x}}{x}}{e^{4x}} = \frac{1}{x}$$

步骤4：积分

$$u_1 = -x, \quad u_2 = \ln|x|$$

步骤5：特解和通解

$$y_p = (-x)e^{2x} + (\ln|x|)(xe^{2x}) = xe^{2x}(\ln|x| - 1)$$ $$y = (C_1 + C_2x)e^{2x} + xe^{2x}(\ln|x| - 1)$$

习题3 很难

一个质量为 $m$ 的物体在重力场中做阻尼运动，阻尼力与速度成正比，比例系数为 $k$。如果物体从高度 $h$ 处以初速度 $v_0$ 向下抛出，建立并求解运动方程。

解题步骤：

步骤1：建立微分方程

受力分析：重力 $mg$（向下），阻尼力 $kv$（与速度反向）

牛顿第二定律：$m\frac{dv}{dt} = mg - kv$

整理：$\frac{dv}{dt} + \frac{k}{m}v = g$

步骤2：这是一阶线性微分方程

通解公式：$v(t) = e^{-\frac{k}{m}t}\left(\int g \cdot e^{\frac{k}{m}t} dt + C\right)$

$= e^{-\frac{k}{m}t}\left(\frac{mg}{k}e^{\frac{k}{m}t} + C\right)$

$= \frac{mg}{k} + Ce^{-\frac{k}{m}t}$

步骤3：利用初始条件

$v(0) = v_0$：$v_0 = \frac{mg}{k} + C$

$C = v_0 - \frac{mg}{k}$

速度函数：$v(t) = \frac{mg}{k} + \left(v_0 - \frac{mg}{k}\right)e^{-\frac{k}{m}t}$

步骤4：求位移

$\frac{dx}{dt} = v(t)$，积分得：

$$x(t) = \int v(t) dt = \frac{mg}{k}t - \frac{m}{k}\left(v_0 - \frac{mg}{k}\right)e^{-\frac{k}{m}t} + C_1$$

由 $x(0) = 0$（以抛出点为原点）：

$0 = -\frac{m}{k}\left(v_0 - \frac{mg}{k}\right) + C_1$

$C_1 = \frac{m}{k}\left(v_0 - \frac{mg}{k}\right)$

步骤5：最终的运动方程

$$x(t) = \frac{mg}{k}t + \frac{m}{k}\left(v_0 - \frac{mg}{k}\right)\left(1 - e^{-\frac{k}{m}t}\right)$$

当 $t \to \infty$ 时，物体趋向于匀速运动，终端速度为 $\frac{mg}{k}$。
