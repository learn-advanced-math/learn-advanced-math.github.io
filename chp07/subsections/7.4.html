7.4 一阶线性微分方程

First-Order Linear Differential Equations

7.4.1 一阶线性微分方程的概念

一阶线性微分方程的标准形式

形如：

$$\frac{dy}{dx} + P(x)y = Q(x)$$

的微分方程称为一阶线性微分方程，其中 $P(x)$ 和 $Q(x)$ 是 $x$ 的连续函数。

齐次与非齐次一阶线性微分方程

  • 齐次一阶线性微分方程：当 $Q(x) = 0$ 时，方程为： $$\frac{dy}{dx} + P(x)y = 0$$
  • 非齐次一阶线性微分方程：当 $Q(x) \neq 0$ 时，方程为： $$\frac{dy}{dx} + P(x)y = Q(x)$$

重要性质：一阶线性微分方程的显著特点是未知函数 $y$ 及其导数 $\frac{dy}{dx}$ 都是一次的，且系数只依赖于自变量 $x$。

7.4.2 齐次一阶线性微分方程的解法

齐次方程的通解

对于齐次一阶线性微分方程：

$$\frac{dy}{dx} + P(x)y = 0$$

其通解为：

$$y = Ce^{-\int P(x)dx}$$

其中 $C$ 是任意常数。

齐次方程求解步骤
步骤1：将方程化为标准形式 $\frac{dy}{dx} + P(x)y = 0$
步骤2：分离变量：$\frac{dy}{y} = -P(x)dx$
步骤3：两边积分：$\int \frac{dy}{y} = -\int P(x)dx$
步骤4：得到：$\ln|y| = -\int P(x)dx + C_1$
步骤5：求出通解：$y = Ce^{-\int P(x)dx}$

例题1：齐次一阶线性微分方程

求解微分方程：$\frac{dy}{dx} - 3y = 0$

解题过程：

步骤1：识别方程类型

这是齐次一阶线性微分方程，$P(x) = -3$，$Q(x) = 0$

步骤2：分离变量

$$\frac{dy}{dx} = 3y$$ $$\frac{dy}{y} = 3dx$$

步骤3：两边积分

$$\int \frac{dy}{y} = \int 3dx$$ $$\ln|y| = 3x + C_1$$

步骤4：求通解

$$y = Ce^{3x} \quad (C = \pm e^{C_1})$$

验证：使用公式 $y = Ce^{-\int P(x)dx} = Ce^{-\int(-3)dx} = Ce^{3x}$ ✓

7.4.3 常数变易法求解非齐次方程

常数变易法的基本思想

对于非齐次一阶线性微分方程 $\frac{dy}{dx} + P(x)y = Q(x)$：

  1. 先求对应齐次方程的通解：$y_0 = Ce^{-\int P(x)dx}$
  2. 将常数 $C$ 替换为待定函数 $u(x)$：$y = u(x)e^{-\int P(x)dx}$
  3. 代入原方程，求出 $u(x)$
  4. 得到非齐次方程的通解

非齐次一阶线性微分方程通解公式

$$y = e^{-\int P(x)dx}\left[\int Q(x)e^{\int P(x)dx}dx + C\right]$$
常数变易法详细步骤
步骤1：求齐次方程 $\frac{dy}{dx} + P(x)y = 0$ 的通解：$y_0 = Ce^{-\int P(x)dx}$
步骤2：设非齐次方程的解为：$y = u(x)e^{-\int P(x)dx}$
步骤3：求导：$\frac{dy}{dx} = u'(x)e^{-\int P(x)dx} - u(x)P(x)e^{-\int P(x)dx}$
步骤4：代入原方程，化简得：$u'(x) = Q(x)e^{\int P(x)dx}$
步骤5：积分求出：$u(x) = \int Q(x)e^{\int P(x)dx}dx + C$
步骤6：得到通解：$y = e^{-\int P(x)dx}\left[\int Q(x)e^{\int P(x)dx}dx + C\right]$

例题2：用常数变易法求解非齐次方程

求解微分方程：$\frac{dy}{dx} + y = x$

解题过程：

步骤1：识别参数

$P(x) = 1$，$Q(x) = x$

步骤2：求齐次方程的通解

齐次方程：$\frac{dy}{dx} + y = 0$

通解：$y_0 = Ce^{-\int 1 dx} = Ce^{-x}$

步骤3：应用常数变易法

设 $y = u(x)e^{-x}$，则：

$$\frac{dy}{dx} = u'(x)e^{-x} - u(x)e^{-x}$$

步骤4：代入原方程

$$u'(x)e^{-x} - u(x)e^{-x} + u(x)e^{-x} = x$$ $$u'(x)e^{-x} = x$$ $$u'(x) = xe^x$$

步骤5：求 $u(x)$

$$u(x) = \int xe^x dx$$

使用分部积分：设 $v = x$，$dw = e^x dx$

$$u(x) = xe^x - \int e^x dx = xe^x - e^x + C = (x-1)e^x + C$$

步骤6：得到通解

$$y = u(x)e^{-x} = [(x-1)e^x + C]e^{-x} = (x-1) + Ce^{-x}$$

验证：使用通解公式

$$\int P(x)dx = \int 1 dx = x$$ $$y = e^{-x}\left[\int xe^x dx + C\right] = e^{-x}[(x-1)e^x + C] = (x-1) + Ce^{-x}$$ ✓

7.4.4 伯努利方程

伯努利方程的标准形式

形如：

$$\frac{dy}{dx} + P(x)y = Q(x)y^n \quad (n \neq 0, 1)$$

的微分方程称为伯努利方程，其中 $P(x)$ 和 $Q(x)$ 是 $x$ 的连续函数。

伯努利方程的解法

通过变量替换 $z = y^{1-n}$，可将伯努利方程化为一阶线性方程：

$$\frac{dz}{dx} + (1-n)P(x)z = (1-n)Q(x)$$
伯努利方程求解步骤
步骤1：将方程化为标准形式 $\frac{dy}{dx} + P(x)y = Q(x)y^n$
步骤2：两边同除以 $y^n$：$y^{-n}\frac{dy}{dx} + P(x)y^{1-n} = Q(x)$
步骤3：令 $z = y^{1-n}$，则 $\frac{dz}{dx} = (1-n)y^{-n}\frac{dy}{dx}$
步骤4：代入得线性方程：$\frac{dz}{dx} + (1-n)P(x)z = (1-n)Q(x)$
步骤5：用常数变易法求解关于 $z$ 的一阶线性方程
步骤6：用 $y^{1-n} = z$ 反解出 $y$

例题3：伯努利方程

求解微分方程：$\frac{dy}{dx} + y = xy^2$

解题过程：

步骤1：识别伯努利方程

$P(x) = 1$，$Q(x) = x$，$n = 2$

步骤2：两边除以 $y^2$

$$y^{-2}\frac{dy}{dx} + y^{-1} = x$$

步骤3：变量替换

令 $z = y^{1-2} = y^{-1}$，则 $\frac{dz}{dx} = -y^{-2}\frac{dy}{dx}$

所以 $y^{-2}\frac{dy}{dx} = -\frac{dz}{dx}$

步骤4：代入得线性方程

$$-\frac{dz}{dx} + z = x$$ $$\frac{dz}{dx} - z = -x$$

步骤5：求解关于 $z$ 的一阶线性方程

这里 $P(x) = -1$，$Q(x) = -x$

齐次方程的解：$z_0 = Ce^{\int 1 dx} = Ce^x$

使用通解公式：

$$z = e^x\left[\int (-x)e^{-x}dx + C\right]$$

计算 $\int (-x)e^{-x}dx$：

使用分部积分：$u = -x$，$dv = e^{-x}dx$

$$\int (-x)e^{-x}dx = xe^{-x} - \int e^{-x}dx = xe^{-x} + e^{-x} = (x+1)e^{-x}$$

所以：

$$z = e^x[(x+1)e^{-x} + C] = (x+1) + Ce^x$$

步骤6：反解出 $y$

由于 $z = y^{-1}$，所以：

$$y = \frac{1}{z} = \frac{1}{(x+1) + Ce^x}$$

7.4.6 练习题

习题1 中等难度

求解微分方程：$\frac{dy}{dx} - 2y = 4e^{3x}$

解题步骤：

步骤1：识别参数

$P(x) = -2$，$Q(x) = 4e^{3x}$

步骤2：计算积分因子

$\int P(x)dx = \int (-2)dx = -2x$

积分因子：$e^{\int P(x)dx} = e^{-2x}$

步骤3：应用通解公式

$$y = e^{2x}\left[\int 4e^{3x} \cdot e^{-2x}dx + C\right]$$ $$= e^{2x}\left[\int 4e^x dx + C\right]$$ $$= e^{2x}[4e^x + C]$$ $$= 4e^{3x} + Ce^{2x}$$

答案：$y = 4e^{3x} + Ce^{2x}$

习题2 较难

求解初值问题：$x\frac{dy}{dx} + y = x^2\ln x$，$y(1) = 0$ （$x > 0$）

解题步骤：

步骤1：化为标准形式

$$\frac{dy}{dx} + \frac{1}{x}y = x\ln x$$

$P(x) = \frac{1}{x}$，$Q(x) = x\ln x$

步骤2：计算积分因子

$\int P(x)dx = \int \frac{1}{x}dx = \ln x$

积分因子：$e^{\int P(x)dx} = e^{\ln x} = x$

步骤3：应用通解公式

$$y = \frac{1}{x}\left[\int x\ln x \cdot x dx + C\right]$$ $$= \frac{1}{x}\left[\int x^2\ln x dx + C\right]$$

步骤4：计算 $\int x^2\ln x dx$

使用分部积分：$u = \ln x$，$dv = x^2 dx$

$$\int x^2\ln x dx = \frac{x^3}{3}\ln x - \int \frac{x^3}{3} \cdot \frac{1}{x}dx$$ $$= \frac{x^3}{3}\ln x - \frac{1}{3}\int x^2 dx$$ $$= \frac{x^3}{3}\ln x - \frac{x^3}{9}$$

步骤5：得到通解

$$y = \frac{1}{x}\left[\frac{x^3}{3}\ln x - \frac{x^3}{9} + C\right]$$ $$= \frac{x^2}{3}\ln x - \frac{x^2}{9} + \frac{C}{x}$$

步骤6：利用初始条件

代入 $y(1) = 0$：

$$0 = \frac{1}{3}\ln 1 - \frac{1}{9} + C = 0 - \frac{1}{9} + C$$ $$C = \frac{1}{9}$$

答案：$y = \frac{x^2}{3}\ln x - \frac{x^2}{9} + \frac{1}{9x}$

习题3 很难

求解伯努利方程：$\frac{dy}{dx} + \frac{y}{x} = y^3x^2$

解题步骤：

步骤1：识别伯努利方程

$P(x) = \frac{1}{x}$，$Q(x) = x^2$，$n = 3$

步骤2：两边除以 $y^3$

$$y^{-3}\frac{dy}{dx} + \frac{1}{x}y^{-2} = x^2$$

步骤3：变量替换

令 $z = y^{1-3} = y^{-2}$，则：

$$\frac{dz}{dx} = -2y^{-3}\frac{dy}{dx}$$

所以 $y^{-3}\frac{dy}{dx} = -\frac{1}{2}\frac{dz}{dx}$

步骤4：代入得线性方程

$$-\frac{1}{2}\frac{dz}{dx} + \frac{1}{x}z = x^2$$ $$\frac{dz}{dx} - \frac{2}{x}z = -2x^2$$

步骤5：求解关于 $z$ 的线性方程

$P(x) = -\frac{2}{x}$，$Q(x) = -2x^2$

$\int P(x)dx = -2\ln x = \ln x^{-2}$

积分因子：$e^{\int P(x)dx} = x^{-2}$

$$z = x^2\left[\int (-2x^2) \cdot x^{-2}dx + C\right]$$ $$= x^2\left[\int (-2)dx + C\right]$$ $$= x^2[-2x + C]$$ $$= -2x^3 + Cx^2$$

步骤6：反解出 $y$

由于 $z = y^{-2}$，所以：

$$y^{-2} = -2x^3 + Cx^2$$ $$y^2 = \frac{1}{-2x^3 + Cx^2} = \frac{1}{x^2(C - 2x)}$$ $$y = \pm \frac{1}{x\sqrt{C - 2x}}$$

答案：$y = \pm \frac{1}{x\sqrt{C - 2x}}$ （需 $C - 2x > 0$）
