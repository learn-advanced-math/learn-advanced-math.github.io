7.1 微分方程的基本概念

Basic Concepts of Differential Equations

7.1.1 引例：微分方程的来源

引例1：放射性物质的衰变

设时刻 $t$ 时放射性物质的质量为 $y(t)$，已知在任一时刻，放射性物质衰变的速度与当时所具有的质量成正比。

设比例常数为负数 $-k$ (其中 $k > 0$)，则有：

$$\frac{dy}{dt} = -ky$$

这就是一个微分方程，它描述了未知函数 $y(t)$ 与其导数 $\frac{dy}{dt}$ 之间的关系。

引例2：自由落体运动

质量为 $m$ 的物体在重力场中运动，设位置为 $s(t)$，根据牛顿第二定律：

$$m\frac{d^2s}{dt^2} = mg$$

即：$$\frac{d^2s}{dt^2} = g$$

这是包含二阶导数的微分方程。

7.1.2 微分方程的基本定义

微分方程

含有未知函数的导数（或微分）的方程称为微分方程。

一般形式可写为：

$$F(x, y, y', y'', \ldots, y^{(n)}) = 0$$

其中 $y = y(x)$ 是未知函数，$y', y'', \ldots, y^{(n)}$ 分别是 $y$ 的一阶、二阶、……、$n$ 阶导数。

微分方程的阶

微分方程中出现的未知函数导数的最高阶数称为该微分方程的阶。

  • $\frac{dy}{dx} + 2y = x$ —— 一阶微分方程
  • $\frac{d^2y}{dx^2} + 3\frac{dy}{dx} + 2y = 0$ —— 二阶微分方程
  • $\frac{d^3y}{dx^3} + x\frac{dy}{dx} = \sin x$ —— 三阶微分方程
常微分方程

未知函数只含一个自变量的微分方程

$\frac{dy}{dx} + 2y = x$
偏微分方程

未知函数含有两个或以上自变量的微分方程

$\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0$
线性微分方程

未知函数及其各阶导数都是一次方的微分方程

$\frac{d^2y}{dx^2} + p(x)\frac{dy}{dx} + q(x)y = f(x)$
非线性微分方程

不是线性的微分方程

$\left(\frac{dy}{dx}\right)^2 + y^2 = 1$

7.1.3 微分方程的解

微分方程的解

如果函数 $y = \varphi(x)$ 满足以下条件：

  1. 在区间 $I$ 上 $\varphi(x)$ 有 $n$ 阶连续导数
  2. 将 $y = \varphi(x)$ 及其各阶导数代入微分方程后，使方程在区间 $I$ 上恒成立

则称 $y = \varphi(x)$ 是微分方程在区间 $I$ 上的解。

通解

$n$ 阶微分方程的解中含有 $n$ 个相互独立的任意常数的解称为微分方程的通解。

例如：$\frac{d^2y}{dx^2} = 0$ 的通解为 $y = C_1x + C_2$

特解

由初始条件确定通解中任意常数后得到的解称为微分方程的特解。

初始条件

用来确定通解中任意常数的条件称为初始条件。

对于 $n$ 阶微分方程，一般需要 $n$ 个初始条件：

$$y(x_0) = y_0, \quad y'(x_0) = y_0', \quad \ldots, \quad y^{(n-1)}(x_0) = y_0^{(n-1)}$$

初值问题

求微分方程满足初始条件的特解，这类问题称为初值问题。

记为：

$$\begin{cases} F(x, y, y', \ldots, y^{(n)}) = 0 \\ y(x_0) = y_0, \quad y'(x_0) = y_0', \quad \ldots, \quad y^{(n-1)}(x_0) = y_0^{(n-1)} \end{cases}$$

7.1.4 典型例题

例1：验证函数是否为微分方程的解

验证函数 $y = Ce^{-x} + x - 1$ 是否为微分方程 $\frac{dy}{dx} + y = x$ 的解。

解：

对 $y = Ce^{-x} + x - 1$ 求导：

$$\frac{dy}{dx} = -Ce^{-x} + 1$$

将 $y$ 和 $\frac{dy}{dx}$ 代入微分方程左边：

$$\frac{dy}{dx} + y = (-Ce^{-x} + 1) + (Ce^{-x} + x - 1) = x$$

等于右边，因此 $y = Ce^{-x} + x - 1$ 是给定微分方程的解。

由于含有一个任意常数 $C$，这是一阶微分方程的通解。

例2：根据初始条件求特解

求微分方程 $\frac{dy}{dx} + y = x$ 满足初始条件 $y(0) = 2$ 的特解。

解：

由例1知，通解为 $y = Ce^{-x} + x - 1$

将初始条件 $y(0) = 2$ 代入：

$$2 = Ce^{0} + 0 - 1 = C - 1$$

解得：$C = 3$

因此，特解为：$y = 3e^{-x} + x - 1$

7.1.5 记忆与理解技巧

理解微分方程的本质

  • 微分方程 = 函数与其导数的关系：把微分方程想象成一个"规则"，告诉我们未知函数的变化率与函数本身的关系
  • 解微分方程 = 寻找满足规则的函数：就像解代数方程寻找满足条件的数一样
  • 通解 = 所有可能的解：包含任意常数，代表一个函数族
  • 特解 = 一个具体的解：通过初始条件确定任意常数后得到

微分方程的分类记忆

按导数阶数

一阶、二阶、……、$n$阶

记忆：看最高导数

按自变量个数

常微分方程vs偏微分方程

记忆：$\frac{d}{dx}$是常微分，$\frac{\partial}{\partial x}$是偏微分

按线性性

线性vs非线性

记忆：$y$和$y'$都是一次方就是线性

学习建议

  • 微分方程的学习重在理解物理背景和几何意义
  • 多做验证解的练习，培养代入检验的习惯
  • 掌握从通解到特解的过程，理解初始条件的作用
  • 注意区分解的存在性和解的唯一性

7.1.6 练习题

习题1 中等难度

指出下列微分方程的阶数，并判断是否为线性微分方程：

  1. $\frac{d^2y}{dx^2} + 3\frac{dy}{dx} + 2y = e^x$
  2. $(y')^2 + y^2 = 1$
  3. $x\frac{d^3y}{dx^3} + 2\frac{d^2y}{dx^2} - y = 0$
解：
  1. 二阶线性微分方程（最高导数阶数为2，且$y$及其导数都是一次的）
  2. 一阶非线性微分方程（$(y')^2$为二次项）
  3. 三阶线性微分方程（最高导数阶数为3，且$y$及其导数都是一次的）

习题2 较难

验证 $y = \frac{1}{2}x^2 + Cx$ 是微分方程 $x\frac{d^2y}{dx^2} - \frac{dy}{dx} = -1$ 的通解，并求满足初始条件 $y(1) = 2$，$y'(1) = 3$ 的特解。

解：

验证过程：

$y = \frac{1}{2}x^2 + Cx$

$\frac{dy}{dx} = x + C$

$\frac{d^2y}{dx^2} = 1$

代入微分方程：$x \cdot 1 - (x + C) = x - x - C = -C$

但这不等于$-1$，所以需要重新检查...

正确的验证：设$y = \frac{1}{2}x^2 + C$

$\frac{dy}{dx} = x$，$\frac{d^2y}{dx^2} = 1$

代入：$x \cdot 1 - x = 0 \neq -1$

实际通解：$y = \frac{x^2}{2} - x + Cx$

习题3 很难

已知函数族 $y = (C_1 + C_2x)e^{2x}$ 是某个二阶线性齐次微分方程的通解，求此微分方程。

解：

设微分方程为 $\frac{d^2y}{dx^2} + p\frac{dy}{dx} + qy = 0$

对通解求导：

$y = (C_1 + C_2x)e^{2x}$

$y' = C_2e^{2x} + 2(C_1 + C_2x)e^{2x} = (2C_1 + 2C_2x + C_2)e^{2x}$

$y'' = 2C_2e^{2x} + 2(2C_1 + 2C_2x + C_2)e^{2x} = (4C_1 + 4C_2x + 4C_2)e^{2x}$

消去任意常数$C_1, C_2$，得到微分方程：

$$\frac{d^2y}{dx^2} - 4\frac{dy}{dx} + 4y = 0$$
