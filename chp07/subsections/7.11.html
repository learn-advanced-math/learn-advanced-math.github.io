7.11.2 微分方程类型总结表

方程类型 一般形式 求解方法 关键技巧 适用条件
可分离变量 $$\frac{dy}{dx} = f(x)g(y)$$ 分离变量法 $$\frac{dy}{g(y)} = f(x)dx$$ $$g(y) \neq 0$$
齐次方程 $$\frac{dy}{dx} = f\left(\frac{y}{x}\right)$$ 换元法 设 $v = \frac{y}{x}$ $f$ 为 $\frac{y}{x}$ 的函数
一阶线性 $$y' + P(x)y = Q(x)$$ 常数变易法 $$y = u(x) e^{-\int P(x)dx}$$ $P(x), Q(x)$ 连续
伯努利方程 $$y' + P(x)y = Q(x)y^n$$ 换元法 设 $z = y^{1-n}$ $$n \neq 0, 1$$
常系数线性 $$y'' + py' + qy = f(x)$$ 特征方程法 $$r^2 + pr + q = 0$$ $p, q$ 为常数
欧拉方程 $$x^2y'' + pxy' + qy = 0$$ 换元法 设 $x = e^t$ $$x > 0$$

7.11.3 记忆口诀与技巧

微分方程求解歌

分离变量最简单，两边积分把解算；
齐次方程有技巧，设置新元来变形；
通解特解加一起，解锁非齐次方程；
线性方程找因子，常数变易是法宝；
高阶降阶有妙招，特征方程特征值；
欧拉方程要换元，指数换化常系数。
