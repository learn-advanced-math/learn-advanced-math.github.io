7.2 可分离变量的微分方程

Separable Variables Differential Equations

7.2.1 可分离变量微分方程的概念

可分离变量的微分方程

如果一阶微分方程可以写成：

$$\frac{dy}{dx} = f(x)g(y)$$

的形式，其中 $f(x)$ 仅含自变量 $x$，$g(y)$ 仅含未知函数 $y$，则称此微分方程为可分离变量的微分方程。

分离变量法

求解可分离变量微分方程的方法称为分离变量法，其基本思想是将方程中的变量 $x$ 和 $y$ 分别移到等号的两边。

注意：能化为 $\frac{dy}{dx} = f(x)g(y)$ 形式的微分方程都可以用分离变量法求解。

7.2.2 可分离变量微分方程的求解步骤

标准求解步骤
步骤1：将微分方程化为 $\frac{dy}{dx} = f(x)g(y)$ 的形式
步骤2：分离变量，得到 $\frac{dy}{g(y)} = f(x)dx$
步骤3：两边同时积分：$\int \frac{dy}{g(y)} = \int f(x)dx$
步骤4：求出积分，得到通解
步骤5：如有初始条件，代入求出特解

理论依据

分离变量法的理论基础是复合函数的积分法则：

如果 $y = y(x)$，则：

$$\int \frac{1}{g(y)} \frac{dy}{dx} dx = \int \frac{1}{g(y)} dy$$

这是因为：$\frac{d}{dx}\left(\int \frac{1}{g(y)} dy\right) = \frac{1}{g(y)} \frac{dy}{dx}$

7.2.3 典型例题解析

例题1：基础分离变量

求解微分方程：$\frac{dy}{dx} = 2xy$

解题过程：

步骤1：识别方程类型

方程已经是 $\frac{dy}{dx} = f(x)g(y)$ 的形式，其中 $f(x) = 2x$，$g(y) = y$

步骤2：分离变量

当 $y \neq 0$ 时，将变量分离：

$$\frac{dy}{y} = 2x dx$$

步骤3：两边积分

$$\int \frac{dy}{y} = \int 2x dx$$ $$\ln|y| = x^2 + C_1$$

步骤4：求通解

$$|y| = e^{x^2 + C_1} = e^{C_1} \cdot e^{x^2}$$

设 $C = \pm e^{C_1}$，得到：

$$y = Ce^{x^2}$$

步骤5：验证特殊解

当 $y = 0$ 时，代入原方程得 $0 = 0$，所以 $y = 0$ 也是解。

这相当于 $C = 0$ 的情况，因此通解为：

$$y = Ce^{x^2} \quad (C \text{为任意常数})$$

例题2：需要变形的分离变量方程

求解微分方程：$(1 + x^2)dy - xy dx = 0$

解题过程：

步骤1：将方程变形为标准形式

$$(1 + x^2)dy = xy dx$$ $$\frac{dy}{dx} = \frac{xy}{1 + x^2}$$

步骤2：分离变量

当 $y \neq 0$ 时：

$$\frac{dy}{y} = \frac{x dx}{1 + x^2}$$

步骤3：两边积分

$$\int \frac{dy}{y} = \int \frac{x dx}{1 + x^2}$$

左边：$\ln|y|$

右边：设 $u = 1 + x^2$，则 $du = 2x dx$

$$\int \frac{x dx}{1 + x^2} = \frac{1}{2}\int \frac{du}{u} = \frac{1}{2}\ln|u| = \frac{1}{2}\ln(1 + x^2)$$

步骤4：求通解

$$\ln|y| = \frac{1}{2}\ln(1 + x^2) + C_1$$ $$|y| = e^{C_1} \sqrt{1 + x^2}$$ $$y = C\sqrt{1 + x^2} \quad (C \text{为任意常数})$$

例题3：带初始条件的微分方程

求解初值问题：$\frac{dy}{dx} = \frac{y^2}{x^2}$，$y(1) = 1$

解题过程：

步骤1：分离变量

当 $y \neq 0$，$x \neq 0$ 时：

$$\frac{dy}{y^2} = \frac{dx}{x^2}$$

步骤2：两边积分

$$\int y^{-2} dy = \int x^{-2} dx$$ $$-\frac{1}{y} = -\frac{1}{x} + C$$

步骤3：求通解

$$\frac{1}{y} = \frac{1}{x} - C$$ $$y = \frac{x}{1 - Cx}$$

步骤4：利用初始条件求特解

代入 $y(1) = 1$：

$$1 = \frac{1}{1 - C \cdot 1} = \frac{1}{1 - C}$$ $$1 - C = 1 \Rightarrow C = 0$$

步骤5：特解

$$y = \frac{x}{1} = x$$

记忆技巧

  • 识别口诀："左右分家，各自积分"
  • 分离原则：$x$ 的函数放一边，$y$ 的函数放另一边

  • 特殊情况：分母为零的情况需要单独讨论

7.2.5 练习题

习题1 中等难度

求解微分方程：$\frac{dy}{dx} = \frac{x}{y}$

解题步骤：

步骤1：分离变量

当 $y \neq 0$ 时：$y dy = x dx$

步骤2：两边积分

$$\int y dy = \int x dx$$ $$\frac{y^2}{2} = \frac{x^2}{2} + C_1$$

步骤3：整理得通解

$$y^2 = x^2 + C \quad (C = 2C_1)$$

或写成：$y^2 - x^2 = C$

这表示双曲线族。

习题2 较难

求解微分方程：$\frac{dy}{dx} = \frac{1 + y^2}{1 + x^2}$

解题步骤：

步骤1：分离变量

$$\frac{dy}{1 + y^2} = \frac{dx}{1 + x^2}$$

步骤2：两边积分

$$\int \frac{dy}{1 + y^2} = \int \frac{dx}{1 + x^2}$$

步骤3：利用标准积分公式

$\int \frac{dt}{1 + t^2} = \arctan t + C$

$$\arctan y = \arctan x + C$$

步骤4：求通解

$$y = \tan(\arctan x + C)$$

利用三角恒等式：

$$y = \frac{\tan(\arctan x) + \tan C}{1 - \tan(\arctan x)\tan C} = \frac{x + \tan C}{1 - x\tan C}$$

设 $k = \tan C$，则通解为：

$$y = \frac{x + k}{1 - kx}$$

习题3 很难

求解初值问题：$x(1 + y^2)dx + y(1 + x^2)dy = 0$，$y(0) = 1$

解题步骤：

步骤1：重新整理方程

$$x(1 + y^2)dx = -y(1 + x^2)dy$$ $$\frac{x dx}{1 + x^2} = -\frac{y dy}{1 + y^2}$$

步骤2：两边积分

$$\int \frac{x dx}{1 + x^2} = -\int \frac{y dy}{1 + y^2}$$

步骤3：计算积分

左边：设 $u = 1 + x^2$，则 $du = 2x dx$

$$\int \frac{x dx}{1 + x^2} = \frac{1}{2}\ln(1 + x^2)$$

右边：同理

$$-\int \frac{y dy}{1 + y^2} = -\frac{1}{2}\ln(1 + y^2)$$

步骤4：得到通解

$$\frac{1}{2}\ln(1 + x^2) = -\frac{1}{2}\ln(1 + y^2) + C_1$$ $$\ln(1 + x^2) + \ln(1 + y^2) = C$$ $$(1 + x^2)(1 + y^2) = K \quad (K = e^C)$$

步骤5：利用初始条件

代入 $y(0) = 1$：

$$(1 + 0)(1 + 1) = K \Rightarrow K = 2$$

步骤6：特解

$$(1 + x^2)(1 + y^2) = 2$$ $$1 + y^2 = \frac{2}{1 + x^2}$$ $$y^2 = \frac{2}{1 + x^2} - 1 = \frac{1 - x^2}{1 + x^2}$$

由于 $y(0) = 1 > 0$，取正号：

$$y = \sqrt{\frac{1 - x^2}{1 + x^2}} = \frac{\sqrt{1 - x^2}}{\sqrt{1 + x^2}}$$
