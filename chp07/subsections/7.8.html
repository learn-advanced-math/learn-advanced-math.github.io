7.8 常系数非齐次线性微分方程

Non-homogeneous Linear Differential Equations with Constant Coefficients

7.8.1 非齐次线性微分方程的基本概念

二阶常系数非齐次线性微分方程的标准形式

形如：

$$y'' + py' + qy = f(x)$$

的微分方程称为二阶常系数非齐次线性微分方程，其中 $p$、$q$ 是常数，$f(x)$ 是 $x$ 的已知函数（非恒等于零）。

解的结构定理

二阶常系数非齐次线性微分方程的通解由两部分组成：

$$y = y_h + y_p$$

其中：

  • $y_h$ 是对应齐次方程 $y'' + py' + qy = 0$ 的通解（齐次通解）
  • $y_p$ 是非齐次方程的一个特解（特解）
标准求解步骤
步骤1：求出对应齐次方程的通解 $y_h$
步骤2：根据 $f(x)$ 的形式，用待定系数法求特解 $y_p$
步骤3：写出非齐次方程的通解 $y = y_h + y_p$

7.8.2 两种常见形式的非齐次方程

第一种形式
$$f(x) = e^{\alpha x} P_n(x)$$

其中 $P_n(x)$ 是 $x$ 的 $n$ 次多项式，$\alpha$ 是常数。

包括特殊情况：

  • $f(x) = P_n(x)$（当 $\alpha = 0$ 时）
  • $f(x) = e^{\alpha x}$（当 $P_n(x) = 1$ 时）
第二种形式
$$f(x) = e^{\alpha x}[P_n(x)\cos \beta x + Q_m(x)\sin \beta x]$$

其中 $P_n(x)$、$Q_m(x)$ 是多项式，$\alpha$、$\beta$ 是常数，$\beta \neq 0$。

包括特殊情况：

  • $f(x) = P_n(x)\cos \beta x + Q_m(x)\sin \beta x$（当 $\alpha = 0$ 时）

7.8.3 第一种形式：$f(x) = e^{\alpha x} P_n(x)$ 的求解

特解的设定方法

根据 $\alpha$ 是否为特征方程的根，特解形式有所不同：

情况 条件 特解形式
情况1 $\alpha$ 不是特征根 $y_p = e^{\alpha x} Q_n(x)$
情况2 $\alpha$ 是单重特征根 $y_p = x e^{\alpha x} Q_n(x)$
情况3 $\alpha$ 是二重特征根 $y_p = x^2 e^{\alpha x} Q_n(x)$

注意：$Q_n(x)$ 是与 $P_n(x)$ 同次的待定系数多项式。

例题1：$\alpha$ 不是特征根

求解微分方程：$y'' - 3y' + 2y = 2e^{3x}$

解题过程：

步骤1：求齐次通解

特征方程：$r^2 - 3r + 2 = 0$，解得 $r_1 = 1$，$r_2 = 2$

齐次通解：$y_h = C_1 e^x + C_2 e^{2x}$

步骤2：设特解

由于 $f(x) = 2e^{3x}$，$\alpha = 3$，而 $3$ 不是特征根

设特解：$y_p = A e^{3x}$

步骤3：求待定系数

$y_p' = 3A e^{3x}$，$y_p'' = 9A e^{3x}$

代入原方程：$9A e^{3x} - 3 \cdot 3A e^{3x} + 2A e^{3x} = 2e^{3x}$

化简：$2A e^{3x} = 2e^{3x}$，所以 $A = 1$

特解：$y_p = e^{3x}$

步骤4：写出通解

$$y = C_1 e^x + C_2 e^{2x} + e^{3x}$$

例题2：$\alpha$ 是单重特征根

求解微分方程：$y'' - 3y' + 2y = e^x$

解题过程：

步骤1：齐次通解已知

$y_h = C_1 e^x + C_2 e^{2x}$

步骤2：设特解

由于 $f(x) = e^x$，$\alpha = 1$，而 $1$ 是单重特征根

设特解：$y_p = Ax e^x$

步骤3：求待定系数

$y_p' = A e^x + Ax e^x = A e^x(1 + x)$

$y_p'' = A e^x(1 + x) + A e^x = A e^x(2 + x)$

代入原方程并化简：$-A e^x = e^x$，所以 $A = -1$

特解：$y_p = -x e^x$

步骤4：写出通解

$$y = C_1 e^x + C_2 e^{2x} - x e^x$$

7.8.4 第二种形式：三角函数型的求解

特解的设定方法

对于 $f(x) = e^{\alpha x}[P_n(x)\cos \beta x + Q_m(x)\sin \beta x]$：

情况 条件 特解形式
情况1 $\alpha \pm \beta i$ 不是特征根 $y_p = e^{\alpha x}[R_l(x)\cos \beta x + S_l(x)\sin \beta x]$
情况2 $\alpha \pm \beta i$ 是特征根 $y_p = x e^{\alpha x}[R_l(x)\cos \beta x + S_l(x)\sin \beta x]$

注意：$l = \max(n, m)$，且 $R_l(x)$、$S_l(x)$ 是 $l$ 次待定系数多项式。

例题3：三角函数型

求解微分方程：$y'' + y = \cos x$

解题过程：

步骤1：求齐次通解

特征方程：$r^2 + 1 = 0$，解得 $r_{1,2} = \pm i$

齐次通解：$y_h = C_1 \cos x + C_2 \sin x$

步骤2：分析特解形式

$f(x) = \cos x$，相当于 $\alpha = 0$，$\beta = 1$

由于 $0 \pm i$ 是特征根，设特解：$y_p = x(A \cos x + B \sin x)$

步骤3：求导并代入

$y_p' = A \cos x + B \sin x + x(-A \sin x + B \cos x)$

$y_p'' = -2A \sin x + 2B \cos x - x(A \cos x + B \sin x)$

代入原方程：$-2A \sin x + 2B \cos x = \cos x$

步骤4：比较系数

$\cos x$ 系数：$2B = 1 \Rightarrow B = \frac{1}{2}$

$\sin x$ 系数：$-2A = 0 \Rightarrow A = 0$

特解：$y_p = \frac{x}{2} \sin x$

步骤5：写出通解

$$y = C_1 \cos x + C_2 \sin x + \frac{x}{2} \sin x$$

7.8.6 练习题

习题1 中等难度

求解微分方程：$y'' - 4y = 8x^2$

解题步骤：

步骤1：求齐次通解

特征方程：$r^2 - 4 = 0$，解得 $r_1 = 2$，$r_2 = -2$

齐次通解：$y_h = C_1 e^{2x} + C_2 e^{-2x}$

步骤2：设特解

$f(x) = 8x^2$，$\alpha = 0$ 不是特征根

设特解：$y_p = ax^2 + bx + c$

步骤3：求待定系数

$y_p' = 2ax + b$，$y_p'' = 2a$

代入原方程：$2a - 4(ax^2 + bx + c) = 8x^2$

比较系数：$-4a = 8$，$-4b = 0$，$2a - 4c = 0$

解得：$a = -2$，$b = 0$，$c = -1$

特解：$y_p = -2x^2 - 1$

步骤4：通解

$$y = C_1 e^{2x} + C_2 e^{-2x} - 2x^2 - 1$$

习题2 较难

求解微分方程：$y'' + 4y = 3\sin 2x$

解题步骤：

步骤1：求齐次通解

特征方程：$r^2 + 4 = 0$，解得 $r_{1,2} = \pm 2i$

齐次通解：$y_h = C_1 \cos 2x + C_2 \sin 2x$

步骤2：分析特解形式

$f(x) = 3\sin 2x$，$\alpha = 0$，$\beta = 2$

由于 $0 \pm 2i$ 是特征根，设特解：$y_p = x(A \cos 2x + B \sin 2x)$

步骤3：求导

$y_p' = A \cos 2x + B \sin 2x + x(-2A \sin 2x + 2B \cos 2x)$

$y_p'' = -4A \sin 2x + 4B \cos 2x - 4x(A \cos 2x + B \sin 2x)$

步骤4：代入方程

$-4A \sin 2x + 4B \cos 2x = 3\sin 2x$

比较系数：$-4A = 3$，$4B = 0$

解得：$A = -\frac{3}{4}$，$B = 0$

特解：$y_p = -\frac{3x}{4} \cos 2x$

步骤5：通解

$$y = C_1 \cos 2x + C_2 \sin 2x - \frac{3x}{4} \cos 2x$$

习题3 很难

求解微分方程：$y'' - 2y' + y = xe^x$

解题步骤：

步骤1：求齐次通解

特征方程：$r^2 - 2r + 1 = 0$，即 $(r-1)^2 = 0$

二重根：$r_{1,2} = 1$

齐次通解：$y_h = (C_1 + C_2 x) e^x$

步骤2：分析特解形式

$f(x) = xe^x$，$\alpha = 1$ 是二重特征根

设特解：$y_p = x^2(ax + b) e^x = (ax^3 + bx^2) e^x$

步骤3：求导

$y_p' = (ax^3 + bx^2) e^x + (3ax^2 + 2bx) e^x = [ax^3 + (b+3a)x^2 + 2bx] e^x$

$y_p'' = [ax^3 + (b+6a)x^2 + (2b+6a)x + 2b] e^x$

步骤4：代入方程并化简

经计算后得到：$6ax + 2b = x$

比较系数：$6a = 1$，$2b = 0$

解得：$a = \frac{1}{6}$，$b = 0$

特解：$y_p = \frac{x^3}{6} e^x$

步骤5：通解

$$y = (C_1 + C_2 x) e^x + \frac{x^3}{6} e^x$$
