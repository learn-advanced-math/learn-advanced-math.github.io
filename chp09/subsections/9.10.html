Method of Least Squares

9.10.1 最小二乘法介绍

最小二乘法的基本思想

最小二乘法（Method of Least Squares）是一种数学优化技术，通过最小化误差的平方和来寻找数据的最佳函数匹配。它是由德国数学家高斯和法国数学家勒让德在$19$世纪初发展起来的。

核心思想：在所有可能的拟合函数中，选择使得观测值与拟合值之间的残差平方和最小的函数作为最优拟合函数。

问题的数学描述

给定 $n$ 个数据点 $(x_1, y_1), (x_2, y_2), \ldots, (x_n, y_n)$，我们希望找到一个函数 $f(x)$ 来拟合这些数据。

残差（误差）：对于第 $i$ 个数据点，残差定义为：

$$e_i = y_i - f(x_i)$$

目标函数：最小二乘法要求最小化残差平方和：

$$S = \sum_{i=1}^{n} e_i^2 = \sum_{i=1}^{n} [y_i - f(x_i)]^2$$

线性最小二乘法

最常见的情况是线性回归，即假设拟合函数为线性函数：

$$f(x) = ax + b$$

此时目标函数变为：

$$S(a, b) = \sum_{i=1}^{n} (y_i - ax_i - b)^2$$

为了找到最优的参数 $a$ 和 $b$，我们对 $S$ 分别关于 $a$ 和 $b$ 求偏导数，并令其等于零：

正规方程组
$$\begin{cases} \frac{\partial S}{\partial a} = -2\sum_{i=1}^{n} x_i(y_i - ax_i - b) = 0 \\ \frac{\partial S}{\partial b} = -2\sum_{i=1}^{n} (y_i - ax_i - b) = 0 \end{cases}$$

整理得到：

$$\begin{cases} a\sum_{i=1}^{n} x_i^2 + b\sum_{i=1}^{n} x_i = \sum_{i=1}^{n} x_i y_i \\ a\sum_{i=1}^{n} x_i + nb = \sum_{i=1}^{n} y_i \end{cases}$$

线性回归参数的解析解

解上述正规方程组，可以得到参数的解析表达式：

$$a = \frac{n\sum_{i=1}^{n} x_i y_i - \sum_{i=1}^{n} x_i \sum_{i=1}^{n} y_i}{n\sum_{i=1}^{n} x_i^2 - \left(\sum_{i=1}^{n} x_i\right)^2}$$
$$b = \frac{\sum_{i=1}^{n} y_i - a\sum_{i=1}^{n} x_i}{n}$$

也可以用样本均值表示：

$$a = \frac{\sum_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n} (x_i - \bar{x})^2}$$
$$b = \bar{y} - a\bar{x}$$

其中 $\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i$，$\bar{y} = \frac{1}{n}\sum_{i=1}^{n} y_i$

最小二乘法求解步骤
  1. 确定拟合函数的形式（如线性、二次、指数等）
  2. 建立目标函数（残差平方和）
  3. 对目标函数中的未知参数求偏导数
  4. 令所有偏导数等于零，得到正规方程组
  5. 解正规方程组，求出最优参数
  6. 验证解的合理性和拟合效果
矩阵形式的最小二乘法

对于线性系统 $\mathbf{Ax} = \mathbf{b}$，当方程组个数多于未知数个数时（超定系统），最小二乘解为：

$$\mathbf{x} = (\mathbf{A}^T\mathbf{A})^{-1}\mathbf{A}^T\mathbf{b}$$

其中 $\mathbf{A}^T$ 是 $\mathbf{A}$ 的转置，$(\mathbf{A}^T\mathbf{A})^{-1}\mathbf{A}^T$ 称为 $\mathbf{A}$ 的伪逆矩阵。

拟合优度评价

为了评价最小二乘拟合的效果，常用以下指标：

评价指标 数学表达式 取值范围 意义
相关系数 $R$ $R = \frac{\sum(x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum(x_i - \bar{x})^2 \sum(y_i - \bar{y})^2}}$ $[-1, 1]$ 衡量线性相关程度
决定系数 $R^2$ $R^2 = 1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}$ $[0, 1]$ 解释变量对因变量变异的解释程度
均方误差 MSE $\text{MSE} = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$ $[0, +\infty)$ 预测误差的平均水平
标准误差 $s$ $s = \sqrt{\frac{\sum(y_i - \hat{y}_i)^2}{n-2}}$ $[0, +\infty)$ 回归直线的标准偏差

最小二乘法的应用领域

数据拟合

通过实验或观测获得的数据点，寻找合适的函数关系。

例子：物理实验中根据测量数据确定物理定律的参数。

回归分析

研究变量之间的统计关系，建立预测模型。

例子：根据历史数据预测房价、股票价格等。

信号处理

在信号中提取有用信息，滤除噪声干扰。

例子：数字滤波器设计、图像去噪等。

系统辨识

根据输入输出数据确定系统的数学模型。

例子：控制系统中确定被控对象的传递函数。

机器学习

作为许多机器学习算法的基础优化方法。

例子：线性回归、神经网络训练等。

工程设计

在约束条件下寻找最优设计参数。

例子：结构优化、电路参数设计等。

例1：简单线性回归

已知数据点：$(1, 2)$，$(2, 3)$，$(3, 5)$，$(4, 4)$，$(5, 6)$。用最小二乘法求拟合直线 $y = ax + b$。

解：

第一步：计算所需的和

$n = 5$，$\sum x_i = 1+2+3+4+5 = 15$，$\sum y_i = 2+3+5+4+6 = 20$

$\sum x_i^2 = 1+4+9+16+25 = 55$，$\sum x_i y_i = 2+6+15+16+30 = 69$

第二步：计算参数 $a$

$$a = \frac{n\sum x_i y_i - \sum x_i \sum y_i}{n\sum x_i^2 - (\sum x_i)^2}$$ $$= \frac{5 \times 69 - 15 \times 20}{5 \times 55 - 15^2} = \frac{345 - 300}{275 - 225} = \frac{45}{50} = 0.9$$

第三步：计算参数 $b$

$$b = \frac{\sum y_i - a\sum x_i}{n} = \frac{20 - 0.9 \times 15}{5} = \frac{20 - 13.5}{5} = 1.3$$

结果：拟合直线为 $y = 0.9x + 1.3$

验证：计算残差平方和

各点的拟合值：$(1, 2.2)$，$(2, 3.1)$，$(3, 4.0)$，$(4, 4.9)$，$(5, 5.8)$

$S = (2-2.2)^2 + (3-3.1)^2 + (5-4.0)^2 + (4-4.9)^2 + (6-5.8)^2 = 2.3$

例2：多项式拟合

对于数据点 $(0, 1)$，$(1, 2)$，$(2, 5)$，$(3, 10)$，用二次多项式 $y = ax^2 + bx + c$ 进行拟合。

解：

建立目标函数：

$$S(a,b,c) = \sum_{i=1}^{4}(y_i - ax_i^2 - bx_i - c)^2$$

求偏导数并令其为零：

$$\frac{\partial S}{\partial a} = -2\sum x_i^2(y_i - ax_i^2 - bx_i - c) = 0$$ $$\frac{\partial S}{\partial b} = -2\sum x_i(y_i - ax_i^2 - bx_i - c) = 0$$ $$\frac{\partial S}{\partial c} = -2\sum (y_i - ax_i^2 - bx_i - c) = 0$$

建立正规方程组：

$$\begin{bmatrix} \sum x_i^4 & \sum x_i^3 & \sum x_i^2 \\ \sum x_i^3 & \sum x_i^2 & \sum x_i \\ \sum x_i^2 & \sum x_i & n \end{bmatrix} \begin{bmatrix} a \\ b \\ c \end{bmatrix} = \begin{bmatrix} \sum x_i^2 y_i \\ \sum x_i y_i \\ \sum y_i \end{bmatrix}$$

计算各项和：

$\sum x_i^4 = 81$，$\sum x_i^3 = 36$，$\sum x_i^2 = 14$，$\sum x_i = 6$，$n = 4$

$\sum x_i^2 y_i = 98$，$\sum x_i y_i = 34$，$\sum y_i = 18$

解得：$a = 1$，$b = 0$，$c = 1$

拟合多项式：$y = x^2 + 1$

例3：加权最小二乘法

当数据点的可靠性不同时，可以引入权重。设有数据点 $(x_i, y_i)$ 和对应权重 $w_i$，求加权最小二乘拟合直线。

解：

加权目标函数：

$$S_w(a,b) = \sum_{i=1}^{n} w_i(y_i - ax_i - b)^2$$

正规方程组：

$$\begin{cases} a\sum w_i x_i^2 + b\sum w_i x_i = \sum w_i x_i y_i \\ a\sum w_i x_i + b\sum w_i = \sum w_i y_i \end{cases}$$

参数解：

$$a = \frac{\sum w_i \sum w_i x_i y_i - \sum w_i x_i \sum w_i y_i}{\sum w_i \sum w_i x_i^2 - (\sum w_i x_i)^2}$$ $$b = \frac{\sum w_i y_i - a\sum w_i x_i}{\sum w_i}$$

权重选择原则：

  • 数据精度高的点赋予较大权重
  • 异常值或可疑点赋予较小权重
  • 常用权重：$w_i = \frac{1}{\sigma_i^2}$（$\sigma_i$ 为第 $i$ 点的标准差）
