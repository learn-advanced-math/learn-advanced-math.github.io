Taylor Formula for Functions of Two Variables

9.9.1 二元函数的泰勒公式介绍

二元函数泰勒公式的引入

一元函数的泰勒公式为我们提供了在某点附近用多项式近似函数的方法。对于二元函数 $f(x, y)$，我们也可以建立类似的泰勒展开式，用多项式来近似表示函数在某点邻域内的性态。

基本思想：将二元函数 $f(x, y)$ 在点 $(x_0, y_0)$ 附近展开为关于 $(x - x_0)$ 和 $(y - y_0)$ 的幂级数。

二元函数的泰勒公式

设函数 $f(x, y)$ 在点 $(x_0, y_0)$ 的某个邻域内具有直到 $n+1$ 阶的连续偏导数，则对该邻域内的任一点 $(x, y)$，有：

$$f(x, y) = \sum_{k=0}^{n} \frac{1}{k!} \left[ (x-x_0)\frac{\partial}{\partial x} + (y-y_0)\frac{\partial}{\partial y} \right]^k f(x_0, y_0) + R_n$$

其中 $R_n$ 是 $n$ 阶拉格朗日余项：

$$R_n = \frac{1}{(n+1)!} \left[ (x-x_0)\frac{\partial}{\partial x} + (y-y_0)\frac{\partial}{\partial y} \right]^{n+1} f(\xi, \eta)$$

这里 $(\xi, \eta)$ 是连接 $(x_0, y_0)$ 和 $(x, y)$ 的线段上的某点。

微分算子记号的含义

在泰勒公式中，微分算子的幂次按二项式定理展开：

算子展开规则
$k=0$ 阶： $$\left[ (x-x_0)\frac{\partial}{\partial x} + (y-y_0)\frac{\partial}{\partial y} \right]^0 f = f$$
$k=1$ 阶： $$\left[ (x-x_0)\frac{\partial}{\partial x} + (y-y_0)\frac{\partial}{\partial y} \right]^1 f = (x-x_0)f_x + (y-y_0)f_y$$
$k=2$ 阶： $$\left[ (x-x_0)\frac{\partial}{\partial x} + (y-y_0)\frac{\partial}{\partial y} \right]^2 f$$ $$= (x-x_0)^2 f_{xx} + 2(x-x_0)(y-y_0)f_{xy} + (y-y_0)^2 f_{yy}$$
一般规律：按二项式定理展开，每一项的系数为二项式系数，指数之和等于 $k$。
一阶泰勒公式（线性近似）
$$f(x, y) \approx f(x_0, y_0) + f_x(x_0, y_0)(x-x_0) + f_y(x_0, y_0)(y-y_0)$$

这就是函数的全微分近似。

二阶泰勒公式（二次近似）
$$f(x, y) \approx f(x_0, y_0) + f_x(x-x_0) + f_y(y-y_0)$$ $$+ \frac{1}{2}[f_{xx}(x-x_0)^2 + 2f_{xy}(x-x_0)(y-y_0) + f_{yy}(y-y_0)^2]$$

用于分析函数的极值性质。

麦克劳林公式

当展开点选择在原点 $(0, 0)$ 时，即 $x_0 = y_0 = 0$，泰勒公式简化为麦克劳林公式：

$$f(x, y) = \sum_{k=0}^{n} \frac{1}{k!} \left[ x\frac{\partial}{\partial x} + y\frac{\partial}{\partial y} \right]^k f(0, 0) + R_n$$

前几阶的展开形式为：

$$f(x, y) = f(0, 0) + xf_x(0, 0) + yf_y(0, 0)$$ $$+ \frac{1}{2}[x^2 f_{xx}(0, 0) + 2xyf_{xy}(0, 0) + y^2 f_{yy}(0, 0)] + \cdots$$

泰勒公式的应用

二元函数的泰勒公式在数学分析中有广泛的应用：

应用领域 使用的阶数 主要目的 典型例子
函数近似 一阶或二阶 在某点附近用简单多项式近似复杂函数 $e^{x+y} \approx 1 + x + y$ （在原点附近）
极值判定 二阶 判断驻点的极值性质 利用海塞矩阵判断极值
误差估计 根据精度要求确定 估算函数近似的误差范围 数值计算中的误差控制
极限计算 适当阶数 计算复杂的多元函数极限 $\lim_{(x,y) \to (0,0)} \frac{e^{x+y} - 1 - x - y}{x^2 + y^2}$

例1：求函数的二阶麦克劳林展开

求函数 $f(x, y) = e^{x+y}$ 在原点 $(0, 0)$ 处的二阶麦克劳林展开式。

解：

第一步：计算函数在原点的值

$$f(0, 0) = e^{0+0} = 1$$

第二步：计算一阶偏导数

$$f_x(x, y) = e^{x+y}, \quad f_x(0, 0) = 1$$ $$f_y(x, y) = e^{x+y}, \quad f_y(0, 0) = 1$$

第三步：计算二阶偏导数

$$f_{xx}(x, y) = e^{x+y}, \quad f_{xx}(0, 0) = 1$$ $$f_{xy}(x, y) = e^{x+y}, \quad f_{xy}(0, 0) = 1$$ $$f_{yy}(x, y) = e^{x+y}, \quad f_{yy}(0, 0) = 1$$

第四步：写出二阶麦克劳林展开式

$$e^{x+y} = 1 + 1 \cdot x + 1 \cdot y + \frac{1}{2}[1 \cdot x^2 + 2 \cdot 1 \cdot xy + 1 \cdot y^2] + o((x^2 + y^2))$$

整理得：

$$e^{x+y} = 1 + x + y + \frac{1}{2}(x^2 + 2xy + y^2) + o((x^2 + y^2))$$ $$= 1 + x + y + \frac{1}{2}(x + y)^2 + o((x^2 + y^2))$$

例2：利用泰勒公式判断极值

已知函数 $f(x, y) = x^4 + y^4 - 2x^2 - 2y^2 + 1$ 在点 $(0, 0)$ 处有驻点，利用泰勒公式判断其极值性质。

解：

第一步：验证 $(0, 0)$ 是驻点

$$f_x = 4x^3 - 4x, \quad f_x(0, 0) = 0$$ $$f_y = 4y^3 - 4y, \quad f_y(0, 0) = 0$$

确实，$(0, 0)$ 是驻点。

第二步：计算二阶偏导数

$$f_{xx} = 12x^2 - 4, \quad f_{xx}(0, 0) = -4$$ $$f_{xy} = 0, \quad f_{xy}(0, 0) = 0$$ $$f_{yy} = 12y^2 - 4, \quad f_{yy}(0, 0) = -4$$

第三步：判别式法判断

$$\Delta = f_{xx}(0, 0) \cdot f_{yy}(0, 0) - [f_{xy}(0, 0)]^2 = (-4) \cdot (-4) - 0^2 = 16 > 0$$

由于 $\Delta > 0$ 且 $f_{xx}(0, 0) = -4 < 0$，所以 $(0, 0)$ 是极大值点。

第四步：用泰勒公式验证

在原点附近的二阶泰勒展开：

$$f(x, y) = f(0, 0) + \frac{1}{2}[f_{xx}(0, 0)x^2 + 2f_{xy}(0, 0)xy + f_{yy}(0, 0)y^2] + \text{高阶项}$$ $$= 1 + \frac{1}{2}[-4x^2 + 0 \cdot xy - 4y^2] + \text{高阶项}$$ $$= 1 - 2(x^2 + y^2) + \text{高阶项}$$

在 $(0, 0)$ 的小邻域内，$-2(x^2 + y^2) < 0$，所以 $f(x, y) < f(0, 0) = 1$，确认 $(0, 0)$ 是极大值点。

例3：利用泰勒公式计算极限

计算极限：$\lim\limits_{(x,y) \to (0,0)} \frac{e^{x+y} - 1 - x - y}{x^2 + y^2}$

解：

第一步：利用例1的结果

由例1，我们已知：

$$e^{x+y} = 1 + x + y + \frac{1}{2}(x + y)^2 + o((x^2 + y^2))$$

第二步：计算分子

$$e^{x+y} - 1 - x - y = \frac{1}{2}(x + y)^2 + o((x^2 + y^2))$$ $$= \frac{1}{2}(x^2 + 2xy + y^2) + o((x^2 + y^2))$$

第三步：计算极限

$$\lim_{(x,y) \to (0,0)} \frac{e^{x+y} - 1 - x - y}{x^2 + y^2} = \lim_{(x,y) \to (0,0)} \frac{\frac{1}{2}(x^2 + 2xy + y^2) + o((x^2 + y^2))}{x^2 + y^2}$$

第四步：分析主导项

分子的主导项是 $\frac{1}{2}(x^2 + 2xy + y^2)$，但这个极限不存在，因为：

$$\frac{x^2 + 2xy + y^2}{x^2 + y^2} = \frac{(x + y)^2}{x^2 + y^2}$$

这个比值沿不同路径趋于 $(0, 0)$ 时有不同的值。

结论：该极限不存在。

说明：这个例子提醒我们，即使使用泰勒展开，如果主导项的系数中仍含有方向性因素，极限可能不存在。
