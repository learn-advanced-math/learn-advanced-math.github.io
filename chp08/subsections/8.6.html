8.6 空间曲线及其方程

Space Curves and Their Equations

8.6.1 空间曲线的表示方法

空间曲线的一般方程

在空间直角坐标系中，空间曲线可以看作两个曲面的交线。若两个曲面的方程分别为：

$$\begin{cases} F(x, y, z) = 0 \\ G(x, y, z) = 0 \end{cases}$$

则它们的交线就是一条空间曲线，上述方程组称为空间曲线的一般方程。

空间曲线的参数方程

空间曲线也可以用参数方程表示：

$$\begin{cases} x = x(t) \\ y = y(t) \\ z = z(t) \end{cases} \quad (a \leq t \leq b)$$

其中 $t$ 是参数，$x(t)$、$y(t)$、$z(t)$ 是 $t$ 的函数。参数方程形式更直观，便于描述曲线的几何性质。

螺旋线

圆柱螺旋线是最常见的空间曲线之一，其参数方程为：

$$\begin{cases} x = a\cos t \\ y = a\sin t \\ z = bt \end{cases} \quad (t \in \mathbb{R})$$

其中：

  • $a > 0$ 是螺旋线的半径
  • $b \neq 0$ 决定螺旋线的升降方向和快慢
  • 螺距：参数 $t$ 增加 $2\pi$ 时，$z$ 坐标的增量 $h = 2\pi|b|$
直线（特殊情况）
$$\begin{cases} x = x_0 + at \\ y = y_0 + bt \\ z = z_0 + ct \end{cases}$$

通过点 $(x_0, y_0, z_0)$，方向向量为 $(a, b, c)$ 的直线

圆（平面曲线）
$$\begin{cases} x = r\cos t \\ y = r\sin t \\ z = h \end{cases}$$

在平面 $z = h$ 上，半径为 $r$ 的圆

椭圆
$$\begin{cases} x = a\cos t \\ y = b\sin t \\ z = h \end{cases}$$

在平面 $z = h$ 上，长轴 $2a$，短轴 $2b$ 的椭圆

例1：求两个曲面的交线

求球面 $x^2 + y^2 + z^2 = 9$ 与圆柱面 $x^2 + y^2 = 4$ 的交线方程，并写出其参数方程。

解：

一般方程：

$$\begin{cases} x^2 + y^2 + z^2 = 9 \\ x^2 + y^2 = 4 \end{cases}$$

将第二个方程代入第一个方程：

$$4 + z^2 = 9 \Rightarrow z^2 = 5 \Rightarrow z = \pm\sqrt{5}$$

参数方程：

由于 $x^2 + y^2 = 4$，可设：

$$\begin{cases} x = 2\cos t \\ y = 2\sin t \\ z = \pm\sqrt{5} \end{cases} \quad (0 \leq t \leq 2\pi)$$

几何意义：交线是两个圆，分别在平面 $z = \sqrt{5}$ 和 $z = -\sqrt{5}$ 上，半径都为 $2$。

例2：螺旋线的性质分析

给定圆柱螺旋线 $\begin{cases} x = 3\cos t \\ y = 3\sin t \\ z = 4t \end{cases}$，求：
(1) 螺旋线的半径和螺距
(2) 当 $t$ 从 $0$ 变化到 $4\pi$ 时，曲线的长度

解：

(1) 半径和螺距：

  • 半径：$a = 3$
  • 螺距：$h = 2\pi \cdot |b| = 2\pi \cdot 4 = 8\pi$

(2) 曲线长度：

曲线长度公式：

$$s = \int_0^{4\pi} \sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2 + \left(\frac{dz}{dt}\right)^2} dt$$

计算各导数：

$$\frac{dx}{dt} = -3\sin t, \quad \frac{dy}{dt} = 3\cos t, \quad \frac{dz}{dt} = 4$$

代入公式：

$$s = \int_0^{4\pi} \sqrt{9\sin^2 t + 9\cos^2 t + 16} dt = \int_0^{4\pi} \sqrt{9 + 16} dt = \int_0^{4\pi} 5 dt = 20\pi$$

结论：螺旋线的半径为 $3$，螺距为 $8\pi$，当 $t \in [0, 4\pi]$ 时曲线长度为 $20\pi$。

8.6.2 空间曲线的投影

投影柱面

设空间曲线的一般方程为：

$$\begin{cases} F(x, y, z) = 0 \\ G(x, y, z) = 0 \end{cases}$$

要求该曲线在 $xOy$ 坐标面上的投影，需要从方程组中消去 $z$，得到方程 $H(x, y) = 0$。

曲面 $H(x, y) = 0$（其中 $z$ 为任意值）称为曲线关于 $xOy$ 面的投影柱面。

投影曲线

空间曲线在坐标面上的投影曲线方程为：

  • 在 $xOy$ 面上的投影：$\begin{cases} H(x, y) = 0 \\ z = 0 \end{cases}$
  • 在 $xOz$ 面上的投影：$\begin{cases} H_1(x, z) = 0 \\ y = 0 \end{cases}$
  • 在 $yOz$ 面上的投影：$\begin{cases} H_2(y, z) = 0 \\ x = 0 \end{cases}$

投影的求法

方法一：消元法

从一般方程组中消去相应的变量。

方法二：参数方程法

若曲线有参数方程 $\begin{cases} x = x(t) \\ y = y(t) \\ z = z(t) \end{cases}$，则：

  • 在 $xOy$ 面上的投影：$\begin{cases} x = x(t) \\ y = y(t) \end{cases}$（消去参数 $t$）
  • 在 $xOz$ 面上的投影：$\begin{cases} x = x(t) \\ z = z(t) \end{cases}$（消去参数 $t$）
  • 在 $yOz$ 面上的投影：$\begin{cases} y = y(t) \\ z = z(t) \end{cases}$（消去参数 $t$）

例3：求空间曲线的投影

求曲线 $\begin{cases} x^2 + y^2 + z^2 = 6 \\ x + y + z = 0 \end{cases}$ 在三个坐标面上的投影曲线方程。

解：

在 $xOy$ 面上的投影：

从第二个方程得：$z = -x - y$

代入第一个方程：

$$x^2 + y^2 + (-x - y)^2 = 6$$ $$x^2 + y^2 + x^2 + 2xy + y^2 = 6$$ $$2x^2 + 2xy + 2y^2 = 6$$ $$x^2 + xy + y^2 = 3$$

所以在 $xOy$ 面上的投影为：$\begin{cases} x^2 + xy + y^2 = 3 \\ z = 0 \end{cases}$

在 $xOz$ 面上的投影：

从第二个方程得：$y = -x - z$

代入第一个方程：

$$x^2 + (-x - z)^2 + z^2 = 6$$ $$x^2 + x^2 + 2xz + z^2 + z^2 = 6$$ $$2x^2 + 2xz + 2z^2 = 6$$ $$x^2 + xz + z^2 = 3$$

所以在 $xOz$ 面上的投影为：$\begin{cases} x^2 + xz + z^2 = 3 \\ y = 0 \end{cases}$

在 $yOz$ 面上的投影：

类似地，可得：$\begin{cases} y^2 + yz + z^2 = 3 \\ x = 0 \end{cases}$

几何意义：原曲线是球面与平面的交线（一个圆），其投影都是椭圆。

练习题

习题1 中等难度

求圆柱螺旋线 $\begin{cases} x = 2\cos t \\ y = 2\sin t \\ z = 3t \end{cases}$ 在 $xOy$ 坐标面上的投影曲线方程。

解：

由参数方程可得：

$$x = 2\cos t, \quad y = 2\sin t$$

消去参数 $t$：

$$\cos t = \frac{x}{2}, \quad \sin t = \frac{y}{2}$$

利用三角恒等式 $\cos^2 t + \sin^2 t = 1$：

$$\left(\frac{x}{2}\right)^2 + \left(\frac{y}{2}\right)^2 = 1$$ $$\frac{x^2}{4} + \frac{y^2}{4} = 1$$ $$x^2 + y^2 = 4$$

答案：在 $xOy$ 面上的投影曲线方程为：$\begin{cases} x^2 + y^2 = 4 \\ z = 0 \end{cases}$

说明：投影是以原点为圆心，半径为 $2$ 的圆。

习题2 较难

给定空间曲线 $\begin{cases} x^2 + y^2 = 4z \\ x^2 + y^2 + z^2 = 9 \end{cases}$，求：
(1) 该曲线在 $xOy$ 面上的投影方程
(2) 该曲线的参数方程

解：

(1) 求在 $xOy$ 面上的投影：

从第一个方程得：$z = \frac{x^2 + y^2}{4}$

代入第二个方程：

$$x^2 + y^2 + \left(\frac{x^2 + y^2}{4}\right)^2 = 9$$

设 $u = x^2 + y^2$，则：

$$u + \frac{u^2}{16} = 9$$ $$16u + u^2 = 144$$ $$u^2 + 16u - 144 = 0$$

解得：$u = \frac{-16 \pm \sqrt{256 + 576}}{2} = \frac{-16 \pm \sqrt{832}}{2} = \frac{-16 \pm 8\sqrt{13}}{2} = -8 \pm 4\sqrt{13}$

因为 $u = x^2 + y^2 \geq 0$，所以 $u = -8 + 4\sqrt{13}$

投影方程：$\begin{cases} x^2 + y^2 = 4\sqrt{13} - 8 \\ z = 0 \end{cases}$

(2) 求参数方程：

设 $x = r\cos t$，$y = r\sin t$，其中 $r = \sqrt{4\sqrt{13} - 8}$

从 $z = \frac{x^2 + y^2}{4} = \frac{r^2}{4} = \frac{4\sqrt{13} - 8}{4} = \sqrt{13} - 2$

参数方程：

$$\begin{cases} x = \sqrt{4\sqrt{13} - 8} \cos t \\ y = \sqrt{4\sqrt{13} - 8} \sin t \\ z = \sqrt{13} - 2 \end{cases} \quad (0 \leq t \leq 2\pi)$$

几何意义：该曲线是抛物面与球面的交线，是一个圆。

习题3 很难

考虑空间曲线族：$\begin{cases} x = a\cos t + b\sin t \\ y = a\sin t - b\cos t \\ z = ct \end{cases}$，其中 $a, b, c$ 为正常数。
(1) 证明该曲线在 $xOy$ 面上的投影是一个圆
(2) 求该曲线的螺距
(3) 当 $a = 3$，$b = 4$，$c = 2$ 时，求曲线从 $t = 0$ 到 $t = 2\pi$ 的弧长

解：

(1) 证明投影是圆：

计算 $x^2 + y^2$：

$$x^2 + y^2 = (a\cos t + b\sin t)^2 + (a\sin t - b\cos t)^2$$ $$= a^2\cos^2 t + 2ab\cos t \sin t + b^2\sin^2 t + a^2\sin^2 t - 2ab\sin t \cos t + b^2\cos^2 t$$ $$= a^2(\cos^2 t + \sin^2 t) + b^2(\sin^2 t + \cos^2 t)$$ $$= a^2 + b^2$$

因此在 $xOy$ 面上的投影方程为：$\begin{cases} x^2 + y^2 = a^2 + b^2 \\ z = 0 \end{cases}$

这是以原点为圆心，半径为 $\sqrt{a^2 + b^2}$ 的圆。得证。

(2) 求螺距：

当参数 $t$ 增加 $2\pi$ 时，$z$ 坐标的增量为：

$$\Delta z = c(t + 2\pi) - ct = 2\pi c$$

螺距：$h = 2\pi c$

(3) 求弧长（$a = 3$，$b = 4$，$c = 2$）：

计算各导数：

$$\frac{dx}{dt} = -3\sin t + 4\cos t$$ $$\frac{dy}{dt} = 3\cos t + 4\sin t$$ $$\frac{dz}{dt} = 2$$

计算导数的模长：

$$\left|\frac{d\mathbf{r}}{dt}\right|^2 = (-3\sin t + 4\cos t)^2 + (3\cos t + 4\sin t)^2 + 2^2$$ $$= 9\sin^2 t - 24\sin t \cos t + 16\cos^2 t + 9\cos^2 t + 24\cos t \sin t + 16\sin^2 t + 4$$ $$= 9(\sin^2 t + \cos^2 t) + 16(\cos^2 t + \sin^2 t) + 4$$ $$= 9 + 16 + 4 = 29$$

因此：$\left|\frac{d\mathbf{r}}{dt}\right| = \sqrt{29}$

弧长为：

$$s = \int_0^{2\pi} \sqrt{29} \, dt = 2\pi\sqrt{29}$$

答案：弧长为 $2\pi\sqrt{29}$

注：这是一种广义的螺旋线，其投影不再是标准的圆，而是椭圆在旋转后的结果，但在此特殊情况下仍为圆。
