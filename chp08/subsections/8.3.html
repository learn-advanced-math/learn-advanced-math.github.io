8.3.1 曲面方程与空间曲线方程

曲面方程的概念

在空间直角坐标系中，一个三元方程 $F(x, y, z) = 0$ 一般表示一张曲面。

曲面方程的特点：

  • 曲面上的点的坐标满足该方程
  • 不在曲面上的点的坐标不满足该方程
  • 三个变量之间存在一个约束关系
常见曲面方程的一般形式：$F(x, y, z) = 0$

空间曲线方程的概念

空间曲线可以看作两个曲面的交线，通常用方程组表示：

$$\begin{cases} F(x, y, z) = 0 \\ G(x, y, z) = 0 \end{cases}$$

空间曲线的参数方程：

$$\begin{cases} x = x(t) \\ y = y(t) \\ z = z(t) \end{cases}$$

例1：曲面方程的理解

分析方程 $x^2 + y^2 + z^2 = 4$ 表示什么曲面。

解：

这是以原点为中心，半径为 $2$ 的球面。

验证：

  • 当 $z = 0$ 时，得到 $x^2 + y^2 = 4$，这是 $xy$ 平面上的圆
  • 对于球面上任意一点 $(x_0, y_0, z_0)$，都满足 $x_0^2 + y_0^2 + z_0^2 = 4$
  • 球心到表面任意一点的距离都等于 $2$

例2：空间曲线方程

求球面 $x^2 + y^2 + z^2 = 4$ 与平面 $z = 1$ 的交线方程。

解：

交线方程为：

$$\begin{cases} x^2 + y^2 + z^2 = 4 \\ z = 1 \end{cases}$$

将 $z = 1$ 代入第一个方程：

$$x^2 + y^2 + 1 = 4$$ $$x^2 + y^2 = 3$$

因此，交线是平面 $z = 1$ 上以原点为中心、半径为 $\sqrt{3}$ 的圆。

💡 几何直观理解

球面与平面的交线：

  • 球面 $x^2 + y^2 + z^2 = 4$ 是以原点为中心、半径为2的球体表面
  • 平面 $z = 1$ 是一个水平平面，高度为1
  • 它们的交线是平面 $z = 1$ 上的圆，圆心在 $(0, 0, 1)$，半径为 $\sqrt{3}$
  • 这个圆的方程可以写成参数形式：$x = \sqrt{3}\cos t$，$y = \sqrt{3}\sin t$，$z = 1$

8.3.2 平面的一般式方程、点法式方程、截距式方程

一般式方程
$$Ax + By + Cz + D = 0$$

其中 $A$、$B$、$C$ 不全为零，$(A, B, C)$ 是平面的法向量

点法式方程
$$A(x - x_0) + B(y - y_0) + C(z - z_0) = 0$$

过点 $(x_0, y_0, z_0)$，法向量为 $(A, B, C)$ 的平面

截距式方程
$$\frac{x}{a} + \frac{y}{b} + \frac{z}{c} = 1$$

在三个坐标轴上的截距分别为 $a$、$b$、$c$

平面方程的确定

确定平面的条件：

  • 三个不共线的点：可以唯一确定一个平面
  • 一个点和一个法向量：可以确定点法式方程
  • 两条相交直线：可以确定平面
  • 两条平行直线：可以确定平面

法向量的求法：如果平面通过三点 $A(x_1,y_1,z_1)$，$B(x_2,y_2,z_2)$，$C(x_3,y_3,z_3)$，则法向量 $\vec{n} = \overrightarrow{AB} \times \overrightarrow{AC}$

例3：求平面的点法式方程

求过点 $M(1, 2, 3)$ 且垂直于向量 $\vec{n} = (2, -1, 3)$ 的平面方程。

解：

使用点法式方程：$A(x - x_0) + B(y - y_0) + C(z - z_0) = 0$

其中 $(x_0, y_0, z_0) = (1, 2, 3)$，$(A, B, C) = (2, -1, 3)$

代入得：

$$2(x - 1) + (-1)(y - 2) + 3(z - 3) = 0$$ $$2x - 2 - y + 2 + 3z - 9 = 0$$ $$2x - y + 3z - 9 = 0$$

例4：三点确定平面方程

求过三点 $A(1, 0, 0)$，$B(0, 1, 0)$，$C(0, 0, 1)$ 的平面方程。

解：

方法一：使用截距式方程

由于三点分别在三个坐标轴上，截距为 $a = 1$，$b = 1$，$c = 1$

$$\frac{x}{1} + \frac{y}{1} + \frac{z}{1} = 1$$ $$x + y + z = 1$$

方法二：使用法向量

$\overrightarrow{AB} = (-1, 1, 0)$，$\overrightarrow{AC} = (-1, 0, 1)$

法向量 $\vec{n} = \overrightarrow{AB} \times \overrightarrow{AC} = (1, 1, 1)$

使用点 $A(1, 0, 0)$：

$$1(x - 1) + 1(y - 0) + 1(z - 0) = 0$$ $$x - 1 + y + z = 0$$ $$x + y + z = 1$$

8.3.3 两平面的夹角；点到平面的距离

两平面的夹角

设两平面方程为：

  • $\pi_1: A_1x + B_1y + C_1z + D_1 = 0$
  • $\pi_2: A_2x + B_2y + C_2z + D_2 = 0$

两平面的夹角 $\theta$ 满足：

$$\cos \theta = \frac{|A_1A_2 + B_1B_2 + C_1C_2|}{\sqrt{A_1^2 + B_1^2 + C_1^2}\sqrt{A_2^2 + B_2^2 + C_2^2}}$$

特殊情况：

  • 平行：$A_1:B_1:C_1 = A_2:B_2:C_2$
  • 垂直：$A_1A_2 + B_1B_2 + C_1C_2 = 0$

点到平面的距离

点 $P_0(x_0, y_0, z_0)$ 到平面 $Ax + By + Cz + D = 0$ 的距离为：

$$d = \frac{|Ax_0 + By_0 + Cz_0 + D|}{\sqrt{A^2 + B^2 + C^2}}$$

推导思路：

  1. 在平面上任选一点 $P_1(x_1, y_1, z_1)$
  2. 构造向量 $\overrightarrow{P_1P_0} = (x_0-x_1, y_0-y_1, z_0-z_1)$
  3. 平面法向量 $\vec{n} = (A, B, C)$
  4. 距离 = $|\overrightarrow{P_1P_0}|$ 在法向量上的投影

例5：求两平面的夹角

求平面 $x + y + z - 1 = 0$ 与平面 $2x - y + 3z + 5 = 0$ 的夹角。

解：

两平面的法向量分别为：

  • $\vec{n_1} = (1, 1, 1)$
  • $\vec{n_2} = (2, -1, 3)$

计算夹角余弦值：

$$\cos \theta = \frac{|1 \cdot 2 + 1 \cdot (-1) + 1 \cdot 3|}{\sqrt{1^2 + 1^2 + 1^2}\sqrt{2^2 + (-1)^2 + 3^2}}$$ $$= \frac{|2 - 1 + 3|}{\sqrt{3}\sqrt{14}} = \frac{4}{\sqrt{42}}$$

因此夹角为：$\theta = \arccos\left(\frac{4}{\sqrt{42}}\right) \approx 51.8°$

例6：求点到平面的距离

求点 $P(1, 2, 3)$ 到平面 $2x - y + 2z - 6 = 0$ 的距离。

解：

使用点到平面距离公式：

$$d = \frac{|Ax_0 + By_0 + Cz_0 + D|}{\sqrt{A^2 + B^2 + C^2}}$$

其中 $(A, B, C, D) = (2, -1, 2, -6)$，$(x_0, y_0, z_0) = (1, 2, 3)$

$$d = \frac{|2 \cdot 1 + (-1) \cdot 2 + 2 \cdot 3 - 6|}{\sqrt{2^2 + (-1)^2 + 2^2}}$$ $$= \frac{|2 - 2 + 6 - 6|}{\sqrt{4 + 1 + 4}} = \frac{0}{3} = 0$$

距离为 $0$，说明点 $P(1, 2, 3)$ 就在平面上。

习题1 中等难度

求过点 $(1, 1, 1)$ 且平行于平面 $2x + 3y - z + 5 = 0$ 的平面方程。

解：

因为两平面平行，所以它们的法向量相同。

给定平面的法向量为 $(2, 3, -1)$

所求平面过点 $(1, 1, 1)$，法向量为 $(2, 3, -1)$

使用点法式方程：

$$2(x - 1) + 3(y - 1) + (-1)(z - 1) = 0$$ $$2x - 2 + 3y - 3 - z + 1 = 0$$ $$2x + 3y - z - 4 = 0$$

习题2 较难

求过直线 $\begin{cases} x + y - z = 1 \\ 2x - y + z = 3 \end{cases}$ 且垂直于平面 $x + 2y + z = 0$ 的平面方程。

解：

步骤1：求直线的方向向量

两平面的法向量分别为 $\vec{n_1} = (1, 1, -1)$，$\vec{n_2} = (2, -1, 1)$

直线方向向量：$\vec{s} = \vec{n_1} \times \vec{n_2} = (0, -3, -3) = 3(0, -1, -1)$

取 $\vec{s} = (0, -1, -1)$

步骤2：求所求平面的法向量

所求平面包含直线，所以 $\vec{s} \perp \vec{n}$

所求平面垂直于已知平面，设已知平面法向量 $\vec{n_0} = (1, 2, 1)$

所以 $\vec{n} \parallel (\vec{s} \times \vec{n_0})$

$$\vec{n} = \vec{s} \times \vec{n_0} = (0, -1, -1) \times (1, 2, 1) = (1, -1, 1)$$

步骤3：求平面上一点

令 $z = 0$，得 $\begin{cases} x + y = 1 \\ 2x - y = 3 \end{cases}$

解得 $x = \frac{4}{3}$，$y = -\frac{1}{3}$，所以 $(\frac{4}{3}, -\frac{1}{3}, 0)$ 在直线上

步骤4：写出平面方程

$$1(x - \frac{4}{3}) + (-1)(y + \frac{1}{3}) + 1(z - 0) = 0$$ $$x - \frac{4}{3} - y - \frac{1}{3} + z = 0$$ $$x - y + z = \frac{5}{3}$$

即：$3x - 3y + 3z = 5$

习题3 很难

求到两平面 $x + 2y - 2z + 3 = 0$ 和 $2x + 4y - 4z - 1 = 0$ 距离相等的点的轨迹方程。

解：

设动点为 $P(x, y, z)$，它到两个平面的距离分别为 $d_1$ 和 $d_2$

$$d_1 = \frac{|x + 2y - 2z + 3|}{\sqrt{1^2 + 2^2 + (-2)^2}} = \frac{|x + 2y - 2z + 3|}{3}$$

对于第二个平面，先化简：$2x + 4y - 4z - 1 = 0$ 即 $x + 2y - 2z - \frac{1}{2} = 0$

$$d_2 = \frac{|x + 2y - 2z - \frac{1}{2}|}{3}$$

由 $d_1 = d_2$ 得：

$$|x + 2y - 2z + 3| = |x + 2y - 2z - \frac{1}{2}|$$

设 $u = x + 2y - 2z$，则 $|u + 3| = |u - \frac{1}{2}|$

情况1：$u + 3 = u - \frac{1}{2}$，得 $3 = -\frac{1}{2}$（无解）

情况2：$u + 3 = -(u - \frac{1}{2})$，得 $2u = -\frac{7}{2}$，即 $u = -\frac{7}{4}$

因此轨迹方程为：

$$x + 2y - 2z = -\frac{7}{4}$$ $$4x + 8y - 8z + 7 = 0$$

这是一个平面，它平行于给定的两个平面，且到两个平面的距离相等。
