12.8 一般周期函数的傅里叶级数

Fourier Series for General Periodic Functions

12.8.1 周期函数的傅里叶级数

一般周期函数的傅里叶级数

设函数 $f(x)$ 是周期为 $T = 2l$ 的周期函数，即 $f(x+2l) = f(x)$，则其傅里叶级数为：

$$f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left(a_n \cos \frac{n\pi x}{l} + b_n \sin \frac{n\pi x}{l}\right)$$

其中傅里叶系数为：

$a_0$ 系数
$a_0 = \frac{1}{l} \int_{-l}^{l} f(x) dx$

（函数在一个周期内的平均值的两倍）

$a_n$ 系数
$a_n = \frac{1}{l} \int_{-l}^{l} f(x) \cos \frac{n\pi x}{l} dx$

（余弦分量的系数）

$b_n$ 系数
$b_n = \frac{1}{l} \int_{-l}^{l} f(x) \sin \frac{n\pi x}{l} dx$

（正弦分量的系数）

变量替换
令 $t = \frac{\pi x}{l}$，则 $x = \frac{lt}{\pi}$

（将一般周期转换为 $2\pi$ 周期）

注意：积分区间可以是任意一个完整周期，如 $[0, 2l]$、$[c, c+2l]$ 等。

周期函数的性质

周期性：如果 $f(x)$ 的周期为 $T$，则：

  • $f(x+T) = f(x)$ 对所有 $x$ 成立
  • $f(x)$ 的最小正周期是所有周期中最小的正数
  • 如果 $T$ 是周期，则 $nT$（$n$ 为正整数）也是周期

积分性质：

$$\int_a^{a+T} f(x) dx = \int_0^T f(x) dx$$

即在任意一个完整周期上的积分值相等。

例1：周期为 $4$ 的矩形波

求周期为 $4$ 的函数 $f(x) = \begin{cases} 1, & 0 < x < 2 \\ -1, & -2 < x < 0 \end{cases}$ 的傅里叶级数。

解：

步骤1：确定参数

周期 $T = 4$，所以 $2l = 4$，$l = 2$

步骤2：计算 $a_0$

$$a_0 = \frac{1}{2} \int_{-2}^{2} f(x) dx = \frac{1}{2} \left[\int_{-2}^{0} (-1) dx + \int_{0}^{2} 1 dx\right]$$ $$= \frac{1}{2}[-2 + 2] = 0$$

步骤3：计算 $a_n$

由于 $f(x)$ 是奇函数，所以：

$$a_n = \frac{1}{2} \int_{-2}^{2} f(x) \cos \frac{n\pi x}{2} dx = 0$$

步骤4：计算 $b_n$

$$b_n = \frac{1}{2} \int_{-2}^{2} f(x) \sin \frac{n\pi x}{2} dx = \frac{2}{2} \int_{0}^{2} \sin \frac{n\pi x}{2} dx$$ $$= \int_{0}^{2} \sin \frac{n\pi x}{2} dx = \left[-\frac{2}{n\pi} \cos \frac{n\pi x}{2}\right]_0^{2}$$ $$= -\frac{2}{n\pi}[\cos n\pi - 1] = \frac{2}{n\pi}[1 - (-1)^n]$$ $$= \begin{cases} \frac{4}{n\pi}, & n \text{为奇数} \\ 0, & n \text{为偶数} \end{cases}$$

结果：

$$f(x) = \frac{4}{\pi}\left(\sin \frac{\pi x}{2} + \frac{1}{3}\sin \frac{3\pi x}{2} + \frac{1}{5}\sin \frac{5\pi x}{2} + \cdots\right)$$

例2：周期为 $6$ 的锯齿波

求周期为 $6$ 的函数 $f(x) = x$，$x \in (-3, 3)$ 的傅里叶级数。

解：

步骤1：确定参数

周期 $T = 6$，所以 $2l = 6$，$l = 3$

步骤2：分析函数性质

$f(x) = x$ 在 $(-3, 3)$ 上是奇函数，所以 $a_0 = a_n = 0$

步骤3：计算 $b_n$

$$b_n = \frac{1}{3} \int_{-3}^{3} x \sin \frac{n\pi x}{3} dx = \frac{2}{3} \int_{0}^{3} x \sin \frac{n\pi x}{3} dx$$

使用分部积分：

$$b_n = \frac{2}{3} \left[-\frac{3x}{n\pi} \cos \frac{n\pi x}{3}\Big|_0^{3} + \frac{3}{n\pi}\int_0^{3} \cos \frac{n\pi x}{3} dx\right]$$ $$= \frac{2}{3} \left[-\frac{9}{n\pi} \cos n\pi + 0\right] = -\frac{6}{n\pi} (-1)^n = \frac{6(-1)^{n+1}}{n\pi}$$

结果：

$$f(x) = \frac{6}{\pi}\left(\sin \frac{\pi x}{3} - \frac{1}{2}\sin \frac{2\pi x}{3} + \frac{1}{3}\sin \pi x - \cdots\right)$$ $$= \frac{6}{\pi}\sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n} \sin \frac{n\pi x}{3}$$

例3：周期为 $8$ 的抛物线函数

求周期为 $8$ 的函数 $f(x) = x^2$，$x \in [-4, 4]$ 的傅里叶级数。

解：

步骤1：确定参数

周期 $T = 8$，所以 $2l = 8$，$l = 4$

步骤2：分析函数性质

$f(x) = x^2$ 是偶函数，所以 $b_n = 0$

步骤3：计算 $a_0$

$$a_0 = \frac{1}{4} \int_{-4}^{4} x^2 dx = \frac{2}{4} \int_{0}^{4} x^2 dx = \frac{1}{2} \cdot \frac{64}{3} = \frac{32}{3}$$

步骤4：计算 $a_n$

$$a_n = \frac{1}{4} \int_{-4}^{4} x^2 \cos \frac{n\pi x}{4} dx = \frac{2}{4} \int_{0}^{4} x^2 \cos \frac{n\pi x}{4} dx$$

使用两次分部积分：

$$a_n = \frac{1}{2} \left[\frac{4x^2}{n\pi} \sin \frac{n\pi x}{4}\Big|_0^{4} - \frac{8}{n\pi}\int_0^{4} x \sin \frac{n\pi x}{4} dx\right]$$

继续计算得：

$$a_n = \frac{64(-1)^n}{n^2\pi^2}$$

结果：

$$f(x) = \frac{16}{3} + \frac{64}{\pi^2}\left(-\cos \frac{\pi x}{4} + \frac{1}{4}\cos \frac{\pi x}{2} - \frac{1}{9}\cos \frac{3\pi x}{4} + \cdots\right)$$

12.8.2 傅里叶级数的复数形式

欧拉公式

欧拉公式是连接三角函数和指数函数的桥梁：

$$e^{i\theta} = \cos \theta + i \sin \theta$$
正弦函数表示
$\sin \theta = \frac{e^{i\theta} - e^{-i\theta}}{2i}$
余弦函数表示
$\cos \theta = \frac{e^{i\theta} + e^{-i\theta}}{2}$
特殊值
$e^{i\pi} + 1 = 0$

（欧拉恒等式）

周期性
$e^{i(\theta + 2\pi)} = e^{i\theta}$

（复指数函数的周期性）

复数形式的几何意义
45°

傅里叶级数的复数形式

利用欧拉公式，可以将傅里叶级数写成复数形式：

$$f(x) = \sum_{n=-\infty}^{\infty} c_n e^{i\frac{n\pi x}{l}}$$

其中复数傅里叶系数为：

$$c_n = \frac{1}{2l} \int_{-l}^{l} f(x) e^{-i\frac{n\pi x}{l}} dx$$

系数关系：复数形式与实数形式的系数关系为：

  • $c_0 = \frac{a_0}{2}$
  • $c_n = \frac{a_n - ib_n}{2}$（$n > 0$）
  • $c_{-n} = \frac{a_n + ib_n}{2} = \overline{c_n}$（$n > 0$）

复数形式的优势

简洁性：

  • 统一了正弦和余弦项的表示
  • 对称的求和形式：$\sum_{n=-\infty}^{\infty}$
  • 便于理论分析和计算

物理意义：

  • $e^{i\frac{n\pi x}{l}}$ 表示频率为 $\frac{n}{2l}$ 的复振荡
  • $|c_n|$ 表示第 $n$ 次谐波的振幅
  • $\arg(c_n)$ 表示第 $n$ 次谐波的相位

频谱分析：

$$|c_n|^2 = \frac{a_n^2 + b_n^2}{4}$$

表示第 $n$ 次谐波的功率。

形式 表达式 系数公式 适用场合
实数形式 $\frac{a_0}{2} + \sum_{n=1}^{\infty} (a_n \cos \frac{n\pi x}{l} + b_n \sin \frac{n\pi x}{l})$ $a_n, b_n$ 分别计算 工程应用，直观理解
复数形式 $\sum_{n=-\infty}^{\infty} c_n e^{i\frac{n\pi x}{l}}$ $c_n = \frac{1}{2l} \int_{-l}^{l} f(x) e^{-i\frac{n\pi x}{l}} dx$ 理论分析，频谱分析

例4：矩形波的复数形式

求周期为 $2\pi$ 的矩形波函数 $f(x) = \begin{cases} 1, & 0 < x < \pi \\ -1, & -\pi < x < 0 \end{cases}$ 的复数形式傅里叶级数。

解：

步骤1：确定参数

周期 $T = 2\pi$，所以 $l = \pi$

步骤2：计算复数系数 $c_n$

$$c_n = \frac{1}{2\pi} \int_{-\pi}^{\pi} f(x) e^{-inx} dx$$ $$= \frac{1}{2\pi} \left[\int_{-\pi}^{0} (-1) e^{-inx} dx + \int_{0}^{\pi} 1 \cdot e^{-inx} dx\right]$$

当 $n \neq 0$ 时：

$$c_n = \frac{1}{2\pi} \left[\frac{e^{-inx}}{in}\Big|_{-\pi}^{0} - \frac{e^{-inx}}{in}\Big|_{0}^{\pi}\right]$$ $$= \frac{1}{2\pi} \left[\frac{1 - e^{in\pi}}{in} - \frac{e^{-in\pi} - 1}{in}\right]$$ $$= \frac{1}{2\pi} \cdot \frac{2(1 - \cos n\pi)}{in} = \frac{1 - (-1)^n}{in\pi}$$

因此：

$$c_n = \begin{cases} 0, & n = 0 \\ \frac{2}{in\pi}, & n \text{为奇数} \\ 0, & n \text{为偶数且} n \neq 0 \end{cases}$$

步骤3：写出复数形式

$$f(x) = \sum_{\substack{n=-\infty \\ n \text{为奇数}}}^{\infty} \frac{2}{in\pi} e^{inx}$$ $$= \frac{2}{\pi} \sum_{k=0}^{\infty} \frac{1}{i(2k+1)} \left[e^{i(2k+1)x} - e^{-i(2k+1)x}\right]$$ $$= \frac{4}{\pi} \sum_{k=0}^{\infty} \frac{\sin(2k+1)x}{2k+1}$$

验证：这与实数形式的结果一致。

例5：复数系数与实数系数的关系

已知函数的实数傅里叶级数为 $f(x) = 1 + 2\cos x + \sin x - \cos 2x + 3\sin 2x$，求其复数形式。

解：

步骤1：识别实数系数

  • $a_0 = 2$，所以 $c_0 = \frac{a_0}{2} = 1$
  • $a_1 = 2$，$b_1 = 1$
  • $a_2 = -1$，$b_2 = 3$

步骤2：计算复数系数

对于 $n = 1$：

$$c_1 = \frac{a_1 - ib_1}{2} = \frac{2 - i}{2} = 1 - \frac{i}{2}$$ $$c_{-1} = \overline{c_1} = 1 + \frac{i}{2}$$

对于 $n = 2$：

$$c_2 = \frac{a_2 - ib_2}{2} = \frac{-1 - 3i}{2} = -\frac{1}{2} - \frac{3i}{2}$$ $$c_{-2} = \overline{c_2} = -\frac{1}{2} + \frac{3i}{2}$$

步骤3：写出复数形式

$$f(x) = c_0 + c_1 e^{ix} + c_{-1} e^{-ix} + c_2 e^{2ix} + c_{-2} e^{-2ix}$$ $$= 1 + \left(1 - \frac{i}{2}\right)e^{ix} + \left(1 + \frac{i}{2}\right)e^{-ix}$$ $$+ \left(-\frac{1}{2} - \frac{3i}{2}\right)e^{2ix} + \left(-\frac{1}{2} + \frac{3i}{2}\right)e^{-2ix}$$

例6：频谱分析应用

对于周期信号 $f(t) = 2\cos(100\pi t) + 0.5\sin(300\pi t)$，分析其频谱特性。

解：

步骤1：转换为复数形式

利用欧拉公式：

$$\cos(100\pi t) = \frac{e^{i100\pi t} + e^{-i100\pi t}}{2}$$ $$\sin(300\pi t) = \frac{e^{i300\pi t} - e^{-i300\pi t}}{2i}$$

因此：

$$f(t) = 2 \cdot \frac{e^{i100\pi t} + e^{-i100\pi t}}{2} + 0.5 \cdot \frac{e^{i300\pi t} - e^{-i300\pi t}}{2i}$$ $$= e^{i100\pi t} + e^{-i100\pi t} - \frac{i}{4}e^{i300\pi t} + \frac{i}{4}e^{-i300\pi t}$$

步骤2：识别频率分量

  • 频率 $f_1 = \frac{100\pi}{2\pi} = 50$ Hz，幅度 $|c_1| = 1$
  • 频率 $f_2 = \frac{300\pi}{2\pi} = 150$ Hz，幅度 $|c_2| = \frac{1}{4}$

步骤3：功率谱

  • 50 Hz 分量的功率：$|c_1|^2 = 1$
  • 150 Hz 分量的功率：$|c_2|^2 = \frac{1}{16}$

结论：信号主要包含50 Hz和150 Hz两个频率分量，50 Hz分量的功率远大于150 Hz分量。
