4.1 不定积分的概念与性质

Indefinite Integrals: Concepts and Properties

4.1.1 核心概念

原函数的定义

设函数 $f(x)$ 在区间 $I$ 上有定义，如果存在函数 $F(x)$，使得在区间 $I$ 上有

$$F'(x) = f(x) \quad \text{或} \quad dF(x) = f(x)dx$$

则称 $F(x)$ 是 $f(x)$ 在区间 $I$ 上的一个原函数。

记忆要点

原函数就是"导数的逆"：如果 $F'(x) = f(x)$，那么 $F(x)$ 就是 $f(x)$ 的原函数。就像问题："什么函数的导数是 $x^2$？"答案是 $\frac{x^3}{3}$。

不定积分的定义

函数 $f(x)$ 在区间 $I$ 上的全体原函数称为 $f(x)$ 在区间 $I$ 上的不定积分，记作

$$\int f(x)dx = F(x) + C$$

其中 $F(x)$ 是 $f(x)$ 的一个原函数，$C$ 是任意常数，称为积分常数。

  • $\int$ 称为积分号
  • $f(x)$ 称为被积函数
  • $f(x)dx$ 称为被积表达式
  • $x$ 称为积分变量
积分符号的记忆

积分号 $\int$ 的由来：这是拉丁字母 "S" 的拉长形式，代表 "Sum"（求和），暗示积分本质上是无穷小量的求和。常数 $C$ 不能忘记，因为导数会"丢失"常数项！

4.1.2 基本积分表

基本积分公式（必须熟记！）

幂函数积分
$$\int x^n dx = \frac{x^{n+1}}{n+1} + C \quad (n \neq -1)$$
倒数积分
$$\int \frac{1}{x} dx = \ln|x| + C$$
自然指数
$$\int e^x dx = e^x + C$$
一般指数
$$\int a^x dx = \frac{a^x}{\ln a} + C \quad (a > 0, a \neq 1)$$
正弦积分
$$\int \sin x dx = -\cos x + C$$
余弦积分
$$\int \cos x dx = \sin x + C$$
正割平方
$$\int \sec^2 x dx = \tan x + C$$
余割平方
$$\int \csc^2 x dx = -\cot x + C$$
反正弦
$$\int \frac{1}{\sqrt{1-x^2}} dx = \arcsin x + C$$
反正切
$$\int \frac{1}{1+x^2} dx = \arctan x + C$$
公式记忆技巧

反向记忆法：每个积分公式都可以通过求导验证！比如 $\int x^2 dx = \frac{x^3}{3} + C$，验证：$(\frac{x^3}{3})'= x^2$ ✓

三角函数记忆：正弦积分得负余弦，余弦积分得正弦。记住"正负相反"！

4.1.3 不定积分的性质

不定积分的基本性质

  1. 线性性质（最重要！）：
    $$\int [kf(x) \pm lg(x)]dx = k\int f(x)dx \pm l\int g(x)dx$$
    其中 $k, l$ 是常数。
  2. 微分与积分的互逆性：
    $$\frac{d}{dx}\int f(x)dx = f(x)$$
    $$\int F'(x)dx = F(x) + C$$
  3. 积分与微分的关系：
    $$\int f(x)dx = F(x) + C \Leftrightarrow dF(x) = f(x)dx$$
性质应用技巧

线性性质是积分计算的基础：遇到复杂函数，先看能否拆分成简单函数的和差。比如 $\int (3x^2 + 2x - 1)dx$ 可以拆分为 $3\int x^2dx + 2\int x dx - \int 1 dx$。

4.1.4 典型例题

例题1：基本公式应用

问题：计算下列不定积分

$$\int (2x^3 - 5x^2 + 3x - 7)dx$$
解答过程：

利用线性性质，将积分拆分：

$$\int (2x^3 - 5x^2 + 3x - 7)dx = 2\int x^3dx - 5\int x^2dx + 3\int x dx - 7\int 1 dx$$

分别计算各项：

$$= 2 \cdot \frac{x^4}{4} - 5 \cdot \frac{x^3}{3} + 3 \cdot \frac{x^2}{2} - 7x + C$$
$$= \frac{x^4}{2} - \frac{5x^3}{3} + \frac{3x^2}{2} - 7x + C$$

例题2：三角函数积分

问题：计算下列不定积分

$$\int (3\sin x - 2\cos x + \sec^2 x)dx$$
解答过程：

分项积分：

$$\int (3\sin x - 2\cos x + \sec^2 x)dx = 3\int \sin x dx - 2\int \cos x dx + \int \sec^2 x dx$$

应用基本公式：

$$= 3(-\cos x) - 2(\sin x) + \tan x + C$$
$$= -3\cos x - 2\sin x + \tan x + C$$

4.1.6 练习题

习题1：基础计算 中等

计算下列不定积分：

$$\int (4x^5 - 3x^2 + 2x - 1)dx$$
解答：

分项积分：

$$= 4\int x^5dx - 3\int x^2dx + 2\int x dx - \int 1 dx$$
$$= 4 \cdot \frac{x^6}{6} - 3 \cdot \frac{x^3}{3} + 2 \cdot \frac{x^2}{2} - x + C$$
$$= \frac{2x^6}{3} - x^3 + x^2 - x + C$$

习题2：混合函数 较难

计算下列不定积分：

$$\int \left(\frac{2}{x} + 3e^x - \sin x + \frac{1}{1+x^2}\right)dx$$
解答：

识别各项对应的基本公式：

$$= 2\int \frac{1}{x}dx + 3\int e^x dx - \int \sin x dx + \int \frac{1}{1+x^2}dx$$
$$= 2\ln|x| + 3e^x - (-\cos x) + \arctan x + C$$
$$= 2\ln|x| + 3e^x + \cos x + \arctan x + C$$

习题3：验证与理解 很难

已知 $\int f(x)dx = x^3 - 2x^2 + x + C$，求：

  1. $f(x)$ 的表达式
  2. $f'(x)$ 的表达式
  3. $\int f'(x)dx$
解答：

(1) 由于 $\int f(x)dx = x^3 - 2x^2 + x + C$，根据微积分基本定理：

$$f(x) = \frac{d}{dx}(x^3 - 2x^2 + x + C) = 3x^2 - 4x + 1$$

(2) 对 $f(x)$ 求导：

$$f'(x) = \frac{d}{dx}(3x^2 - 4x + 1) = 6x - 4$$

(3) 计算 $f'(x)$ 的不定积分：

$$\int f'(x)dx = \int (6x - 4)dx = 3x^2 - 4x + C_1$$

注意：这个结果正好是 $f(x)$（相差一个常数）。
