9.4 多元复合函数的求导法

Differentiation of Composite Functions of Several Variables

9.4.1 复合函数的求导法和全导数

一元函数与多元函数复合
$z = f(u, v)$
$u = u(t), \quad v = v(t)$
$\Downarrow$
$z = z(t)$
多元函数与多元函数复合
$z = f(u, v)$
$u = u(x, y), \quad v = v(x, y)$
$\Downarrow$
$z = z(x, y)$

链式求导法则（一元与多元复合）

设函数 $z = f(u, v)$ 在点 $(u, v)$ 处可微，函数 $u = u(t)$、$v = v(t)$ 在点 $t$ 处可导，则复合函数 $z = f(u(t), v(t))$ 在点 $t$ 处可导，且

全导数公式
$$\frac{\mathrm{d}z}{\mathrm{d}t} = \frac{\partial z}{\partial u} \frac{\mathrm{d}u}{\mathrm{d}t} + \frac{\partial z}{\partial v} \frac{\mathrm{d}v}{\mathrm{d}t}$$

记忆方法：沿着每条"路径"求导，然后相加。从 $z$ 到 $t$ 有两条路径：$z \to u \to t$ 和 $z \to v \to t$。

链式求导法则（多元与多元复合）

设函数 $z = f(u, v)$ 在点 $(u, v)$ 处可微，函数 $u = u(x, y)$、$v = v(x, y)$ 在点 $(x, y)$ 处可微，则复合函数 $z = f(u(x, y), v(x, y))$ 在点 $(x, y)$ 处可微，且

偏导数公式
$$\frac{\partial z}{\partial x} = \frac{\partial z}{\partial u} \frac{\partial u}{\partial x} + \frac{\partial z}{\partial v} \frac{\partial v}{\partial x}$$
$$\frac{\partial z}{\partial y} = \frac{\partial z}{\partial u} \frac{\partial u}{\partial y} + \frac{\partial z}{\partial v} \frac{\partial v}{\partial y}$$
链式法则的一般形式

对于更复杂的复合函数 $z = f(u_1, u_2, \ldots, u_m)$，其中 $u_i = u_i(x_1, x_2, \ldots, x_n)$：

$$\frac{\partial z}{\partial x_j} = \sum_{i=1}^{m} \frac{\partial z}{\partial u_i} \frac{\partial u_i}{\partial x_j} \quad (j = 1, 2, \ldots, n)$$

这个公式包含了所有可能的复合情况。

例1：一元与多元复合函数求导

设 $z = u^2 + v^2$，$u = e^t$，$v = \sin t$，求 $\frac{\mathrm{d}z}{\mathrm{d}t}$。

解：

计算各偏导数：

$$\frac{\partial z}{\partial u} = 2u = 2e^t$$ $$\frac{\partial z}{\partial v} = 2v = 2\sin t$$

计算中间变量的导数：

$$\frac{\mathrm{d}u}{\mathrm{d}t} = e^t$$ $$\frac{\mathrm{d}v}{\mathrm{d}t} = \cos t$$

应用链式法则：

$$\frac{\mathrm{d}z}{\mathrm{d}t} = \frac{\partial z}{\partial u} \frac{\mathrm{d}u}{\mathrm{d}t} + \frac{\partial z}{\partial v} \frac{\mathrm{d}v}{\mathrm{d}t}$$ $$= 2e^t \cdot e^t + 2\sin t \cdot \cos t$$ $$= 2e^{2t} + \sin 2t$$

例2：多元与多元复合函数求导

设 $z = uv + u^2$，$u = x + y$，$v = xy$，求 $\frac{\partial z}{\partial x}$ 和 $\frac{\partial z}{\partial y}$。

解：

计算 $z$ 对中间变量的偏导数：

$$\frac{\partial z}{\partial u} = v + 2u = xy + 2(x + y)$$ $$\frac{\partial z}{\partial v} = u = x + y$$

计算中间变量对自变量的偏导数：

$$\frac{\partial u}{\partial x} = 1, \quad \frac{\partial u}{\partial y} = 1$$ $$\frac{\partial v}{\partial x} = y, \quad \frac{\partial v}{\partial y} = x$$

应用链式法则：

$$\frac{\partial z}{\partial x} = \frac{\partial z}{\partial u} \frac{\partial u}{\partial x} + \frac{\partial z}{\partial v} \frac{\partial v}{\partial x}$$ $$= [xy + 2(x + y)] \cdot 1 + (x + y) \cdot y$$ $$= xy + 2x + 2y + xy + y^2 = 2xy + 2x + 2y + y^2$$ $$\frac{\partial z}{\partial y} = \frac{\partial z}{\partial u} \frac{\partial u}{\partial y} + \frac{\partial z}{\partial v} \frac{\partial v}{\partial y}$$ $$= [xy + 2(x + y)] \cdot 1 + (x + y) \cdot x$$ $$= xy + 2x + 2y + x^2 + xy = 2xy + 2x + 2y + x^2$$

例3：含有多层复合的情况

设 $z = f(u)$，$u = x^2 + y^2$，$x = r\cos\theta$，$y = r\sin\theta$，求 $\frac{\partial z}{\partial r}$ 和 $\frac{\partial z}{\partial \theta}$。

解：

分析复合结构：$z = f(u(x(r,\theta), y(r,\theta)))$

这是三层复合：$z \to u \to (x,y) \to (r,\theta)$

计算各层的导数：

$$\frac{\partial z}{\partial u} = f'(u)$$ $$\frac{\partial u}{\partial x} = 2x, \quad \frac{\partial u}{\partial y} = 2y$$ $$\frac{\partial x}{\partial r} = \cos\theta, \quad \frac{\partial x}{\partial \theta} = -r\sin\theta$$ $$\frac{\partial y}{\partial r} = \sin\theta, \quad \frac{\partial y}{\partial \theta} = r\cos\theta$$

应用链式法则：

$$\frac{\partial z}{\partial r} = \frac{\partial z}{\partial u} \left(\frac{\partial u}{\partial x} \frac{\partial x}{\partial r} + \frac{\partial u}{\partial y} \frac{\partial y}{\partial r}\right)$$ $$= f'(u)(2x \cos\theta + 2y \sin\theta)$$ $$= f'(u) \cdot 2(x\cos\theta + y\sin\theta)$$ $$= f'(u) \cdot 2(r\cos^2\theta + r\sin^2\theta) = 2rf'(u)$$ $$\frac{\partial z}{\partial \theta} = \frac{\partial z}{\partial u} \left(\frac{\partial u}{\partial x} \frac{\partial x}{\partial \theta} + \frac{\partial u}{\partial y} \frac{\partial y}{\partial \theta}\right)$$ $$= f'(u)(2x(-r\sin\theta) + 2y(r\cos\theta))$$ $$= f'(u) \cdot 2r(-x\sin\theta + y\cos\theta)$$ $$= f'(u) \cdot 2r(-r\cos\theta\sin\theta + r\sin\theta\cos\theta) = 0$$

9.4.2 全微分形式不变性

全微分形式不变性定理

设函数 $z = f(u, v)$ 可微，无论 $u$、$v$ 是自变量还是中间变量，$z$ 的全微分形式都是：

$$\mathrm{d}z = \frac{\partial z}{\partial u}\mathrm{d}u + \frac{\partial z}{\partial v}\mathrm{d}v$$

这个性质称为全微分形式不变性。

形式不变性的含义

情况1：当 $u$、$v$ 是自变量时：

$$\mathrm{d}z = \frac{\partial z}{\partial u}\mathrm{d}u + \frac{\partial z}{\partial v}\mathrm{d}v$$

情况2：当 $u = u(x, y)$、$v = v(x, y)$ 是中间变量时：

$$\mathrm{d}z = \frac{\partial z}{\partial u}\mathrm{d}u + \frac{\partial z}{\partial v}\mathrm{d}v$$

其中 $\mathrm{d}u = \frac{\partial u}{\partial x}\mathrm{d}x + \frac{\partial u}{\partial y}\mathrm{d}y$，$\mathrm{d}v = \frac{\partial v}{\partial x}\mathrm{d}x + \frac{\partial v}{\partial y}\mathrm{d}y$

重要性：这个性质使得我们在计算复合函数的全微分时，可以一层一层地应用微分公式，而不需要先展开复合关系。

全微分形式不变性的应用

利用全微分形式不变性，可以简化复合函数偏导数的计算：

  1. 先写出函数的全微分形式
  2. 将中间变量的微分用自变量表示
  3. 整理得到最终的全微分
  4. 比较系数得到偏导数

例4：利用全微分形式不变性求偏导数

设 $z = f(u, v)$，$u = x + y$，$v = x - y$，其中 $f$ 具有连续的偏导数。用全微分形式不变性求 $\frac{\partial z}{\partial x}$ 和 $\frac{\partial z}{\partial y}$。

解：

由全微分形式不变性：

$$\mathrm{d}z = \frac{\partial f}{\partial u}\mathrm{d}u + \frac{\partial f}{\partial v}\mathrm{d}v$$

计算中间变量的微分：

$$\mathrm{d}u = \mathrm{d}(x + y) = \mathrm{d}x + \mathrm{d}y$$ $$\mathrm{d}v = \mathrm{d}(x - y) = \mathrm{d}x - \mathrm{d}y$$

代入得：

$$\mathrm{d}z = \frac{\partial f}{\partial u}(\mathrm{d}x + \mathrm{d}y) + \frac{\partial f}{\partial v}(\mathrm{d}x - \mathrm{d}y)$$ $$= \left(\frac{\partial f}{\partial u} + \frac{\partial f}{\partial v}\right)\mathrm{d}x + \left(\frac{\partial f}{\partial u} - \frac{\partial f}{\partial v}\right)\mathrm{d}y$$

比较系数得：

$$\frac{\partial z}{\partial x} = \frac{\partial f}{\partial u} + \frac{\partial f}{\partial v}$$ $$\frac{\partial z}{\partial y} = \frac{\partial f}{\partial u} - \frac{\partial f}{\partial v}$$

例5：验证全微分形式不变性

设 $z = u^2 + uv$，$u = x^2$，$v = xy$，分别用直接法和全微分形式不变性求 $\mathrm{d}z$。

解：

方法1：直接法

将中间变量代入：

$$z = (x^2)^2 + (x^2)(xy) = x^4 + x^3y$$

求偏导数：

$$\frac{\partial z}{\partial x} = 4x^3 + 3x^2y, \quad \frac{\partial z}{\partial y} = x^3$$

全微分：

$$\mathrm{d}z = (4x^3 + 3x^2y)\mathrm{d}x + x^3\mathrm{d}y$$

方法2：全微分形式不变性

计算 $z$ 对中间变量的偏导数：

$$\frac{\partial z}{\partial u} = 2u + v = 2x^2 + xy$$ $$\frac{\partial z}{\partial v} = u = x^2$$

计算中间变量的微分：

$$\mathrm{d}u = 2x\mathrm{d}x, \quad \mathrm{d}v = y\mathrm{d}x + x\mathrm{d}y$$

应用形式不变性：

$$\mathrm{d}z = \frac{\partial z}{\partial u}\mathrm{d}u + \frac{\partial z}{\partial v}\mathrm{d}v$$ $$= (2x^2 + xy)(2x\mathrm{d}x) + x^2(y\mathrm{d}x + x\mathrm{d}y)$$ $$= (4x^3 + 2x^2y)\mathrm{d}x + x^2y\mathrm{d}x + x^3\mathrm{d}y$$ $$= (4x^3 + 3x^2y)\mathrm{d}x + x^3\mathrm{d}y$$

结论：两种方法得到相同的结果，验证了全微分形式不变性。

练习题

习题1 中等难度

设 $z = e^{uv}$，$u = x + y$，$v = x - y$，求 $\frac{\partial z}{\partial x}$ 和 $\frac{\partial z}{\partial y}$。

解：

计算 $z$ 对中间变量的偏导数：

$$\frac{\partial z}{\partial u} = ve^{uv} = (x-y)e^{(x+y)(x-y)} = (x-y)e^{x^2-y^2}$$ $$\frac{\partial z}{\partial v} = ue^{uv} = (x+y)e^{(x+y)(x-y)} = (x+y)e^{x^2-y^2}$$

计算中间变量对自变量的偏导数：

$$\frac{\partial u}{\partial x} = 1, \quad \frac{\partial u}{\partial y} = 1$$ $$\frac{\partial v}{\partial x} = 1, \quad \frac{\partial v}{\partial y} = -1$$

应用链式法则：

$$\frac{\partial z}{\partial x} = \frac{\partial z}{\partial u} \frac{\partial u}{\partial x} + \frac{\partial z}{\partial v} \frac{\partial v}{\partial x}$$ $$= (x-y)e^{x^2-y^2} \cdot 1 + (x+y)e^{x^2-y^2} \cdot 1$$ $$= [(x-y) + (x+y)]e^{x^2-y^2} = 2xe^{x^2-y^2}$$ $$\frac{\partial z}{\partial y} = \frac{\partial z}{\partial u} \frac{\partial u}{\partial y} + \frac{\partial z}{\partial v} \frac{\partial v}{\partial y}$$ $$= (x-y)e^{x^2-y^2} \cdot 1 + (x+y)e^{x^2-y^2} \cdot (-1)$$ $$= [(x-y) - (x+y)]e^{x^2-y^2} = -2ye^{x^2-y^2}$$

习题2 较难

设 $z = f(x+y, xy)$，其中 $f$ 具有连续的二阶偏导数，求 $\frac{\partial^2 z}{\partial x \partial y}$。

解：

设 $u = x + y$，$v = xy$，则 $z = f(u, v)$

首先求 $\frac{\partial z}{\partial x}$：

$$\frac{\partial z}{\partial x} = \frac{\partial f}{\partial u} \frac{\partial u}{\partial x} + \frac{\partial f}{\partial v} \frac{\partial v}{\partial x}$$ $$= \frac{\partial f}{\partial u} \cdot 1 + \frac{\partial f}{\partial v} \cdot y = \frac{\partial f}{\partial u} + y\frac{\partial f}{\partial v}$$

对 $y$ 求偏导数：

$$\frac{\partial^2 z}{\partial y \partial x} = \frac{\partial}{\partial y}\left(\frac{\partial f}{\partial u} + y\frac{\partial f}{\partial v}\right)$$

计算各项：

$$\frac{\partial}{\partial y}\left(\frac{\partial f}{\partial u}\right) = \frac{\partial^2 f}{\partial u^2} \frac{\partial u}{\partial y} + \frac{\partial^2 f}{\partial u \partial v} \frac{\partial v}{\partial y}$$ $$= \frac{\partial^2 f}{\partial u^2} \cdot 1 + \frac{\partial^2 f}{\partial u \partial v} \cdot x = \frac{\partial^2 f}{\partial u^2} + x\frac{\partial^2 f}{\partial u \partial v}$$ $$\frac{\partial}{\partial y}\left(y\frac{\partial f}{\partial v}\right) = \frac{\partial f}{\partial v} + y\frac{\partial}{\partial y}\left(\frac{\partial f}{\partial v}\right)$$ $$= \frac{\partial f}{\partial v} + y\left(\frac{\partial^2 f}{\partial v \partial u} \frac{\partial u}{\partial y} + \frac{\partial^2 f}{\partial v^2} \frac{\partial v}{\partial y}\right)$$ $$= \frac{\partial f}{\partial v} + y\left(\frac{\partial^2 f}{\partial u \partial v} + x\frac{\partial^2 f}{\partial v^2}\right)$$

合并结果：

$$\frac{\partial^2 z}{\partial x \partial y} = \frac{\partial^2 f}{\partial u^2} + x\frac{\partial^2 f}{\partial u \partial v} + \frac{\partial f}{\partial v} + y\frac{\partial^2 f}{\partial u \partial v} + xy\frac{\partial^2 f}{\partial v^2}$$ $$= \frac{\partial^2 f}{\partial u^2} + (x+y)\frac{\partial^2 f}{\partial u \partial v} + \frac{\partial f}{\partial v} + xy\frac{\partial^2 f}{\partial v^2}$$

习题3 很难

设 $u = f(x, y, z)$，$x = r\sin\phi\cos\theta$，$y = r\sin\phi\sin\theta$，$z = r\cos\phi$，其中 $f$ 具有连续的偏导数。证明： $$\left(\frac{\partial u}{\partial r}\right)^2 + \frac{1}{r^2}\left(\frac{\partial u}{\partial \phi}\right)^2 + \frac{1}{r^2\sin^2\phi}\left(\frac{\partial u}{\partial \theta}\right)^2 = \left(\frac{\partial u}{\partial x}\right)^2 + \left(\frac{\partial u}{\partial y}\right)^2 + \left(\frac{\partial u}{\partial z}\right)^2$$

解：

计算雅可比矩阵的元素：

$$\frac{\partial x}{\partial r} = \sin\phi\cos\theta, \quad \frac{\partial x}{\partial \phi} = r\cos\phi\cos\theta, \quad \frac{\partial x}{\partial \theta} = -r\sin\phi\sin\theta$$ $$\frac{\partial y}{\partial r} = \sin\phi\sin\theta, \quad \frac{\partial y}{\partial \phi} = r\cos\phi\sin\theta, \quad \frac{\partial y}{\partial \theta} = r\sin\phi\cos\theta$$ $$\frac{\partial z}{\partial r} = \cos\phi, \quad \frac{\partial z}{\partial \phi} = -r\sin\phi, \quad \frac{\partial z}{\partial \theta} = 0$$

由链式法则：

$$\frac{\partial u}{\partial r} = \frac{\partial u}{\partial x}\sin\phi\cos\theta + \frac{\partial u}{\partial y}\sin\phi\sin\theta + \frac{\partial u}{\partial z}\cos\phi$$ $$\frac{\partial u}{\partial \phi} = \frac{\partial u}{\partial x}r\cos\phi\cos\theta + \frac{\partial u}{\partial y}r\cos\phi\sin\theta - \frac{\partial u}{\partial z}r\sin\phi$$ $$\frac{\partial u}{\partial \theta} = -\frac{\partial u}{\partial x}r\sin\phi\sin\theta + \frac{\partial u}{\partial y}r\sin\phi\cos\theta$$

计算左边各项的平方：

$$\left(\frac{\partial u}{\partial r}\right)^2 = \left(\frac{\partial u}{\partial x}\right)^2\sin^2\phi\cos^2\theta + \left(\frac{\partial u}{\partial y}\right)^2\sin^2\phi\sin^2\theta + \left(\frac{\partial u}{\partial z}\right)^2\cos^2\phi$$ $$ + 2\frac{\partial u}{\partial x}\frac{\partial u}{\partial y}\sin^2\phi\cos\theta\sin\theta + 2\frac{\partial u}{\partial x}\frac{\partial u}{\partial z}\sin\phi\cos\phi\cos\theta + 2\frac{\partial u}{\partial y}\frac{\partial u}{\partial z}\sin\phi\cos\phi\sin\theta$$

继续计算（过程较长，这里给出关键步骤）：

经过详细计算，可以验证：

$$\left(\frac{\partial u}{\partial r}\right)^2 + \frac{1}{r^2}\left(\frac{\partial u}{\partial \phi}\right)^2 + \frac{1}{r^2\sin^2\phi}\left(\frac{\partial u}{\partial \theta}\right)^2$$

展开后，所有交叉项相互抵消，只剩下：

$$= \left(\frac{\partial u}{\partial x}\right)^2(\sin^2\phi\cos^2\theta + \cos^2\phi\cos^2\theta + \sin^2\theta)$$ $$+ \left(\frac{\partial u}{\partial y}\right)^2(\sin^2\phi\sin^2\theta + \cos^2\phi\sin^2\theta + \cos^2\theta)$$ $$+ \left(\frac{\partial u}{\partial z}\right)^2(\cos^2\phi + \sin^2\phi)$$ $$= \left(\frac{\partial u}{\partial x}\right)^2 + \left(\frac{\partial u}{\partial y}\right)^2 + \left(\frac{\partial u}{\partial z}\right)^2$$

说明：这个等式表明梯度的模长在坐标变换下保持不变，体现了物理量的客观性。
