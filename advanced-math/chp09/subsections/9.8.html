9.8 多元函数的极值及其求法

Extrema of Multivariable Functions and Their Methods

9.8.1 多元函数的极值与最值，驻点

多元函数的极值

设函数 $z = f(x, y)$ 在点 $(x_0, y_0)$ 的某个邻域内有定义，如果对于该邻域内异于 $(x_0, y_0)$ 的任意点 $(x, y)$，恒有：

  • $f(x, y) < f(x_0, y_0)$，则称 $f(x_0, y_0)$ 为函数的一个极大值
  • $f(x, y) > f(x_0, y_0)$，则称 $f(x_0, y_0)$ 为函数的一个极小值

极大值和极小值统称为极值，使函数取得极值的点称为极值点。

驻点（临界点）

设函数 $z = f(x, y)$ 在点 $(x_0, y_0)$ 处可偏导，如果：

$$f_x(x_0, y_0) = 0 \quad \text{且} \quad f_y(x_0, y_0) = 0$$

则称点 $(x_0, y_0)$ 为函数 $f(x, y)$ 的一个驻点（或临界点、稳定点）。

极值的必要条件

设函数 $z = f(x, y)$ 在点 $(x_0, y_0)$ 处可偏导，且在该点取得极值，则：

$$f_x(x_0, y_0) = 0 \quad \text{且} \quad f_y(x_0, y_0) = 0$$

说明：这个定理表明，在可偏导的极值点处，偏导数必须为零，即极值点必定是驻点。但驻点不一定是极值点。

极值的充分条件

设函数 $z = f(x, y)$ 在点 $(x_0, y_0)$ 的某个邻域内连续且有一阶和二阶连续偏导数，又 $f_x(x_0, y_0) = f_y(x_0, y_0) = 0$，记：

$$A = f_{xx}(x_0, y_0), \quad B = f_{xy}(x_0, y_0), \quad C = f_{yy}(x_0, y_0)$$

令 $\Delta = AC - B^2$，则：

  • $\Delta > 0$：$(x_0, y_0)$ 是极值点
    • 当 $A < 0$（或 $C < 0$）时，$(x_0, y_0)$ 是极大值点
    • 当 $A > 0$（或 $C > 0$）时，$(x_0, y_0)$ 是极小值点
  • $\Delta < 0$：$(x_0, y_0)$ 不是极值点（鞍点）
  • $\Delta = 0$：无法判定，需要进一步分析
求无约束极值的步骤
  1. 求出函数 $f(x, y)$ 的所有驻点，即解方程组： $$\begin{cases} f_x(x, y) = 0 \\ f_y(x, y) = 0 \end{cases}$$
  2. 对每个驻点 $(x_0, y_0)$，计算二阶偏导数： $$A = f_{xx}(x_0, y_0), \quad B = f_{xy}(x_0, y_0), \quad C = f_{yy}(x_0, y_0)$$
  3. 计算判别式 $\Delta = AC - B^2$
  4. 根据判别式的符号判断极值的性质
  5. 如果是边界问题，还需检查边界上的最值

最大值与最小值

设函数 $z = f(x, y)$ 在有界闭区域 $D$ 上连续，则：

  • 函数在 $D$ 上必定有最大值和最小值
  • 最值可能在区域 $D$ 的内部的驻点处取得
  • 最值可能在区域 $D$ 的边界上取得

求最值的策略：

  1. 求出 $D$ 内部的所有驻点，计算函数值
  2. 求出边界上的最值（通常化为一元函数的最值问题）
  3. 比较所有候选点的函数值，确定最大值和最小值

例1：求二元函数的极值

求函数 $f(x, y) = x^3 + y^3 - 3xy$ 的极值。

解：

第一步：求驻点

计算一阶偏导数：

$$f_x = 3x^2 - 3y = 3(x^2 - y)$$ $$f_y = 3y^2 - 3x = 3(y^2 - x)$$

解方程组：

$$\begin{cases} x^2 - y = 0 \\ y^2 - x = 0 \end{cases}$$

由第一个方程得 $y = x^2$，代入第二个方程：

$$(x^2)^2 - x = 0 \Rightarrow x^4 - x = 0 \Rightarrow x(x^3 - 1) = 0$$

解得 $x = 0$ 或 $x = 1$

  • 当 $x = 0$ 时，$y = 0^2 = 0$，得驻点 $(0, 0)$
  • 当 $x = 1$ 时，$y = 1^2 = 1$，得驻点 $(1, 1)$

第二步：判断极值性质

计算二阶偏导数：

$$f_{xx} = 6x, \quad f_{xy} = -3, \quad f_{yy} = 6y$$

对于点 $(0, 0)$：

$$A = f_{xx}(0, 0) = 0, \quad B = f_{xy}(0, 0) = -3, \quad C = f_{yy}(0, 0) = 0$$ $$\Delta = AC - B^2 = 0 \cdot 0 - (-3)^2 = -9 < 0$$

所以 $(0, 0)$ 不是极值点，是鞍点。

对于点 $(1, 1)$：

$$A = f_{xx}(1, 1) = 6, \quad B = f_{xy}(1, 1) = -3, \quad C = f_{yy}(1, 1) = 6$$ $$\Delta = AC - B^2 = 6 \cdot 6 - (-3)^2 = 36 - 9 = 27 > 0$$

由于 $\Delta > 0$ 且 $A = 6 > 0$，所以 $(1, 1)$ 是极小值点。

极小值为：$f(1, 1) = 1 + 1 - 3 = -1$

例2：有界闭区域上的最值

求函数 $f(x, y) = x^2 + y^2 - 2x - 4y + 5$ 在区域 $D: x^2 + y^2 \leq 4$ 上的最大值和最小值。

解：

第一步：求区域内部的驻点

$$f_x = 2x - 2 = 0 \Rightarrow x = 1$$ $$f_y = 2y - 4 = 0 \Rightarrow y = 2$$

驻点为 $(1, 2)$。检验：$1^2 + 2^2 = 5 > 4$，所以 $(1, 2)$ 在区域外，区域内部无驻点。

第二步：求边界上的最值

边界为 $x^2 + y^2 = 4$，设 $x = 2\cos t$，$y = 2\sin t$，$t \in [0, 2\pi]$

代入目标函数：

$$f(2\cos t, 2\sin t) = 4\cos^2 t + 4\sin^2 t - 4\cos t - 8\sin t + 5$$ $$= 4 - 4\cos t - 8\sin t + 5 = 9 - 4(\cos t + 2\sin t)$$

设 $g(t) = \cos t + 2\sin t$，求其最值：

$$g'(t) = -\sin t + 2\cos t = 0 \Rightarrow \tan t = 2$$

设 $\tan t = 2$，则 $\sin t = \frac{2}{\sqrt{5}}$，$\cos t = \frac{1}{\sqrt{5}}$（第一象限解）

$$g_{\max} = \frac{1}{\sqrt{5}} + 2 \cdot \frac{2}{\sqrt{5}} = \frac{5}{\sqrt{5}} = \sqrt{5}$$

类似地，$g_{\min} = -\sqrt{5}$

第三步：确定最值

$$f_{\min} = 9 - 4\sqrt{5}, \quad f_{\max} = 9 + 4\sqrt{5}$$

最小值点：$x = \frac{2}{\sqrt{5}}$，$y = \frac{4}{\sqrt{5}}$

最大值点：$x = -\frac{2}{\sqrt{5}}$，$y = -\frac{4}{\sqrt{5}}$

9.8.2 条件极值，拉格朗日乘数法

条件极值

在约束条件 $\varphi(x, y) = 0$ 下，求函数 $f(x, y)$ 的极值，这类问题称为条件极值问题。

几何意义：在曲线 $\varphi(x, y) = 0$ 上，求函数 $f(x, y)$ 的最大值和最小值。

与无约束极值的区别：

  • 无约束极值：在整个定义域内寻找极值
  • 条件极值：仅在满足约束条件的点集上寻找极值

拉格朗日乘数法

求函数 $f(x, y)$ 在约束条件 $\varphi(x, y) = 0$ 下的条件极值，可以构造拉格朗日函数：

$$L(x, y, \lambda) = f(x, y) + \lambda \varphi(x, y)$$

其中 $\lambda$ 称为拉格朗日乘数。

必要条件：如果 $(x_0, y_0)$ 是条件极值点，且 $\varphi_x(x_0, y_0)$、$\varphi_y(x_0, y_0)$ 不全为零，则存在 $\lambda_0$，使得：

$$\begin{cases} L_x = f_x + \lambda \varphi_x = 0 \\ L_y = f_y + \lambda \varphi_y = 0 \\ L_\lambda = \varphi(x, y) = 0 \end{cases}$$
拉格朗日乘数法步骤
  1. 构造拉格朗日函数： $$L(x, y, \lambda) = f(x, y) + \lambda \varphi(x, y)$$
  2. 建立方程组： $$\begin{cases} L_x = 0 \\ L_y = 0 \\ L_\lambda = 0 \end{cases}$$
  3. 解方程组，得到所有可能的极值点
  4. 比较各点的函数值，确定最值
  5. 根据实际问题的意义验证结果
多约束条件情况

对于 $m$ 个约束条件：$\varphi_i(x_1, \ldots, x_n) = 0$，$i = 1, 2, \ldots, m$

拉格朗日函数为：

$$L = f(x_1, \ldots, x_n) + \sum_{i=1}^m \lambda_i \varphi_i(x_1, \ldots, x_n)$$

求解 $n+m$ 个方程：

$$\frac{\partial L}{\partial x_j} = 0, \quad j = 1, 2, \ldots, n$$ $$\frac{\partial L}{\partial \lambda_i} = 0, \quad i = 1, 2, \ldots, m$$

几何解释

在条件极值点处，目标函数 $f(x, y)$ 的梯度与约束函数 $\varphi(x, y)$ 的梯度共线：

$$\nabla f = -\lambda \nabla \varphi$$

即：$\nabla f \parallel \nabla \varphi$

几何意义：在约束曲线上，目标函数的等值线与约束曲线相切的点就是条件极值点。

例3：拉格朗日乘数法求条件极值

求函数 $f(x, y) = xy$ 在约束条件 $x^2 + y^2 = 1$ 下的极值。

解：

第一步：构造拉格朗日函数

$$L(x, y, \lambda) = xy + \lambda(x^2 + y^2 - 1)$$

第二步：建立方程组

$$\begin{cases} L_x = y + 2\lambda x = 0 \\ L_y = x + 2\lambda y = 0 \\ L_\lambda = x^2 + y^2 - 1 = 0 \end{cases}$$

第三步：解方程组

从前两个方程得：

$$y = -2\lambda x, \quad x = -2\lambda y$$

将第一式代入第二式：$x = -2\lambda(-2\lambda x) = 4\lambda^2 x$

若 $x \neq 0$，则 $1 = 4\lambda^2$，即 $\lambda = \pm \frac{1}{2}$

情况1：$\lambda = \frac{1}{2}$

$$y = -2 \cdot \frac{1}{2} \cdot x = -x$$

代入约束条件：$x^2 + (-x)^2 = 2x^2 = 1$，得 $x = \pm \frac{1}{\sqrt{2}}$

得到驻点：$\left(\frac{1}{\sqrt{2}}, -\frac{1}{\sqrt{2}}\right)$ 和 $\left(-\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}\right)$

情况2：$\lambda = -\frac{1}{2}$

$$y = -2 \cdot \left(-\frac{1}{2}\right) \cdot x = x$$

代入约束条件：$x^2 + x^2 = 2x^2 = 1$，得 $x = \pm \frac{1}{\sqrt{2}}$

得到驻点：$\left(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}\right)$ 和 $\left(-\frac{1}{\sqrt{2}}, -\frac{1}{\sqrt{2}}\right)$

第四步：比较函数值

  • $f\left(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}\right) = \frac{1}{\sqrt{2}} \cdot \frac{1}{\sqrt{2}} = \frac{1}{2}$（最大值）
  • $f\left(-\frac{1}{\sqrt{2}}, -\frac{1}{\sqrt{2}}\right) = \left(-\frac{1}{\sqrt{2}}\right) \cdot \left(-\frac{1}{\sqrt{2}}\right) = \frac{1}{2}$（最大值）
  • $f\left(\frac{1}{\sqrt{2}}, -\frac{1}{\sqrt{2}}\right) = \frac{1}{\sqrt{2}} \cdot \left(-\frac{1}{\sqrt{2}}\right) = -\frac{1}{2}$（最小值）
  • $f\left(-\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}\right) = \left(-\frac{1}{\sqrt{2}}\right) \cdot \frac{1}{\sqrt{2}} = -\frac{1}{2}$（最小值）

结论：条件极大值为 $\frac{1}{2}$，条件极小值为 $-\frac{1}{2}$。

例4：实际应用问题

用铁丝围成一个长方体框架，已知铁丝长度为 $12$ 米，问长方体的长、宽、高各为多少时，体积最大？

解：

设长方体的长、宽、高分别为 $x$、$y$、$z$。

目标函数：$V = xyz$（体积最大）

约束条件：$4(x + y + z) = 12$，即 $x + y + z = 3$

构造拉格朗日函数：

$$L(x, y, z, \lambda) = xyz + \lambda(x + y + z - 3)$$

建立方程组：

$$\begin{cases} L_x = yz + \lambda = 0 \\ L_y = xz + \lambda = 0 \\ L_z = xy + \lambda = 0 \\ L_\lambda = x + y + z - 3 = 0 \end{cases}$$

从前三个方程得：$yz = xz = xy = -\lambda$

由于 $x, y, z > 0$，所以 $y = x$，$z = x$，即 $x = y = z$

代入约束条件：$x + x + x = 3$，得 $x = 1$

结论：当长方体的长、宽、高都等于 $1$ 米时，体积最大，最大体积为 $1$ 立方米。
