9.7 方向导数与梯度

Directional Derivatives and Gradient

9.7.1 方向导数

方向导数的定义

设函数 $z = f(x, y)$ 在点 $P_0(x_0, y_0)$ 的某一邻域内有定义，$\vec{l}$ 是从点 $P_0$ 出发的射线 $P_0P$ 的方向。如果函数 $f(x, y)$ 在点 $P_0$ 沿方向 $\vec{l}$ 的方向导数存在，则定义为：

$$\frac{\partial f}{\partial \vec{l}}\bigg|_{P_0} = \lim_{\rho \to 0^+} \frac{f(x_0 + \rho \cos \alpha, y_0 + \rho \sin \alpha) - f(x_0, y_0)}{\rho}$$

其中 $\alpha$ 是方向 $\vec{l}$ 与 $x$ 轴正向的夹角，$\rho$ 是 $|P_0P|$ 的长度。

几何意义：方向导数表示函数在该点沿指定方向的瞬时变化率，即函数值沿该方向增加的速度。

方向导数的计算公式

如果函数 $z = f(x, y)$ 在点 $P_0(x_0, y_0)$ 可微分，则 $f(x, y)$ 在该点沿任一方向 $\vec{l}$ 的方向导数都存在，且有：

$$\frac{\partial f}{\partial \vec{l}}\bigg|_{P_0} = f_x(x_0, y_0) \cos \alpha + f_y(x_0, y_0) \cos \beta$$

其中 $\cos \alpha$ 和 $\cos \beta$ 是方向 $\vec{l}$ 的方向余弦。

推广到三元函数：对于三元函数 $u = f(x, y, z)$，在点 $P_0(x_0, y_0, z_0)$ 沿方向 $\vec{l}(\cos \alpha, \cos \beta, \cos \gamma)$ 的方向导数为：

$$\frac{\partial f}{\partial \vec{l}}\bigg|_{P_0} = f_x(x_0, y_0, z_0) \cos \alpha + f_y(x_0, y_0, z_0) \cos \beta + f_z(x_0, y_0, z_0) \cos \gamma$$

例1：计算方向导数

求函数 $f(x, y) = x^2 + 2xy - y^2$ 在点 $P_0(1, 1)$ 沿从该点到点 $A(2, 3)$ 的方向的方向导数。

解：

步骤1：求偏导数

$$f_x = 2x + 2y, \quad f_y = 2x - 2y$$

在点 $P_0(1, 1)$：$f_x(1, 1) = 4$，$f_y(1, 1) = 0$

步骤2：确定方向

从 $P_0(1, 1)$ 到 $A(2, 3)$ 的向量：$\vec{P_0A} = (1, 2)$

单位方向向量：$\vec{l} = \frac{(1, 2)}{\sqrt{1^2 + 2^2}} = \frac{(1, 2)}{\sqrt{5}}$

所以：$\cos \alpha = \frac{1}{\sqrt{5}}$，$\sin \alpha = \frac{2}{\sqrt{5}}$

步骤3：计算方向导数

$$\frac{\partial f}{\partial \vec{l}}\bigg|_{P_0} = 4 \cdot \frac{1}{\sqrt{5}} + 0 \cdot \frac{2}{\sqrt{5}} = \frac{4}{\sqrt{5}} = \frac{4\sqrt{5}}{5}$$

例2：三元函数的方向导数

求函数 $u = xyz$ 在点 $M(1, 1, 1)$ 沿方向 $\vec{l} = (1, 2, 2)$ 的方向导数。

解：

步骤1：求偏导数

$$u_x = yz, \quad u_y = xz, \quad u_z = xy$$

在点 $M(1, 1, 1)$：$u_x(1, 1, 1) = 1$，$u_y(1, 1, 1) = 1$，$u_z(1, 1, 1) = 1$

步骤2：方向余弦

$|\vec{l}| = \sqrt{1^2 + 2^2 + 2^2} = 3$

方向余弦：$\cos \alpha = \frac{1}{3}$，$\cos \beta = \frac{2}{3}$，$\cos \gamma = \frac{2}{3}$

步骤3：计算方向导数

$$\frac{\partial u}{\partial \vec{l}}\bigg|_M = 1 \cdot \frac{1}{3} + 1 \cdot \frac{2}{3} + 1 \cdot \frac{2}{3} = \frac{5}{3}$$

9.7.2 梯度，微分算子，等值线，等值面

梯度的定义

设函数 $z = f(x, y)$ 在平面区域 $D$ 内具有一阶连续偏导数，则对于每一点 $P(x, y) \in D$，都可定义一个向量：

$$\text{grad } f(x, y) = \nabla f(x, y) = f_x(x, y) \vec{i} + f_y(x, y) \vec{j}$$

这个向量称为函数 $f(x, y)$ 在点 $P(x, y)$ 的梯度，记作 $\text{grad } f$ 或 $\nabla f$。

三元函数的梯度：

$$\nabla f(x, y, z) = f_x(x, y, z) \vec{i} + f_y(x, y, z) \vec{j} + f_z(x, y, z) \vec{k}$$

梯度的重要性质

  1. 方向导数与梯度的关系： $$\frac{\partial f}{\partial \vec{l}} = \nabla f \cdot \vec{e}_l = |\nabla f| \cos \theta$$ 其中 $\vec{e}_l$ 是方向 $\vec{l}$ 的单位向量，$\theta$ 是梯度与方向 $\vec{l}$ 的夹角。
  2. 梯度的方向：梯度的方向是函数值增加最快的方向。
  3. 梯度的模：$|\nabla f|$ 等于函数在该点的最大方向导数。
  4. 梯度与等值线垂直：在二维情况下，梯度垂直于通过该点的等值线；在三维情况下，梯度垂直于通过该点的等值面。

微分算子（Hamilton算子）

符号 $\nabla$（读作"nabla"或"del"）称为微分算子或Hamilton算子，定义为：

$$\nabla = \frac{\partial}{\partial x} \vec{i} + \frac{\partial}{\partial y} \vec{j} + \frac{\partial}{\partial z} \vec{k}$$

这样，梯度可以写成：$\nabla f = \nabla \cdot f$

等值线
$f(x, y) = C$

在二维平面上，函数值相等的点构成的曲线称为等值线或等高线。梯度垂直于等值线，指向函数值增加的方向。

等值面
$f(x, y, z) = C$

在三维空间中，函数值相等的点构成的曲面称为等值面或等位面。梯度垂直于等值面，指向函数值增加的方向。

例3：计算梯度

求函数 $f(x, y) = x^2 + y^2$ 在点 $(1, 2)$ 的梯度，并说明其几何意义。

解：

步骤1：计算偏导数

$$f_x = 2x, \quad f_y = 2y$$

步骤2：求梯度

在点 $(1, 2)$：

$$\nabla f(1, 2) = 2 \vec{i} + 4 \vec{j} = (2, 4)$$

步骤3：几何意义

  • 梯度的方向：从 $(1, 2)$ 指向 $(2, 4)$ 方向（单位化后）
  • 梯度的模：$|\nabla f(1, 2)| = \sqrt{2^2 + 4^2} = 2\sqrt{5}$
  • 这表示在点 $(1, 2)$ 处，函数沿梯度方向的方向导数最大，值为 $2\sqrt{5}$
  • 梯度垂直于通过点 $(1, 2)$ 的等值线 $x^2 + y^2 = 5$

例4：利用梯度求最大方向导数

求函数 $u = x^2 + y^2 + z^2$ 在点 $M(1, 1, 1)$ 处的最大方向导数及其方向。

解：

步骤1：计算梯度

$$u_x = 2x, \quad u_y = 2y, \quad u_z = 2z$$

在点 $M(1, 1, 1)$：

$$\nabla u(1, 1, 1) = (2, 2, 2)$$

步骤2：最大方向导数

最大方向导数等于梯度的模：

$$|\nabla u(1, 1, 1)| = \sqrt{2^2 + 2^2 + 2^2} = 2\sqrt{3}$$

步骤3：最大方向导数的方向

梯度的方向就是最大方向导数的方向：

$$\frac{\nabla u}{|\nabla u|} = \frac{(2, 2, 2)}{2\sqrt{3}} = \frac{(1, 1, 1)}{\sqrt{3}}$$

9.7.3 数量场，向量场，势场

数量场（标量场）

如果对于空间区域内的每一点 $M(x, y, z)$，都有一个确定的数量（标量）$f(x, y, z)$ 与之对应，则称在这个区域内确定了一个数量场。

例子：温度场、密度场、电势场等。

向量场

如果对于空间区域内的每一点 $M(x, y, z)$，都有一个确定的向量 $\vec{F}(x, y, z)$ 与之对应，则称在这个区域内确定了一个向量场。

例子：速度场、力场、电场、磁场等。

势场

如果向量场 $\vec{F}(x, y, z)$ 存在数量函数 $u(x, y, z)$，使得： $$\vec{F} = -\nabla u$$ 则称 $\vec{F}$ 为势场，$u$ 称为势函数。

例子：保守力场、静电场等。

标量场与向量场可视化

标量场性质： 每点对应一个数值
向量场性质： 每点对应一个向量
物理意义： 温度分布场

梯度场

给定数量场 $u = f(x, y, z)$，其梯度 $\nabla f$ 构成一个向量场，这个向量场称为原数量场的梯度场。

重要性质：

  • 梯度场是保守场（势场）
  • 梯度场的旋度恒为零：$\nabla \times (\nabla f) = \vec{0}$
  • 在梯度场中，沿任何闭合路径的线积分为零

梯度的运算法则

设 $u, v$ 是可微函数，$c$ 是常数，则：

  1. $\nabla c = \vec{0}$（常数的梯度为零向量）
  2. $\nabla(cu) = c\nabla u$（齐次性）
  3. $\nabla(u + v) = \nabla u + \nabla v$（可加性）
  4. $\nabla(uv) = u\nabla v + v\nabla u$（乘积法则）
  5. $\nabla\left(\frac{u}{v}\right) = \frac{v\nabla u - u\nabla v}{v^2}$（商法则，$v \neq 0$）
  6. $\nabla f(u) = f'(u)\nabla u$（复合函数法则）

例5：判断向量场是否为势场

判断向量场 $\vec{F}(x, y, z) = (2xy + z^2, x^2 + 2yz, 2xz + y^2)$ 是否为势场，如果是，求其势函数。

解：

步骤1：检验是否为势场

设 $\vec{F} = P\vec{i} + Q\vec{j} + R\vec{k}$，其中：

$$P = 2xy + z^2, \quad Q = x^2 + 2yz, \quad R = 2xz + y^2$$

检验 $\nabla \times \vec{F} = \vec{0}$：

$$\frac{\partial R}{\partial y} - \frac{\partial Q}{\partial z} = 2y - 2y = 0$$ $$\frac{\partial P}{\partial z} - \frac{\partial R}{\partial x} = 2z - 2z = 0$$ $$\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} = 2x - 2x = 0$$

由于旋度为零，所以 $\vec{F}$ 是势场。

步骤2：求势函数

设势函数为 $u(x, y, z)$，则 $\vec{F} = -\nabla u$，即：

$$-\frac{\partial u}{\partial x} = 2xy + z^2$$

积分得：$u = -x^2y - xz^2 + f(y, z)$

由 $-\frac{\partial u}{\partial y} = x^2 + 2yz$，得：

$$x^2 - \frac{\partial f}{\partial y} = x^2 + 2yz$$

所以 $\frac{\partial f}{\partial y} = -2yz$，积分得：$f(y, z) = -y^2z + C$

最终得到：

$$u(x, y, z) = -x^2y - xz^2 - y^2z + C$$
