5.3 定积分的换元法与分部积分法

Substitution and Integration by Parts for Definite Integrals

5.3.1 定积分的换元法

定积分的第一类换元法（凑微分法）

设函数 $f(x)$ 在区间 $[a,b]$ 上连续，$\varphi(x)$ 在 $[\alpha,\beta]$ 上连续可导，且：

  1. $\varphi(\alpha) = a$，$\varphi(\beta) = b$
  2. 当 $\alpha \leq t \leq \beta$ 时，$a \leq \varphi(t) \leq b$

则有：

$$\int_a^b f(x) dx = \int_\alpha^\beta f[\varphi(t)] \varphi'(t) dt$$

定积分的第二类换元法

设 $x = \psi(t)$ 是单调的连续可导函数，且 $\psi(\alpha) = a$，$\psi(\beta) = b$，则：

$$\int_a^b f(x) dx = \int_\alpha^\beta f[\psi(t)] \psi'(t) dt$$

注意：定积分换元后，积分限也要相应改变，不需要再换回原变量。

第一类换元法

设 $u = \varphi(x)$

$du = \varphi'(x)dx$

积分限：$x=a \Rightarrow u=\varphi(a)$
$x=b \Rightarrow u=\varphi(b)$

第二类换元法

设 $x = \psi(t)$

$dx = \psi'(t)dt$

积分限：$x=a \Rightarrow t=\alpha$
$x=b \Rightarrow t=\beta$

例1：第一类换元法

计算 $\int_0^1 x e^{x^2} dx$

解：
1 设 $u = x^2$，则 $du = 2x dx$，所以 $x dx = \frac{1}{2} du$
2 当 $x = 0$ 时，$u = 0$；当 $x = 1$ 时，$u = 1$
3 原积分变为：$\int_0^1 x e^{x^2} dx = \int_0^1 e^u \cdot \frac{1}{2} du = \frac{1}{2} \int_0^1 e^u du$
4 $= \frac{1}{2} [e^u]_0^1 = \frac{1}{2}(e^1 - e^0) = \frac{1}{2}(e - 1)$

例2：第二类换元法（三角替换）

计算 $\int_0^1 \sqrt{1-x^2} dx$

解：
1 设 $x = \sin t$，则 $dx = \cos t dt$，$\sqrt{1-x^2} = \sqrt{1-\sin^2 t} = |\cos t| = \cos t$（在积分区间内）
2 当 $x = 0$ 时，$t = 0$；当 $x = 1$ 时，$t = \frac{\pi}{2}$
3 原积分变为：$\int_0^1 \sqrt{1-x^2} dx = \int_0^{\pi/2} \cos t \cdot \cos t dt = \int_0^{\pi/2} \cos^2 t dt$
4 使用降幂公式：$\cos^2 t = \frac{1 + \cos 2t}{2}$
5 $= \int_0^{\pi/2} \frac{1 + \cos 2t}{2} dt = \frac{1}{2} \left[t + \frac{\sin 2t}{2}\right]_0^{\pi/2} = \frac{1}{2} \cdot \frac{\pi}{2} = \frac{\pi}{4}$

5.3.2 定积分的分部积分法

定积分的分部积分公式

设函数 $u(x)$ 和 $v(x)$ 在 $[a,b]$ 上连续可导，则：

$$\int_a^b u(x) v'(x) dx = [u(x)v(x)]_a^b - \int_a^b u'(x) v(x) dx$$

或者写成：

$$\int_a^b u dv = [uv]_a^b - \int_a^b v du$$

选择原则：通常选择 $u$ 使得 $u'$ 比 $u$ 简单，选择 $dv$ 使得容易积分得到 $v$。

分部积分法的选择顺序（ILATE法则）

选择 $u$ 的优先级顺序：

  1. Inverse functions（反函数）：$\arcsin x$，$\arctan x$，$\ln x$ 等
  2. Logarithmic functions（对数函数）：$\ln x$，$\log x$ 等
  3. Algebraic functions（代数函数）：$x^n$，$\sqrt{x}$ 等
  4. Trigonometric functions（三角函数）：$\sin x$，$\cos x$ 等
  5. Exponential functions（指数函数）：$e^x$，$a^x$ 等

例3：多项式与指数函数的乘积

计算 $\int_0^1 x e^x dx$

解：
1 设 $u = x$，$dv = e^x dx$，则 $du = dx$，$v = e^x$
2 应用分部积分公式： $$\int_0^1 x e^x dx = [x e^x]_0^1 - \int_0^1 e^x dx$$
3 $= [x e^x]_0^1 - [e^x]_0^1$
4 $= (1 \cdot e^1 - 0 \cdot e^0) - (e^1 - e^0) = e - (e - 1) = 1$

例4：多项式与三角函数的乘积

计算 $\int_0^{\pi} x \sin x dx$

解：
1 设 $u = x$，$dv = \sin x dx$，则 $du = dx$，$v = -\cos x$
2 应用分部积分公式： $$\int_0^{\pi} x \sin x dx = [x(-\cos x)]_0^{\pi} - \int_0^{\pi} (-\cos x) dx$$
3 $= [-x \cos x]_0^{\pi} + \int_0^{\pi} \cos x dx$
4 $= [-x \cos x]_0^{\pi} + [\sin x]_0^{\pi}$
5 $= (-\pi \cos \pi - 0 \cos 0) + (\sin \pi - \sin 0) = -\pi(-1) + 0 = \pi$

习题1 中等难度

计算定积分：$\int_0^{\pi/2} x \cos x dx$

解：

设 $u = x$，$dv = \cos x dx$，则 $du = dx$，$v = \sin x$

$$\int_0^{\pi/2} x \cos x dx = [x \sin x]_0^{\pi/2} - \int_0^{\pi/2} \sin x dx$$ $$= \left[\frac{\pi}{2} \sin \frac{\pi}{2} - 0 \sin 0\right] - [-\cos x]_0^{\pi/2}$$ $$= \frac{\pi}{2} \cdot 1 - \left(-\cos \frac{\pi}{2} + \cos 0\right) = \frac{\pi}{2} - (0 + 1) = \frac{\pi}{2} - 1$$

习题2 较难

计算定积分：$\int_1^e x^2 \ln x dx$

解：

设 $u = \ln x$，$dv = x^2 dx$，则 $du = \frac{1}{x} dx$，$v = \frac{x^3}{3}$

$$\int_1^e x^2 \ln x dx = \left[\frac{x^3}{3} \ln x\right]_1^e - \int_1^e \frac{x^3}{3} \cdot \frac{1}{x} dx$$ $$= \left[\frac{x^3}{3} \ln x\right]_1^e - \frac{1}{3} \int_1^e x^2 dx$$ $$= \left(\frac{e^3}{3} \ln e - \frac{1^3}{3} \ln 1\right) - \frac{1}{3} \left[\frac{x^3}{3}\right]_1^e$$ $$= \frac{e^3}{3} - 0 - \frac{1}{9}(e^3 - 1) = \frac{e^3}{3} - \frac{e^3 - 1}{9} = \frac{3e^3 - e^3 + 1}{9} = \frac{2e^3 + 1}{9}$$

习题3 很难

计算定积分：$\int_0^{\pi/2} e^x \sin x dx$

解：

这是一个需要两次分部积分的问题。

第一次分部积分：

设 $u = \sin x$，$dv = e^x dx$，则 $du = \cos x dx$，$v = e^x$

$$I = \int_0^{\pi/2} e^x \sin x dx = [e^x \sin x]_0^{\pi/2} - \int_0^{\pi/2} e^x \cos x dx$$ $$= e^{\pi/2} \sin \frac{\pi}{2} - e^0 \sin 0 - \int_0^{\pi/2} e^x \cos x dx = e^{\pi/2} - \int_0^{\pi/2} e^x \cos x dx$$

第二次分部积分：

对 $\int_0^{\pi/2} e^x \cos x dx$，设 $u = \cos x$，$dv = e^x dx$，则 $du = -\sin x dx$，$v = e^x$

$$\int_0^{\pi/2} e^x \cos x dx = [e^x \cos x]_0^{\pi/2} + \int_0^{\pi/2} e^x \sin x dx$$ $$= (e^{\pi/2} \cos \frac{\pi}{2} - e^0 \cos 0) + I = 0 - 1 + I = -1 + I$$

解方程：

$$I = e^{\pi/2} - (-1 + I) = e^{\pi/2} + 1 - I$$ $$2I = e^{\pi/2} + 1$$ $$I = \frac{e^{\pi/2} + 1}{2}$$
