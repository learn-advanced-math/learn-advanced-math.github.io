12.7.1 引例：矩形波，谐波分析

引例：矩形波的分解

考虑周期为 $2\pi$ 的矩形波函数：

$$f(x) = \begin{cases} 1, & 0 < x < \pi \\ -1, & -\pi < x < 0 \end{cases}$$

并且 $f(x+2\pi) = f(x)$（周期性）

这个看似简单的矩形波可以用无穷多个正弦函数的叠加来表示：

$$f(x) = \frac{4}{\pi}\left(\sin x + \frac{1}{3}\sin 3x + \frac{1}{5}\sin 5x + \cdots\right)$$
$$= \frac{4}{\pi}\sum_{n=0}^{\infty} \frac{\sin(2n+1)x}{2n+1}$$

谐波分析的意义

基波：$\sin x$ 是基本频率分量

谐波：$\sin 3x, \sin 5x, \ldots$ 是频率为基波整数倍的分量

物理意义：任何周期函数都可以分解为不同频率的简谐振动的叠加，这就是谐波分析的核心思想。

应用领域：

  • 信号处理：音频信号的频谱分析
  • 电子工程：电路中的周期信号分析
  • 图像处理：二维傅里叶变换
  • 量子力学：波函数的展开

例1：简单周期函数的谐波分析

分析函数 $f(x) = x$，$x \in (-\pi, \pi)$，并周期延拓的谐波组成。

解：

这是一个奇函数，所以其傅里叶级数只包含正弦项：

$$f(x) = \sum_{n=1}^{\infty} b_n \sin nx$$

计算系数：

$$b_n = \frac{2}{\pi} \int_0^{\pi} x \sin nx dx$$

使用分部积分：

$$b_n = \frac{2}{\pi} \left[-\frac{x \cos nx}{n}\Big|_0^{\pi} + \frac{1}{n}\int_0^{\pi} \cos nx dx\right]$$ $$= \frac{2}{\pi} \left[-\frac{\pi \cos n\pi}{n} + 0\right] = -\frac{2\cos n\pi}{n} = \frac{2(-1)^{n+1}}{n}$$

结果：

$$f(x) = 2\left(\sin x - \frac{\sin 2x}{2} + \frac{\sin 3x}{3} - \frac{\sin 4x}{4} + \cdots\right)$$

谐波成分：包含所有整数倍频率，振幅按 $\frac{1}{n}$ 衰减

12.7.2 三角级数，三角函数系的正交性

三角级数

形如下式的无穷级数称为三角级数：

$$\frac{a_0}{2} + \sum_{n=1}^{\infty} (a_n \cos nx + b_n \sin nx)$$

其中 $a_0, a_1, a_2, \ldots$ 和 $b_1, b_2, \ldots$ 是常数，称为傅里叶系数。

注意：$\frac{a_0}{2}$ 的写法是为了公式的统一性，这样 $a_0$ 就是平均值的两倍。

三角函数系

在区间 $[-\pi, \pi]$ 上的三角函数系：

$$1, \cos x, \sin x, \cos 2x, \sin 2x, \ldots, \cos nx, \sin nx, \ldots$$

更一般地，在区间 $[-l, l]$ 上：

$$1, \cos \frac{\pi x}{l}, \sin \frac{\pi x}{l}, \cos \frac{2\pi x}{l}, \sin \frac{2\pi x}{l}, \ldots$$

三角函数系的正交性

在区间 $[-\pi, \pi]$ 上，三角函数系具有以下正交性质：

积分类型 积分值 条件
$\int_{-\pi}^{\pi} \cos mx \cos nx dx$ $0$ $m \neq n$
$\int_{-\pi}^{\pi} \sin mx \sin nx dx$ $0$ $m \neq n$
$\int_{-\pi}^{\pi} \cos mx \sin nx dx$ $0$ 任意 $m, n$
$\int_{-\pi}^{\pi} \cos^2 nx dx$ $\pi$ $n \geq 1$
$\int_{-\pi}^{\pi} \sin^2 nx dx$ $\pi$ $n \geq 1$
$\int_{-\pi}^{\pi} 1^2 dx$ $2\pi$ /

正交性的意义：不同频率的三角函数在一个周期内积分为零，这使得我们能够独立地计算每个傅里叶系数。

例2：验证正交性质

验证 $\int_{-\pi}^{\pi} \cos 2x \sin 3x dx = 0$。

解：

使用积化和差公式：

$$\cos 2x \sin 3x = \frac{1}{2}[\sin(3x + 2x) - \sin(3x - 2x)]$$ $$= \frac{1}{2}[\sin 5x - \sin x]$$

因此：

$$\int_{-\pi}^{\pi} \cos 2x \sin 3x dx = \frac{1}{2}\int_{-\pi}^{\pi} (\sin 5x - \sin x) dx$$ $$= \frac{1}{2}\left[-\frac{\cos 5x}{5} + \cos x\right]_{-\pi}^{\pi}$$ $$= \frac{1}{2}\left[\left(-\frac{\cos 5\pi}{5} + \cos \pi\right) - \left(-\frac{\cos(-5\pi)}{5} + \cos(-\pi)\right)\right]$$ $$= \frac{1}{2}\left[\left(\frac{1}{5} - 1\right) - \left(\frac{1}{5} - 1\right)\right] = 0$$

结论：验证了正交性质成立。

例3：计算正交积分

计算 $\int_{-\pi}^{\pi} \cos^2 3x dx$。

解：

使用二倍角公式：$\cos^2 3x = \frac{1 + \cos 6x}{2}$

$$\int_{-\pi}^{\pi} \cos^2 3x dx = \int_{-\pi}^{\pi} \frac{1 + \cos 6x}{2} dx$$ $$= \frac{1}{2}\int_{-\pi}^{\pi} dx + \frac{1}{2}\int_{-\pi}^{\pi} \cos 6x dx$$ $$= \frac{1}{2} \cdot 2\pi + \frac{1}{2} \cdot 0 = \pi$$

结论：与正交性表格中的结果一致。

12.7.3 函数展开成傅里叶级数

傅里叶级数展开

设函数 $f(x)$ 在 $[-\pi, \pi]$ 上可积，则其傅里叶级数为：

$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty} (a_n \cos nx + b_n \sin nx)$$

其中傅里叶系数由以下公式计算：

$a_0$ 系数
$a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) dx$

（函数的平均值的两倍）

$a_n$ 系数
$a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \cos nx dx$

（余弦分量的系数）

$b_n$ 系数
$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \sin nx dx$

（正弦分量的系数）

Dirichlet条件
有限个间断点和极值点

（收敛条件）

傅里叶级数的收敛性

Dirichlet定理：如果函数 $f(x)$ 在 $[-\pi, \pi]$ 上满足：

  1. 连续或只有有限个第一类间断点
  2. 只有有限个极值点

则傅里叶级数在每点的收敛情况为：

  • 连续点：级数收敛到 $f(x)$
  • 间断点：级数收敛到 $\frac{f(x^-) + f(x^+)}{2}$
  • 端点：级数收敛到 $\frac{f(-\pi^+) + f(\pi^-)}{2}$

例4：展开矩形波函数

求函数 $f(x) = \begin{cases} 1, & 0 < x < \pi \\ -1, & -\pi < x < 0 \end{cases}$ 的傅里叶级数。

解：

步骤1：计算 $a_0$

$$a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) dx = \frac{1}{\pi} \left[\int_{-\pi}^{0} (-1) dx + \int_{0}^{\pi} 1 dx\right]$$ $$= \frac{1}{\pi}[-\pi + \pi] = 0$$

步骤2：计算 $a_n$

由于 $f(x)$ 是奇函数，$f(x)\cos nx$ 是奇函数，所以：

$$a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \cos nx dx = 0$$

步骤3：计算 $b_n$

由于 $f(x)$ 是奇函数，$f(x)\sin nx$ 是偶函数，所以：

$$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \sin nx dx = \frac{2}{\pi} \int_{0}^{\pi} \sin nx dx$$ $$= \frac{2}{\pi} \left[-\frac{\cos nx}{n}\right]_0^{\pi} = \frac{2}{\pi n}[1 - \cos n\pi]$$ $$= \frac{2}{\pi n}[1 - (-1)^n] = \begin{cases} \frac{4}{\pi n}, & n \text{为奇数} \\ 0, & n \text{为偶数} \end{cases}$$

结果：

$$f(x) = \frac{4}{\pi}\left(\sin x + \frac{\sin 3x}{3} + \frac{\sin 5x}{5} + \cdots\right)$$ $$= \frac{4}{\pi}\sum_{k=0}^{\infty} \frac{\sin(2k+1)x}{2k+1}$$

例5：展开锯齿波函数

求函数 $f(x) = x$，$x \in (-\pi, \pi)$ 的傅里叶级数。

解：

步骤1：分析函数性质

$f(x) = x$ 是奇函数，所以 $a_0 = a_n = 0$，只需计算 $b_n$。

步骤2：计算 $b_n$

$$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} x \sin nx dx = \frac{2}{\pi} \int_{0}^{\pi} x \sin nx dx$$

使用分部积分，设 $u = x$，$dv = \sin nx dx$：

$$b_n = \frac{2}{\pi} \left[-\frac{x \cos nx}{n}\Big|_0^{\pi} + \frac{1}{n}\int_0^{\pi} \cos nx dx\right]$$ $$= \frac{2}{\pi} \left[-\frac{\pi \cos n\pi}{n} + \frac{\sin n\pi}{n^2}\right]$$ $$= \frac{2}{\pi} \left[-\frac{\pi (-1)^n}{n} + 0\right] = \frac{2(-1)^{n+1}}{n}$$

结果：

$$f(x) = 2\left(\sin x - \frac{\sin 2x}{2} + \frac{\sin 3x}{3} - \frac{\sin 4x}{4} + \cdots\right)$$ $$= 2\sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n} \sin nx$$

例6：展开抛物线函数

求函数 $f(x) = x^2$，$x \in [-\pi, \pi]$ 的傅里叶级数。

解：

步骤1：分析函数性质

$f(x) = x^2$ 是偶函数，所以 $b_n = 0$，只需计算 $a_0$ 和 $a_n$。

步骤2：计算 $a_0$

$$a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} x^2 dx = \frac{2}{\pi} \int_{0}^{\pi} x^2 dx = \frac{2}{\pi} \cdot \frac{\pi^3}{3} = \frac{2\pi^2}{3}$$

步骤3：计算 $a_n$

$$a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} x^2 \cos nx dx = \frac{2}{\pi} \int_{0}^{\pi} x^2 \cos nx dx$$

使用两次分部积分：

$$a_n = \frac{2}{\pi} \left[\frac{x^2 \sin nx}{n}\Big|_0^{\pi} - \frac{2}{n}\int_0^{\pi} x \sin nx dx\right]$$ $$= \frac{2}{\pi} \left[0 - \frac{2}{n} \cdot \frac{\pi(-1)^{n+1}}{n}\right] = \frac{4(-1)^n}{n^2}$$

结果：

$$f(x) = \frac{\pi^2}{3} + 4\left(-\cos x + \frac{\cos 2x}{4} - \frac{\cos 3x}{9} + \cdots\right)$$ $$= \frac{\pi^2}{3} + 4\sum_{n=1}^{\infty} \frac{(-1)^n}{n^2} \cos nx$$

12.7.4 正弦级数和余弦级数；奇延拓与偶延拓

函数的延拓

对于定义在 $[0, \pi]$ 上的函数 $f(x)$，可以通过不同方式延拓到 $[-\pi, \pi]$：

奇延拓
$f_{odd}(-x) = -f_{odd}(x)$

延拓后的函数是奇函数

傅里叶级数只含正弦项

偶延拓
$f_{even}(-x) = f_{even}(x)$

延拓后的函数是偶函数

傅里叶级数只含余弦项

正弦级数（傅里叶正弦级数）

对函数 $f(x)$ 在 $[0, \pi]$ 上进行奇延拓，得到的傅里叶级数只含正弦项：

$$f(x) = \sum_{n=1}^{\infty} b_n \sin nx$$

其中系数为：

$$b_n = \frac{2}{\pi} \int_{0}^{\pi} f(x) \sin nx dx$$

特点：正弦级数在端点 $x = 0, \pi$ 处总是收敛到 $0$。

余弦级数（傅里叶余弦级数）

对函数 $f(x)$ 在 $[0, \pi]$ 上进行偶延拓，得到的傅里叶级数只含余弦项：

$$f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} a_n \cos nx$$

其中系数为：

$$a_0 = \frac{2}{\pi} \int_{0}^{\pi} f(x) dx$$
$$a_n = \frac{2}{\pi} \int_{0}^{\pi} f(x) \cos nx dx$$

特点：余弦级数在端点处收敛到函数值，适合处理边界条件为函数值的问题。

延拓方式的选择原则

奇延拓适用于：

  • 边界条件：$f(0) = f(\pi) = 0$
  • 物理问题：弦的振动（两端固定）
  • 热传导：两端温度为零

偶延拓适用于：

  • 边界条件：导数 $f'(0) = f'(\pi) = 0$
  • 物理问题：杆的振动（两端自由）
  • 热传导：两端绝热

例7：正弦级数展开

将函数 $f(x) = \pi - x$，$x \in [0, \pi]$ 展开为正弦级数。

解：

步骤1：进行奇延拓

将 $f(x) = \pi - x$ 奇延拓到 $[-\pi, \pi]$：

$$f_{odd}(x) = \begin{cases} \pi - x, & 0 \leq x \leq \pi \\ -(\pi + x), & -\pi \leq x < 0 \end{cases}$$

步骤2：计算正弦系数

$$b_n = \frac{2}{\pi} \int_{0}^{\pi} (\pi - x) \sin nx dx$$ $$= \frac{2}{\pi} \left[\pi \int_{0}^{\pi} \sin nx dx - \int_{0}^{\pi} x \sin nx dx\right]$$

计算第一个积分：

$$\int_{0}^{\pi} \sin nx dx = \left[-\frac{\cos nx}{n}\right]_0^{\pi} = \frac{1 - \cos n\pi}{n} = \frac{1 - (-1)^n}{n}$$

计算第二个积分（分部积分）：

$$\int_{0}^{\pi} x \sin nx dx = \frac{\pi (-1)^{n+1}}{n}$$

因此：

$$b_n = \frac{2}{\pi} \left[\pi \cdot \frac{1 - (-1)^n}{n} - \frac{\pi (-1)^{n+1}}{n}\right]$$ $$= \frac{2[1 - (-1)^n] + 2(-1)^{n+1}}{n} = \frac{2}{n}$$

结果：

$$f(x) = 2\left(\sin x + \frac{\sin 2x}{2} + \frac{\sin 3x}{3} + \cdots\right) = 2\sum_{n=1}^{\infty} \frac{\sin nx}{n}$$

例8：余弦级数展开

将函数 $f(x) = x$，$x \in [0, \pi]$ 展开为余弦级数。

解：

步骤1：进行偶延拓

将 $f(x) = x$ 偶延拓到 $[-\pi, \pi]$：

$$f_{even}(x) = |x|, \quad x \in [-\pi, \pi]$$

步骤2：计算余弦系数

$$a_0 = \frac{2}{\pi} \int_{0}^{\pi} x dx = \frac{2}{\pi} \cdot \frac{\pi^2}{2} = \pi$$ $$a_n = \frac{2}{\pi} \int_{0}^{\pi} x \cos nx dx$$

使用分部积分：

$$a_n = \frac{2}{\pi} \left[\frac{x \sin nx}{n}\Big|_0^{\pi} - \frac{1}{n}\int_0^{\pi} \sin nx dx\right]$$ $$= \frac{2}{\pi} \left[0 - \frac{1}{n} \cdot \frac{1 - \cos n\pi}{n}\right]$$ $$= -\frac{2[1 - (-1)^n]}{\pi n^2} = \begin{cases} -\frac{4}{\pi n^2}, & n \text{为奇数} \\ 0, & n \text{为偶数} \end{cases}$$

结果：

$$f(x) = \frac{\pi}{2} - \frac{4}{\pi}\left(\cos x + \frac{\cos 3x}{9} + \frac{\cos 5x}{25} + \cdots\right)$$ $$= \frac{\pi}{2} - \frac{4}{\pi}\sum_{k=0}^{\infty} \frac{\cos(2k+1)x}{(2k+1)^2}$$

例9：比较不同延拓方式

对函数 $f(x) = 1$，$x \in [0, \pi]$，分别求其正弦级数和余弦级数展开。

解：

正弦级数展开（奇延拓）：

$$b_n = \frac{2}{\pi} \int_{0}^{\pi} 1 \cdot \sin nx dx = \frac{2}{\pi} \cdot \frac{1 - \cos n\pi}{n}$$ $$= \frac{2[1 - (-1)^n]}{\pi n} = \begin{cases} \frac{4}{\pi n}, & n \text{为奇数} \\ 0, & n \text{为偶数} \end{cases}$$

正弦级数：$f(x) = \frac{4}{\pi}\left(\sin x + \frac{\sin 3x}{3} + \frac{\sin 5x}{5} + \cdots\right)$

余弦级数展开（偶延拓）：

$$a_0 = \frac{2}{\pi} \int_{0}^{\pi} 1 dx = 2$$ $$a_n = \frac{2}{\pi} \int_{0}^{\pi} 1 \cdot \cos nx dx = \frac{2}{\pi} \cdot \frac{\sin n\pi}{n} = 0$$

余弦级数：$f(x) = 1$（精确表示！）

对比分析：

  • 正弦级数在端点收敛到 $0$，需要无穷项求和
  • 余弦级数直接等于常函数，只需一项
  • 选择合适的延拓方式可以大大简化表达式
