3.4 函数的单调性与曲线的凹凸性

Monotonicity and Concavity of Functions

3.4.1 函数单调性的判定法

函数单调性的定义

设函数 $f(x)$ 在区间 $I$ 上有定义，如果对于区间 $I$ 上任意两点 $x_1, x_2$，当 $x_1 < x_2$ 时，恒有：

  • 单调递增：$f(x_1) \leq f(x_2)$（若 $f(x_1) < f(x_2)$ 则称严格单调递增）
  • 单调递减：$f(x_1) \geq f(x_2)$（若 $f(x_1) > f(x_2)$ 则称严格单调递减）

函数单调性的判定定理

设函数 $f(x)$ 在 $[a,b]$ 上连续，在 $(a,b)$ 内可导，则：

  1. 如果在 $(a,b)$ 内 $f'(x) > 0$，则 $f(x)$ 在 $[a,b]$ 上单调递增
  2. 如果在 $(a,b)$ 内 $f'(x) < 0$，则 $f(x)$ 在 $[a,b]$ 上单调递减
  3. 如果在 $(a,b)$ 内 $f'(x) = 0$，则 $f(x)$ 在 $[a,b]$ 上为常函数

判定步骤：

  1. 求出函数的定义域
  2. 求出函数的一阶导数 $f'(x)$
  3. 求出 $f'(x) = 0$ 的根和 $f'(x)$ 不存在的点
  4. 用这些点将定义域分成若干个区间
  5. 判断 $f'(x)$ 在各区间内的符号
  6. 根据导数符号确定函数的单调性

例1：判断函数的单调性

判断函数 $f(x) = x^3 - 3x^2 - 9x + 2$ 的单调性。

解：

步骤1：求导数

$f'(x) = 3x^2 - 6x - 9 = 3(x^2 - 2x - 3) = 3(x-3)(x+1)$

步骤2：求 $f'(x) = 0$ 的根

令 $f'(x) = 0$，得 $x = -1$ 或 $x = 3$

步骤3：判断各区间内导数的符号

  • 当 $x \in (-\infty, -1)$ 时，$f'(x) = 3(-)(-) = 3(+) > 0$，函数单调递增
  • 当 $x \in (-1, 3)$ 时，$f'(x) = 3(+)(-) = 3(-) < 0$，函数单调递减
  • 当 $x \in (3, +\infty)$ 时，$f'(x) = 3(+)(+) = 3(+) > 0$，函数单调递增

结论：函数在 $(-\infty, -1]$ 和 $[3, +\infty)$ 上单调递增，在 $[-1, 3]$ 上单调递减。

例2：证明不等式

证明：当 $x > 0$ 时，$\ln(1+x) < x$。

证明：

设 $f(x) = x - \ln(1+x)$，$x > 0$

$f'(x) = 1 - \frac{1}{1+x} = \frac{1+x-1}{1+x} = \frac{x}{1+x}$

当 $x > 0$ 时，$f'(x) = \frac{x}{1+x} > 0$，所以 $f(x)$ 在 $(0, +\infty)$ 上单调递增。

又 $f(0) = 0 - \ln(1) = 0$

因此当 $x > 0$ 时，$f(x) > f(0) = 0$，即 $x - \ln(1+x) > 0$

所以 $\ln(1+x) < x$。

3.4.2 曲线的凹凸性与拐点

曲线凹凸性的定义

设函数 $f(x)$ 在区间 $I$ 上连续，如果对于 $I$ 上任意两点 $x_1, x_2$ 和任意 $\lambda \in (0,1)$，恒有：

$$f[\lambda x_1 + (1-\lambda)x_2] < \lambda f(x_1) + (1-\lambda)f(x_2)$$

则称 $f(x)$ 在 $I$ 上的图形是凹的（向下凹，凹函数）。

$$f[\lambda x_1 + (1-\lambda)x_2] > \lambda f(x_1) + (1-\lambda)f(x_2)$$

则称 $f(x)$ 在 $I$ 上的图形是凸的（向上凸，凸函数）。

几何意义：凹函数的图像位于任意两点连线的下方；凸函数的图像位于任意两点连线的上方。

曲线凹凸性的判定定理

设函数 $f(x)$ 在 $[a,b]$ 上连续，在 $(a,b)$ 内具有一阶和二阶导数，则：

  1. 若在 $(a,b)$ 内 $f''(x) > 0$，则 $f(x)$ 在 $[a,b]$ 上的图形是凹的
  2. 若在 $(a,b)$ 内 $f''(x) < 0$，则 $f(x)$ 在 $[a,b]$ 上的图形是凸的

记忆方法："二阶导数正，图像向下凹；二阶导数负，图像向上凸"

拐点的定义

连续曲线上凹凸性改变的点称为曲线的拐点。

拐点的必要条件：若 $(x_0, f(x_0))$ 是曲线 $y = f(x)$ 的拐点，则 $f''(x_0) = 0$ 或 $f''(x_0)$ 不存在。

拐点的充分条件：若 $f''(x_0) = 0$（或不存在），且 $f''(x)$ 在 $x_0$ 的左右两侧变号，则 $(x_0, f(x_0))$ 是拐点。

凹函数特征

• 二阶导数 $f''(x) > 0$

• 图像向下弯曲

• 切线在曲线下方

• 如碗状开口向上

凸函数特征

• 二阶导数 $f''(x) < 0$

• 图像向上弯曲

• 切线在曲线上方

• 如山峰状开口向下

拐点特征

• 二阶导数为零或不存在

• 凹凸性发生改变

• 曲率变号的点

• 切线穿过曲线

例3：求曲线的凹凸区间和拐点

求曲线 $y = x^4 - 4x^3 + 6x^2$ 的凹凸区间和拐点。

解：

步骤1：求一阶和二阶导数

$y' = 4x^3 - 12x^2 + 12x$

$y'' = 12x^2 - 24x + 12 = 12(x^2 - 2x + 1) = 12(x-1)^2$

步骤2：求 $y'' = 0$ 的根

令 $y'' = 0$，得 $12(x-1)^2 = 0$，解得 $x = 1$（二重根）

步骤3：判断凹凸性

由于 $y'' = 12(x-1)^2 \geq 0$，且仅当 $x = 1$ 时 $y'' = 0$

所以曲线在 $(-\infty, +\infty)$ 上都是凹的。

步骤4：判断拐点

虽然 $x = 1$ 处 $y'' = 0$，但 $y''$ 在 $x = 1$ 两侧不变号（都为正），

所以 $x = 1$ 不是拐点。

结论：曲线在整个定义域上都是凹的，没有拐点。

例4：拐点的实际应用

某产品的边际成本函数为 $MC(x) = 3x^2 - 20x + 50$（元/件），求平均成本最小时的产量。

解：

步骤1：求成本函数

由 $MC(x) = C'(x) = 3x^2 - 20x + 50$，积分得：

$C(x) = x^3 - 10x^2 + 50x + C_0$

步骤2：求平均成本函数

设 $C_0 = 0$，则平均成本函数为：

$AC(x) = \frac{C(x)}{x} = x^2 - 10x + 50$

步骤3：求平均成本的最小值

$AC'(x) = 2x - 10$

令 $AC'(x) = 0$，得 $x = 5$

$AC''(x) = 2 > 0$，所以 $x = 5$ 是极小值点。

结论：当产量为 $5$ 件时，平均成本最小。

练习题

习题1 中等难度

求函数 $f(x) = \frac{x^2}{x^2 + 1}$ 的单调区间，并证明该函数在 $(-\infty, +\infty)$ 上有界。

解：

第一部分：求单调区间

$f'(x) = \frac{2x(x^2+1) - x^2 \cdot 2x}{(x^2+1)^2} = \frac{2x(x^2+1-x^2)}{(x^2+1)^2} = \frac{2x}{(x^2+1)^2}$

令 $f'(x) = 0$，得 $x = 0$

当 $x < 0$ 时，$f'(x) < 0$，函数单调递减

当 $x > 0$ 时，$f'(x) > 0$，函数单调递增

所以函数在 $(-\infty, 0]$ 上单调递减，在 $[0, +\infty)$ 上单调递增。

第二部分：证明有界性

$f(x) = \frac{x^2}{x^2 + 1} = \frac{x^2 + 1 - 1}{x^2 + 1} = 1 - \frac{1}{x^2 + 1}$

由于 $x^2 + 1 \geq 1$，所以 $0 < \frac{1}{x^2 + 1} \leq 1$

因此 $0 \leq f(x) < 1$，函数有界。

习题2 较难

已知函数 $f(x) = ax^3 + bx^2 + cx + d$ 在 $x = 1$ 处有极值 $2$，在 $x = -1$ 处有拐点，且拐点的纵坐标为 $\frac{2}{3}$。求函数 $f(x)$ 的表达式。

解：

根据题意，我们有四个条件：

  1. $f(1) = 2$（在 $x = 1$ 处的函数值）
  2. $f'(1) = 0$（在 $x = 1$ 处有极值）
  3. $f(-1) = \frac{2}{3}$（拐点的纵坐标）
  4. $f''(-1) = 0$（在 $x = -1$ 处有拐点）

计算导数：

$f'(x) = 3ax^2 + 2bx + c$

$f''(x) = 6ax + 2b$

根据条件建立方程组：

$f(1) = a + b + c + d = 2$ ——— (1)

$f'(1) = 3a + 2b + c = 0$ ——— (2)

$f(-1) = -a + b - c + d = \frac{2}{3}$ ——— (3)

$f''(-1) = -6a + 2b = 0$ ——— (4)

从方程(4)得：$b = 3a$

代入方程(2)：$3a + 6a + c = 0$，得 $c = -9a$

代入方程(1)和(3)：

$a + 3a - 9a + d = 2$，即 $d = 2 + 5a$ ——— (5)

$-a + 3a + 9a + d = \frac{2}{3}$，即 $11a + d = \frac{2}{3}$ ——— (6)

将(5)代入(6)：$11a + 2 + 5a = \frac{2}{3}$

$16a = \frac{2}{3} - 2 = -\frac{4}{3}$

所以 $a = -\frac{1}{12}$

因此：$b = 3 \times (-\frac{1}{12}) = -\frac{1}{4}$

$c = -9 \times (-\frac{1}{12}) = \frac{3}{4}$

$d = 2 + 5 \times (-\frac{1}{12}) = 2 - \frac{5}{12} = \frac{19}{12}$

答案：$f(x) = -\frac{1}{12}x^3 - \frac{1}{4}x^2 + \frac{3}{4}x + \frac{19}{12}$

习题3 很难

设函数 $f(x)$ 在 $[0,1]$ 上连续，在 $(0,1)$ 内二阶可导，且 $f(0) = f(1) = 0$，$f(x) > 0$ 对 $x \in (0,1)$ 成立。证明：在 $(0,1)$ 内至少存在一点 $\xi$ 使得 $f''(\xi) < 0$。

证明（反证法）：

假设在 $(0,1)$ 内对所有 $x$ 都有 $f''(x) \geq 0$，即函数图像在 $(0,1)$ 内是凹的。

步骤1：利用凹函数性质

由于 $f(x)$ 在 $(0,1)$ 内是凹函数，根据凹函数的性质，对于任意 $x \in (0,1)$，有：

$$f(x) \leq \frac{1-x}{1-0}f(0) + \frac{x-0}{1-0}f(1) = (1-x) \cdot 0 + x \cdot 0 = 0$$

步骤2：得出矛盾

这与题设条件 $f(x) > 0$ 对 $x \in (0,1)$ 成立相矛盾。

步骤3：结论

因此，原假设不成立，必定在 $(0,1)$ 内至少存在一点 $\xi$ 使得 $f''(\xi) < 0$。

几何解释：由于 $f(0) = f(1) = 0$ 且 $f(x) > 0$ 在 $(0,1)$ 内恒成立，函数图像必须"凸起"于 $x$ 轴之上。如果整个区间内都是凹的，那么连接端点的直线（即 $x$ 轴）应该在函数图像下方，这与 $f(x) > 0$ 矛盾。因此必须存在凸的部分。
