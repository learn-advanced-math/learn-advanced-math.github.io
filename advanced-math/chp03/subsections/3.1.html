3.1 微分中值定理

Mean Value Theorems in Differential Calculus

3.1.1 罗尔定理

罗尔定理（Rolle's Theorem）

如果函数 $f(x)$ 满足：

  1. 在闭区间 $[a,b]$ 上连续
  2. 在开区间 $(a,b)$ 内可导
  3. $f(a) = f(b)$

那么在 $(a,b)$ 内至少存在一点 $\xi$，使得 $f'(\xi) = 0$。

几何意义：在满足条件的曲线上，至少存在一点处的切线平行于 $x$ 轴。

物理意义：如果一个质点在两个时刻的位置相同，那么在这个时间段内至少存在一个瞬间，质点的瞬时速度为零。

例1：验证罗尔定理的条件

验证函数 $f(x) = x^2 - 4x + 3$ 在区间 $[1,3]$ 上满足罗尔定理的条件，并求出满足条件的 $\xi$。

解：

步骤1：验证连续性

$f(x) = x^2 - 4x + 3$ 是多项式函数，在 $[1,3]$ 上连续。

步骤2：验证可导性

$f'(x) = 2x - 4$ 在 $(1,3)$ 内处处存在，故函数在 $(1,3)$ 内可导。

步骤3：验证端点函数值相等

$f(1) = 1 - 4 + 3 = 0$

$f(3) = 9 - 12 + 3 = 0$

因此 $f(1) = f(3)$

步骤4：求 $\xi$

由 $f'(\xi) = 0$，即 $2\xi - 4 = 0$，解得 $\xi = 2 \in (1,3)$

例2：罗尔定理的应用

证明：方程 $x^3 - 3x + 1 = 0$ 在区间 $(0,1)$ 内不能有两个根。

证明（反证法）：

设方程在 $(0,1)$ 内有两个不同的根 $x_1$ 和 $x_2$，不妨设 $0 < x_1 < x_2 < 1$。

则有 $f(x_1) = f(x_2) = 0$，其中 $f(x) = x^3 - 3x + 1$。

由于 $f(x)$ 在 $[x_1, x_2]$ 上连续，在 $(x_1, x_2)$ 内可导，且 $f(x_1) = f(x_2) = 0$，

根据罗尔定理，存在 $\xi \in (x_1, x_2) \subset (0,1)$，使得 $f'(\xi) = 0$。

但 $f'(x) = 3x^2 - 3 = 3(x^2 - 1) < 0$（当 $x \in (0,1)$ 时）

这与 $f'(\xi) = 0$ 矛盾。因此原假设不成立，方程在 $(0,1)$ 内不能有两个根。

3.1.2 拉格朗日中值定理

拉格朗日中值定理（Lagrange Mean Value Theorem）

如果函数 $f(x)$ 满足：

  1. 在闭区间 $[a,b]$ 上连续
  2. 在开区间 $(a,b)$ 内可导

那么在 $(a,b)$ 内至少存在一点 $\xi$，使得：

$$f'(\xi) = \frac{f(b) - f(a)}{b - a}$$

几何意义：在满足条件的曲线上，至少存在一点处的切线平行于连接两端点的弦。

拉格朗日公式：$f(b) - f(a) = f'(\xi)(b - a)$，其中 $\xi \in (a,b)$

例3：拉格朗日中值定理的应用

证明：当 $x > 0$ 时，$\ln(1+x) < x$

证明：

设 $f(t) = \ln t$，在区间 $[1, 1+x]$（其中 $x > 0$）上应用拉格朗日中值定理：

$f(t)$ 在 $[1, 1+x]$ 上连续，在 $(1, 1+x)$ 内可导，所以存在 $\xi \in (1, 1+x)$，使得：

$$f'(\xi) = \frac{f(1+x) - f(1)}{(1+x) - 1}$$

即：

$$\frac{1}{\xi} = \frac{\ln(1+x) - \ln 1}{x} = \frac{\ln(1+x)}{x}$$

因此：$\ln(1+x) = \frac{x}{\xi}$

由于 $\xi > 1$，所以 $\frac{1}{\xi} < 1$，因此：

$$\ln(1+x) = \frac{x}{\xi} < x$$

例4：用中值定理证明不等式

证明：当 $0 < x < \frac{\pi}{2}$ 时，$\frac{\sin x}{x} > \cos x$

证明：

设 $f(t) = \sin t$，在区间 $[0, x]$（其中 $0 < x < \frac{\pi}{2}$）上应用拉格朗日中值定理：

存在 $\xi \in (0, x)$，使得：

$$f'(\xi) = \frac{f(x) - f(0)}{x - 0}$$

即：$\cos \xi = \frac{\sin x}{x}$

由于函数 $y = \cos t$ 在 $(0, \frac{\pi}{2})$ 内单调递减，且 $0 < \xi < x < \frac{\pi}{2}$，

所以 $\cos \xi > \cos x$，因此：

$$\frac{\sin x}{x} = \cos \xi > \cos x$$

3.1.3 柯西中值定理

柯西中值定理（Cauchy Mean Value Theorem）

如果函数 $f(x)$ 和 $g(x)$ 满足：

  1. 在闭区间 $[a,b]$ 上连续
  2. 在开区间 $(a,b)$ 内可导
  3. $g'(x) \neq 0$ 在 $(a,b)$ 内

那么在 $(a,b)$ 内至少存在一点 $\xi$，使得：

$$\frac{f'(\xi)}{g'(\xi)} = \frac{f(b) - f(a)}{g(b) - g(a)}$$

注：当 $g(x) = x$ 时，柯西中值定理退化为拉格朗日中值定理。

几何意义：参数方程 $x = g(t), y = f(t)$ 表示的曲线上，存在一点处的切线斜率等于连接两端点直线的斜率。

罗尔定理
条件：$f(a) = f(b)$
结论：$f'(\xi) = 0$
拉格朗日中值定理
条件：一般情况
结论：$f'(\xi) = \frac{f(b) - f(a)}{b - a}$
柯西中值定理
条件：两个函数
结论：$\frac{f'(\xi)}{g'(\xi)} = \frac{f(b) - f(a)}{g(b) - g(a)}$

例5：柯西中值定理的应用

对函数 $f(x) = x^2$ 和 $g(x) = x^3$ 在区间 $[1,2]$ 上验证柯西中值定理。

解：

步骤1：验证条件

$f(x) = x^2$ 和 $g(x) = x^3$ 在 $[1,2]$ 上连续，在 $(1,2)$ 内可导。

$g'(x) = 3x^2 \neq 0$ 在 $(1,2)$ 内成立。

步骤2：计算端点值

$f(1) = 1, f(2) = 4, g(1) = 1, g(2) = 8$

步骤3：应用柯西中值定理

存在 $\xi \in (1,2)$，使得：

$$\frac{f'(\xi)}{g'(\xi)} = \frac{f(2) - f(1)}{g(2) - g(1)}$$

即：

$$\frac{2\xi}{3\xi^2} = \frac{4-1}{8-1} = \frac{3}{7}$$

化简得：$\frac{2}{3\xi} = \frac{3}{7}$，解得 $\xi = \frac{14}{9} \approx 1.56 \in (1,2)$

练习题

习题1 中等难度

证明：方程 $x^5 + x - 1 = 0$ 在区间 $(0,1)$ 内有且仅有一个实根。

解：

存在性证明：

设 $f(x) = x^5 + x - 1$，则 $f(x)$ 在 $[0,1]$ 上连续。

$f(0) = -1 < 0$，$f(1) = 1 > 0$

由零点定理，存在 $c \in (0,1)$ 使得 $f(c) = 0$。

唯一性证明：

$f'(x) = 5x^4 + 1 > 0$ 在 $(0,1)$ 内恒成立

所以 $f(x)$ 在 $(0,1)$ 内严格单调递增，因此至多有一个零点。

结合存在性，方程在 $(0,1)$ 内有且仅有一个实根。

习题2 较难

设 $f(x)$ 在 $[a,b]$ 上连续，在 $(a,b)$ 内二阶可导，且 $f(a) = f(b) = 0$，$f(c) > 0$（其中 $c \in (a,b)$）。证明：在 $(a,b)$ 内至少存在一点 $\xi$ 使得 $f''(\xi) < 0$。

证明：

由于 $f(a) = f(b) = 0$，$f(c) > 0$，可知 $f(x)$ 在 $[a,b]$ 上的最大值在内部某点达到。

设 $f(x)$ 在 $x = x_0 \in (a,b)$ 处取得最大值 $M > 0$。

由于 $x_0$ 是内部极大值点，有 $f'(x_0) = 0$。

由极值的二阶判别法，$f''(x_0) \leq 0$。

下面用反证法证明 $f''(x_0) < 0$：

假设 $f''(x_0) = 0$，由于 $f'(x_0) = 0$，在 $x_0$ 的某个邻域内，利用泰勒展开：

$$f(x) = f(x_0) + f'(x_0)(x-x_0) + \frac{f''(x_0)}{2!}(x-x_0)^2 + o((x-x_0)^2)$$ $$= M + o((x-x_0)^2)$$

这意味着在 $x_0$ 附近，$f(x) \approx M$，与 $f(x)$ 在 $x_0$ 处取得严格最大值矛盾。

因此 $f''(x_0) < 0$，取 $\xi = x_0$ 即可。

习题3 很难

设函数 $f(x) = \ln x$ 和 $g(x) = x^2$ 在区间 $[1,e]$ 上满足柯西中值定理的条件。求满足定理结论的 $\xi$ 值，并验证柯西中值定理在此处如何退化为拉格朗日中值定理。

解：

这是一道综合考查柯西中值定理应用和理论理解的题目。

步骤1：验证柯西中值定理的条件

对于 $f(x) = \ln x$ 和 $g(x) = x^2$ 在区间 $[1,e]$ 上：

  • $f(x) = \ln x$ 在 $[1,e]$ 上连续，在 $(1,e)$ 内可导
  • $g(x) = x^2$ 在 $[1,e]$ 上连续，在 $(1,e)$ 内可导
  • $g'(x) = 2x \neq 0$ 在 $(1,e)$ 内恒成立

步骤2：计算端点函数值

$f(1) = \ln 1 = 0$，$f(e) = \ln e = 1$

$g(1) = 1^2 = 1$，$g(e) = e^2$

步骤3：应用柯西中值定理

根据柯西中值定理，存在 $\xi \in (1,e)$ 使得：

$$\frac{f'(\xi)}{g'(\xi)} = \frac{f(e) - f(1)}{g(e) - g(1)}$$

即：

$$\frac{\frac{1}{\xi}}{2\xi} = \frac{1 - 0}{e^2 - 1}$$

化简左边：$\frac{\frac{1}{\xi}}{2\xi} = \frac{1}{2\xi^2}$

所以方程变为：

$$\frac{1}{2\xi^2} = \frac{1}{e^2 - 1}$$

步骤4：求解 $\xi$

从 $\frac{1}{2\xi^2} = \frac{1}{e^2 - 1}$ 得到：

$$2\xi^2 = e^2 - 1$$ $$\xi^2 = \frac{e^2 - 1}{2}$$ $$\xi = \sqrt{\frac{e^2 - 1}{2}}$$

由于 $e \approx 2.718$，所以：

$$\xi = \sqrt{\frac{(2.718)^2 - 1}{2}} = \sqrt{\frac{7.387 - 1}{2}} = \sqrt{3.194} \approx 1.787$$

验证：$1 < 1.787 < 2.718$，确实 $\xi \in (1,e)$

步骤5：理论分析——柯西中值定理的退化

考虑当 $g(x) = x$ 时的情况：

如果我们取 $g(x) = x$，那么 $g'(x) = 1$，柯西中值定理变为：

$$\frac{f'(\xi)}{1} = \frac{f(e) - f(1)}{e - 1}$$

这正是拉格朗日中值定理的形式！

步骤6：几何意义解释

柯西中值定理的几何意义是：参数方程

$$\begin{cases} x = g(t) = t^2 \\ y = f(t) = \ln t \end{cases}$$

表示的曲线上存在一点 $(\xi^2, \ln \xi)$，其切线斜率等于连接端点的弦的斜率。

切线斜率：$\frac{dy/dt}{dx/dt} = \frac{f'(\xi)}{g'(\xi)} = \frac{1/\xi}{2\xi} = \frac{1}{2\xi^2}$

弦的斜率：$\frac{\ln e - \ln 1}{e^2 - 1^2} = \frac{1}{e^2 - 1}$

结论：

满足柯西中值定理的点为 $\xi = \sqrt{\frac{e^2-1}{2}} \approx 1.787$

当辅助函数 $g(x) = x$ 时，柯西中值定理就退化为拉格朗日中值定理，这体现了数学理论的统一性和一般性。
