Element Method in Definite Integration

6.1.1 定积分的元素法相关的概念

微元法（元素法）的基本思想

微元法是解决定积分应用问题的基本方法，其核心思想是：

  1. 分割：将所求量的总体分割成许多微小部分
  2. 近似：每个微小部分近似表示为一个微元
  3. 求和：对所有微元求和
  4. 取极限：令分割无限细化，求和转化为定积分
若所求量 $U$ 对应于区间 $[a,b]$，且在 $[x, x+dx]$ 上的微元为 $dU = f(x)dx$，
则总量为：$$U = \int_a^b f(x)dx$$

2
设定微元
取微小区间，写出微元表达式 $dU = f(x)dx$
3
建立积分
确定积分上下限，建立定积分表达式
4
计算积分
运用积分方法计算定积分，得到最终结果

微元法的数学基础

微元法基于定积分的定义和微分的概念：

  • 可加性：所求量具有可加性，即总量等于各部分量的和
  • 连续性：被积函数在积分区间上连续
  • 微分近似：在无穷小区间内，曲线可用直线近似

关键步骤：如何正确设定微元 $dU = f(x)dx$ 是应用微元法的核心。

例1：用微元法求曲边梯形面积

求由曲线 $y = x^2$、直线 $x = 0$、$x = 2$ 和 $x$ 轴所围成的图形面积。

解：

第一步：建立坐标系

以给定的 $x$ 轴为横轴，$y$ 轴为纵轴，积分区间为 $[0, 2]$。

第二步：设定微元

在 $[x, x+dx]$ 上，面积微元为：

$$dS = y \cdot dx = x^2 dx$$

第三步：建立积分

总面积为：

$$S = \int_0^2 x^2 dx$$

第四步：计算积分

$$S = \int_0^2 x^2 dx = \left[\frac{x^3}{3}\right]_0^2 = \frac{8}{3} - 0 = \frac{8}{3}$$

例2：用微元法求变力做功

一个弹簧的自然长度为 $10$ cm，要把它拉长到 $15$ cm，需要力 $20$ N。求把弹簧从自然长度拉长到 $12$ cm 所做的功。

解：

第一步：建立坐标系和力的关系

设弹簧的自然长度为原点，拉长 $x$ cm 时的力为 $F(x)$。

由胡克定律：$F(x) = kx$，其中 $k$ 为弹性系数。

由条件：当 $x = 5$ cm 时，$F = 20$ N，所以 $k = 4$ N/cm。

因此：$F(x) = 4x$ (N)，$x$ 的单位为 cm。

第二步：设定微元

在 $[x, x+dx]$ 上，功的微元为：

$$dW = F(x) \cdot dx = 4x \, dx \text{ (N·cm)}$$

第三步：建立积分

从 $x = 0$ 到 $x = 2$ cm 的总功为：

$$W = \int_0^2 4x \, dx$$

第四步：计算积分

$$W = \int_0^2 4x \, dx = 4 \cdot \frac{x^2}{2}\Big|_0^2 = 2x^2\Big|_0^2 = 8 \text{ N·cm} = 0.08 \text{ J}$$

习题1：平面图形面积 中等难度

用微元法求由抛物线 $y^2 = 4x$ 和直线 $x = 4$ 所围成的图形面积。

解题思路：
  1. 选择适当的积分变量（$x$ 或 $y$）
  2. 确定积分区间
  3. 建立面积微元表达式
  4. 建立并计算定积分
详细解答：

选择 $y$ 为积分变量，因为对于给定的 $y$，$x$ 的范围容易确定。

抛物线 $y^2 = 4x$ 与直线 $x = 4$ 的交点：$y^2 = 16$，即 $y = \pm 4$

面积微元：$dS = (4 - \frac{y^2}{4}) dy$

总面积：$S = \int_{-4}^4 (4 - \frac{y^2}{4}) dy = 2\int_0^4 (4 - \frac{y^2}{4}) dy = 2[4y - \frac{y^3}{12}]_0^4 = \frac{64}{3}$

习题2：旋转体体积 较难

由曲线 $y = \sin x$，$x \in [0, \pi]$ 与 $x$ 轴围成的图形绕 $y$ 轴旋转所得旋转体的体积。

解题思路：
  1. 这是绕 $y$ 轴旋转，考虑用柱壳法
  2. 设定体积微元为 $dV = 2\pi x \cdot y \, dx$
  3. 建立积分表达式
详细解答：

使用柱壳法，体积微元为：$dV = 2\pi x \sin x \, dx$

总体积：$V = \int_0^\pi 2\pi x \sin x \, dx = 2\pi \int_0^\pi x \sin x \, dx$

使用分部积分：$\int x \sin x \, dx = -x \cos x + \sin x + C$

$V = 2\pi[-x \cos x + \sin x]_0^\pi = 2\pi[(\pi + 0) - (0 + 0)] = 2\pi^2$

习题3：物理应用 很难

半径为 $R$ 的半圆形薄片，密度为 $\rho(x) = k(R - x)$（$k$ 为常数，$x$ 为到直径的距离），求该薄片的质量。

解题思路：
  1. 建立坐标系，以直径为 $x$ 轴
  2. 考虑密度随位置变化，需要用微元法
  3. 设定质量微元
详细解答：

建立坐标系：以半圆的直径为 $x$ 轴，圆心为原点

半圆方程：$y = \sqrt{R^2 - x^2}$，$x \in [-R, R]$

在 $[x, x+dx]$ 处，微元的面积：$dS = y \, dx = \sqrt{R^2 - x^2} \, dx$

微元的质量：$dm = \rho(x) \cdot dS = k(R - |x|) \cdot \sqrt{R^2 - x^2} \, dx$

由对称性，只需计算 $x \geq 0$ 的部分再乘以2：

$M = 2 \int_0^R k(R - x) \cdot \sqrt{R^2 - x^2} \, dx = 2k \int_0^R (R - x)\sqrt{R^2 - x^2} \, dx$

$= 2k[R \int_0^R \sqrt{R^2 - x^2} \, dx - \int_0^R x\sqrt{R^2 - x^2} \, dx]$

$= 2k[R \cdot \frac{\pi R^2}{4} - \frac{R^3}{3}] = \frac{\pi kR^3}{2} - \frac{2kR^3}{3}$
