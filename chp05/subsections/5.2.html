5.2 微积分基本公式

Fundamental Theorem of Calculus

5.2.1 变速直线运动中的位置与速度

记忆要点

微积分基本定理的本质：揭示了微分与积分的互逆关系。位置函数的导数是速度函数，速度函数的积分是位移。这种互逆关系是微积分理论的核心。

变速直线运动的基本关系

设物体在直线上运动，其位置函数为 $s(t)$，速度函数为 $v(t)$，加速度函数为 $a(t)$，则有：

速度与位置的关系
$v(t) = s'(t) = \frac{ds}{dt}$
加速度与速度的关系
$a(t) = v'(t) = \frac{dv}{dt} = s''(t)$

逆向思考：

  • 已知速度函数 $v(t)$，如何求位移？答：$\Delta s = \int_{t_1}^{t_2} v(t) \, dt$
  • 已知加速度函数 $a(t)$，如何求速度变化？答：$\Delta v = \int_{t_1}^{t_2} a(t) \, dt$

位移与路程的区别

位移：位置的变化量，有方向性，可能为正、负或零

$$\text{位移} = s(t_2) - s(t_1) = \int_{t_1}^{t_2} v(t) \, dt$$

路程：运动轨迹的总长度，总是非负的

$$\text{路程} = \int_{t_1}^{t_2} |v(t)| \, dt$$

注意：当速度不变号时，位移的绝对值等于路程；当速度变号时，需要分段计算。

例1：自由落体运动

一物体从高度 $h_0 = 80 \text{m}$ 处自由下落，重力加速度 $g = 10 \text{m/s}^2$。求：

  1. 速度函数 $v(t)$
  2. 位置函数 $s(t)$
  3. 物体落地时间和落地速度
解：

（1）求速度函数

由于 $a(t) = -g = -10$，初始速度 $v(0) = 0$

$$v(t) = \int a(t) \, dt = \int (-10) \, dt = -10t + C$$

由 $v(0) = 0$ 得 $C = 0$，所以 $v(t) = -10t$

（2）求位置函数

设向下为负方向，初始位置 $s(0) = 80$

$$s(t) = \int v(t) \, dt = \int (-10t) \, dt = -5t^2 + C$$

由 $s(0) = 80$ 得 $C = 80$，所以 $s(t) = -5t^2 + 80$

（3）求落地时间和速度

落地时 $s(t) = 0$：$-5t^2 + 80 = 0$，解得 $t = 4 \text{s}$

落地速度：$v(4) = -10 \times 4 = -40 \text{m/s}$

例2：变速运动的位移计算

已知物体的速度函数为 $v(t) = t^2 - 6t + 8$（$\text{m/s}$），求物体在 $t = 0$ 到 $t = 6$ 秒内的位移和路程。

解：

步骤1：求位移

$$\text{位移} = \int_0^6 (t^2 - 6t + 8) \, dt = \left[\frac{t^3}{3} - 3t^2 + 8t\right]_0^6$$ $$= \frac{216}{3} - 108 + 48 = 72 - 108 + 48 = 12 \text{m}$$

步骤2：分析速度的符号

$v(t) = t^2 - 6t + 8 = (t-2)(t-4)$

  • 当 $t \in [0,2)$ 时，$v(t) > 0$（向正方向运动）
  • 当 $t \in (2,4)$ 时，$v(t) < 0$（向负方向运动）
  • 当 $t \in (4,6]$ 时，$v(t) > 0$（向正方向运动）

步骤3：求路程

$$\text{路程} = \int_0^2 (t^2 - 6t + 8) \, dt + \int_2^4 |t^2 - 6t + 8| \, dt + \int_4^6 (t^2 - 6t + 8) \, dt$$ $$= \int_0^2 (t^2 - 6t + 8) \, dt - \int_2^4 (t^2 - 6t + 8) \, dt + \int_4^6 (t^2 - 6t + 8) \, dt$$

计算各段：

  • $\int_0^2 (t^2 - 6t + 8) \, dt = \frac{20}{3} \text{m}$
  • $\int_2^4 (t^2 - 6t + 8) \, dt = -\frac{8}{3} \text{m}$
  • $\int_4^6 (t^2 - 6t + 8) \, dt = \frac{20}{3} \text{m}$

因此：$\text{路程} = \frac{20}{3} + \frac{8}{3} + \frac{20}{3} = 16 \text{m}$

5.2.2 积分上限的函数及其导数

积分上限函数的定义

设函数 $f(x)$ 在区间 $[a, b]$ 上连续，则对于任意 $x \in [a, b]$，定义函数：

$$\Phi(x) = \int_a^x f(t) \, dt$$

称 $\Phi(x)$ 为以 $x$ 为上限的积分函数，或简称积分上限函数。

重要性质：

  • $\Phi(a) = \int_a^a f(t) \, dt = 0$
  • $\Phi(x)$ 在 $[a, b]$ 上连续
  • $\Phi(x)$ 在 $[a, b]$ 上可导，且 $\Phi'(x) = f(x)$

微积分基本定理（第一形式）

如果函数 $f(x)$ 在区间 $[a, b]$ 上连续，则积分上限函数

$$\Phi(x) = \int_a^x f(t) \, dt$$

在 $[a, b]$ 上可导，且

$$\Phi'(x) = \frac{d}{dx} \int_a^x f(t) \, dt = f(x)$$

证明思路：利用导数定义和积分中值定理

$$\Phi'(x) = \lim_{\Delta x \to 0} \frac{\Phi(x + \Delta x) - \Phi(x)}{\Delta x} = \lim_{\Delta x \to 0} \frac{\int_x^{x+\Delta x} f(t) \, dt}{\Delta x} = f(x)$$

积分上限函数求导的扩展公式

更一般地，如果 $\alpha(x)$ 和 $\beta(x)$ 都是可导函数，则：

$$\frac{d}{dx} \int_{\alpha(x)}^{\beta(x)} f(t) \, dt = f(\beta(x)) \cdot \beta'(x) - f(\alpha(x)) \cdot \alpha'(x)$$

记忆口诀："上限代入乘上导，下限代入乘下导，下限前面添负号"

例3：积分上限函数的导数

求下列函数的导数：

  1. $F(x) = \int_0^x \sin t^2 \, dt$
  2. $G(x) = \int_1^{x^2} \frac{1}{1 + t^2} \, dt$
  3. $H(x) = \int_{x^2}^{x^3} e^{t^2} \, dt$
解：

（1）直接应用基本定理：

$$F'(x) = \sin x^2$$

（2）上限是 $x$ 的函数：

$$G'(x) = \frac{1}{1 + (x^2)^2} \cdot (x^2)' = \frac{1}{1 + x^4} \cdot 2x = \frac{2x}{1 + x^4}$$

（3）上、下限都是 $x$ 的函数：

$$H'(x) = e^{(x^3)^2} \cdot (x^3)' - e^{(x^2)^2} \cdot (x^2)'$$ $$= e^{x^6} \cdot 3x^2 - e^{x^4} \cdot 2x = 3x^2 e^{x^6} - 2x e^{x^4}$$

5.2.3 牛顿-莱布尼兹公式

牛顿-莱布尼兹公式（微积分基本定理第二形式）

如果函数 $F(x)$ 是连续函数 $f(x)$ 在区间 $[a, b]$ 上的一个原函数，即 $F'(x) = f(x)$，则：

$$\int_a^b f(x) \, dx = F(b) - F(a) = \left[F(x)\right]_a^b$$

这个公式的意义：

  • 将定积分的计算转化为原函数值的差
  • 建立了不定积分与定积分的联系
  • 使得大多数定积分可以通过求原函数来计算

微积分基本定理的统一表述

微积分基本定理揭示了微分和积分的互逆关系：

第一基本定理
$$\frac{d}{dx} \int_a^x f(t) \, dt = f(x)$$

积分的导数等于被积函数

第二基本定理
$$\int_a^b f(x) \, dx = F(b) - F(a)$$

定积分等于原函数值的差

记忆要点

核心思想：微分和积分是互逆运算。就像加法和减法、乘法和除法一样，一个运算可以"撤销"另一个运算的效果。这是微积分理论的基石。

例4：利用牛顿-莱布尼兹公式计算定积分

计算下列定积分：

  1. $\int_0^1 x^2 \, dx$
  2. $\int_0^{\pi} \sin x \, dx$
  3. $\int_1^e \frac{1}{x} \, dx$
解：

（1）$\int_0^1 x^2 \, dx$

$x^2$ 的原函数是 $\frac{x^3}{3}$，所以：

$$\int_0^1 x^2 \, dx = \left[\frac{x^3}{3}\right]_0^1 = \frac{1^3}{3} - \frac{0^3}{3} = \frac{1}{3}$$

（2）$\int_0^{\pi} \sin x \, dx$

$\sin x$ 的原函数是 $-\cos x$，所以：

$$\int_0^{\pi} \sin x \, dx = \left[-\cos x\right]_0^{\pi} = -\cos \pi - (-\cos 0) = -(-1) + 1 = 2$$

（3）$\int_1^e \frac{1}{x} \, dx$

$\frac{1}{x}$ 的原函数是 $\ln x$，所以：

$$\int_1^e \frac{1}{x} \, dx = \left[\ln x\right]_1^e = \ln e - \ln 1 = 1 - 0 = 1$$

例5：含参数的定积分

设 $F(a) = \int_0^a x^2 e^{-x} \, dx$，求 $F'(a)$。

解：

方法一：直接利用积分上限函数的导数公式

$$F'(a) = \frac{d}{da} \int_0^a x^2 e^{-x} \, dx = a^2 e^{-a}$$

方法二：先计算积分再求导（验证）

使用分部积分法求 $\int x^2 e^{-x} \, dx$：

设 $u = x^2$，$dv = e^{-x} dx$，则 $du = 2x dx$，$v = -e^{-x}$

$$\int x^2 e^{-x} \, dx = -x^2 e^{-x} + 2\int x e^{-x} \, dx$$

继续对 $\int x e^{-x} \, dx$ 分部积分，最终得到：

$$\int x^2 e^{-x} \, dx = -(x^2 + 2x + 2)e^{-x} + C$$

因此：

$$F(a) = \left[-(x^2 + 2x + 2)e^{-x}\right]_0^a = -(a^2 + 2a + 2)e^{-a} + 2$$ $$F'(a) = a^2 e^{-a}$$

习题1 中等难度

求函数 $F(x) = \int_0^{x^2} \sin t^2 \, dt$ 的导数。

解：

利用积分上限函数求导公式，注意上限是 $x^2$：

$$F'(x) = \sin(x^2)^2 \cdot \frac{d}{dx}(x^2) = \sin(x^4) \cdot 2x = 2x \sin(x^4)$$

习题2 较难

设 $f(x)$ 连续，且 $\int_0^x f(t) \, dt = x^2 + \int_0^x tf(t) \, dt$，求 $f(x)$。

解：

设 $F(x) = \int_0^x f(t) \, dt$，$G(x) = \int_0^x tf(t) \, dt$

由题意：$F(x) = x^2 + G(x)$

对等式两边求导：

$$F'(x) = 2x + G'(x)$$

即：$f(x) = 2x + xf(x)$

整理得：$f(x) - xf(x) = 2x$

$$f(x)(1 - x) = 2x$$

因此：$f(x) = \frac{2x}{1 - x}$ （$x \neq 1$）

验证：当 $x = 0$ 时，$F(0) = 0 = 0^2 + G(0)$ ✓

习题3 较难

已知 $F(x) = \int_0^x (t^2 + 1) e^{-t^2} \, dt$，求：

  1. $F'(x)$
  2. $F''(x)$
  3. 证明：$F''(x) + 2xF'(x) = 2xe^{-x^2}$
解：

（1）求 $F'(x)$

由微积分基本定理：

$$F'(x) = \frac{d}{dx} \int_0^x (t^2 + 1) e^{-t^2} \, dt = (x^2 + 1) e^{-x^2}$$

（2）求 $F''(x)$

对 $F'(x) = (x^2 + 1) e^{-x^2}$ 求导，使用乘积法则：

$$F''(x) = \frac{d}{dx}[(x^2 + 1) e^{-x^2}]$$ $$= (x^2 + 1)' \cdot e^{-x^2} + (x^2 + 1) \cdot (e^{-x^2})'$$ $$= 2x \cdot e^{-x^2} + (x^2 + 1) \cdot e^{-x^2} \cdot (-2x)$$ $$= 2xe^{-x^2} - 2x(x^2 + 1)e^{-x^2}$$ $$= 2xe^{-x^2}[1 - (x^2 + 1)]$$ $$= 2xe^{-x^2}(-x^2) = -2x^3 e^{-x^2}$$

（3）证明恒等式

计算左边：

$$F''(x) + 2xF'(x) = -2x^3 e^{-x^2} + 2x(x^2 + 1)e^{-x^2}$$ $$= -2x^3 e^{-x^2} + 2x^3 e^{-x^2} + 2xe^{-x^2}$$ $$= 2xe^{-x^2}$$

这正好等于右边，证毕。

思考：这个恒等式实际上反映了函数 $F(x)$ 满足的一个微分方程，体现了积分上限函数与其导数之间的深层关系。
